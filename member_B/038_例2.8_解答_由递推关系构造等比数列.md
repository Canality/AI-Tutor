## 一、题目内容

> **原题位置**
> 《新高考你真的掌握了吗？·数列与不等式》 P38 例2.8

**来源**
(无明确来源，教材例题)

**题目：**
已知数列 $\{a_n\}$ 满足 $a_1 = 1$，$a_{n+1} = 2a_n + 3$ ($n \in \mathbb{N}^*$)，求 $\{a_n\}$ 的通项公式。


## 二、答案与解析

### 答案
$\{a_n\}$ 的通项公式为 $a_n = 2^{n+1} - 3$ ($n \in \mathbb{N}^*$)。

### 解析

**第一步：构造新数列。**

已知递推关系：$a_{n+1} = 2a_n + 3$

我们尝试在两边同时加上一个常数 $x$，使得新的数列 $\{a_n + x\}$ 成为等比数列。

设：
$$
a_{n+1} + x = 2(a_n + x)
$$

展开右边：
$$
a_{n+1} + x = 2a_n + 2x
$$

将 $a_{n+1} = 2a_n + 3$ 代入左边：
$$
(2a_n + 3) + x = 2a_n + 2x
$$

化简：
$$
2a_n + 3 + x = 2a_n + 2x
\Rightarrow 3 + x = 2x
\Rightarrow x = 3
$$

所以，当 $x = 3$ 时，有：
$$
a_{n+1} + 3 = 2(a_n + 3)
$$

这表明数列 $\{a_n + 3\}$ 是一个公比为 2 的等比数列。

**第二步：求首项并写出通项公式。**

计算首项：
$$
a_1 + 3 = 1 + 3 = 4
$$

所以数列 $\{a_n + 3\}$ 是以 4 为首项，2 为公比的等比数列。

其通项公式为：
$$
a_n + 3 = 4 \cdot 2^{n-1} = 2^2 \cdot 2^{n-1} = 2^{n+1}
$$

**第三步：求出 $\{a_n\}$ 的通项公式。**

由 $a_n + 3 = 2^{n+1}$ 得：
$$
a_n = 2^{n+1} - 3
$$

**第四步：验证。**

- $n=1$: $a_1 = 2^{1+1} - 3 = 4 - 3 = 1$，符合。
- $n=2$: $a_2 = 2a_1 + 3 = 2 \times 1 + 3 = 5$，而 $2^{2+1} - 3 = 8 - 3 = 5$，符合。
- $n=3$: $a_3 = 2a_2 + 3 = 2 \times 5 + 3 = 13$，而 $2^{3+1} - 3 = 16 - 3 = 13$，符合。

综上所述，数列 $\{a_n\}$ 的通项公式为 $a_n = 2^{n+1} - 3$ ($n \in \mathbb{N}^*$)。

【方法总结 2.4】 形如 $a_{n+1} = p a_n + q$ ($p \neq 1, q \neq 0$) 的递推式，两边同时加上 $x$ 可构造成等比数列 $\{a_n + x\}$ ($n \in \mathbb{N}^*$)，通过比较可得 $x = \frac{q}{p - 1}$。

> 对于方法总结 2.4，希望同学们亲自推导，而不要去硬记这个结论。

## 三、分步教学引导

### 步骤1：观察递推结构
- **引导语**："我们手头的递推关系是 $a_{n+1} = 2a_n + 3$。这个关系有什么特点？"
- **关键提示**：这是一个形如 $a_{n+1} = p a_n + q$ 的线性非齐次递推关系。
- **期望学生回答**：这是一个等比数列的变形，但多了一个常数项 3。

### 步骤2：构造新数列
- **引导语**："为了消除常数项，我们可以尝试在两边同时加上一个常数 $x$。你能猜到这个 $x$ 应该是多少吗？"
- **关键提示**：假设 $a_{n+1} + x = 2(a_n + x)$，然后解出 $x$。
- **期望学生回答**：我假设 $a_{n+1} + x = 2(a_n + x)$，代入 $a_{n+1} = 2a_n + 3$，得到 $2a_n + 3 + x = 2a_n + 2x$，所以 $3 + x = 2x$，解得 $x = 3$。

### 步骤3：证明等比数列
- **引导语**："现在我们有 $a_{n+1} + 3 = 2(a_n + 3)$。这说明了什么？"
- **关键提示**：这是一个等比数列的定义。
- **期望学生回答**：这说明数列 $\{a_n + 3\}$ 是公比为 2 的等比数列。

### 步骤4：求通项公式
- **引导语**："我们现在知道 $\{a_n + 3\}$ 是等比数列，首项 $a_1 + 3 = 4$，公比为 2。你能写出 $a_n + 3$ 的通项公式吗？"
- **关键提示**：等比数列通项公式为 $b_n = b_1 \cdot r^{n-1}$。
- **期望学生回答**：$a_n + 3 = 4 \cdot 2^{n-1} = 2^{n+1}$。

### 步骤5：还原到原数列
- **引导语**："我们已经求出了 $a_n + 3 = 2^{n+1}$。你能从中解出 $a_n$ 吗？"
- **关键提示**：移项即可。
- **期望学生回答**：$a_n = 2^{n+1} - 3$。

### 步骤6：理解方法总结
- **引导语**："我们发现 $x = 3$，而 $q = 3$, $p = 2$，所以 $x = \frac{q}{p - 1}$。你能解释一下为什么这个公式成立吗？"
- **关键提示**：从 $a_{n+1} + x = p(a_n + x)$ 推导。
- **期望学生回答**：因为 $a_{n+1} + x = p a_n + p x$，又 $a_{n+1} = p a_n + q$，所以 $p a_n + q + x = p a_n + p x$，即 $q + x = p x$，解得 $x = \frac{q}{p - 1}$。
