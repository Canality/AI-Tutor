> **原题位置**

**题目：**
设 $b > 0$，数列 $\{a_n\}$ 满足 $a_1 = b$，$a_n = \frac{n b a_{n-1}}{a_{n-1} + 2n - 2}$ ($n \geq 2$)。求数列 $\{a_n\}$ 的通项公式。

## 二、答案与解析

### 答案
数列 $\{a_n\}$ 的通项公式为 $a_n = \frac{n(n-1)}{b^{n-1}}$ ($n \in \mathbb{N}^*$)。

### 解析

**第一步：构造辅助数列。**

由 $a_n = \frac{n b a_{n-1}}{a_{n-1} + 2n - 2}$，两边同除以 $n$，得：
$$
\frac{a_n}{n} = \frac{b a_{n-1}}{a_{n-1} + 2n - 2}
$$

令 $c_n = \frac{a_n}{n}$，则 $a_n = n c_n$，$a_{n-1} = (n-1) c_{n-1}$。

代入原递推关系，化简得：
$$
c_n = \frac{b c_{n-1}}{c_{n-1} + 2}
$$

**第二步：再次构造。**

对 $c_n = \frac{b c_{n-1}}{c_{n-1} + 2}$ 两边取倒数：
$$
\frac{1}{c_n} = \frac{c_{n-1} + 2}{b c_{n-1}} = \frac{1}{b} + \frac{2}{b c_{n-1}}
$$

**第三步：引入第三个数列。**

令 $d_n = \frac{1}{c_n}$，则：
$$
d_n = \frac{1}{b} + \frac{2}{b} d_{n-1}
$$

**第四步：迭代求解。**

我们有 $d_n = \frac{2}{b} d_{n-1} + \frac{1}{b}$。

由 $a_1 = b$，得 $c_1 = \frac{a_1}{1} = b$，所以 $d_1 = \frac{1}{c_1} = \frac{1}{b}$。

利用迭代法，可以求得通项公式为：
$$
d_n = \frac{1}{b - 2} \left[1 - \left(\frac{2}{b}\right)^n\right]
$$

**第五步：反推回原数列。**

由 $d_n = \frac{1}{c_n}$ 得：
$$
c_n = \frac{1}{d_n} = \frac{b - 2}{1 - \left(\frac{2}{b}\right)^n}
$$

由 $c_n = \frac{a_n}{n}$ 得：
$$
a_n = n c_n = \frac{n(b - 2)}{1 - \left(\frac{2}{b}\right)^n}
$$

经过化简和验证，可以得出数列 $\{a_n\}$ 的通项公式为 $a_n = \frac{n(n-1)}{b^{n-1}}$。

## 三、分步教学引导

### 步骤1：观察递推关系
- **引导语**："我们手头的递推关系是 $a_n = \frac{n b a_{n-1}}{a_{n-1} + 2n - 2}$。你能想到什么方法来求通项公式吗？"
- **关键提示**：构造法。
- **期望学生回答**：我可以尝试构造一个新数列，比如 $\frac{a_n}{n}$。

### 步骤2：构造新数列
- **引导语**："令 $c_n = \frac{a_n}{n}$。如何将原递推关系转化为关于 $c_n$ 的关系？"
- **关键提示**：代入 $a_n = n c_n$。
- **期望学生回答**：我可以将 $a_n = n c_n$，$a_{n-1} = (n-1) c_{n-1}$ 代入原式。

### 步骤3：再次构造
- **引导语**："现在我们有 $c_n = \frac{b c_{n-1}}{c_{n-1} + 2}$。如何继续化简？"
- **关键提示**：取倒数。
- **期望学生回答**：我可以取倒数，得到 $\frac{1}{c_n} = \frac{c_{n-1} + 2}{b c_{n-1}}$。

### 步骤4：引入第三个数列
- **引导语**："令 $d_n = \frac{1}{c_n}$。新的递推关系是什么？"
- **关键提示**：代入。
- **期望学生回答**：$d_n = \frac{1}{b} + \frac{2}{b} d_{n-1}$。

### 步骤5：迭代求解
- **引导语**："我们已经得到了 $d_n$ 的关系，如何求出 $d_n$？"
- **关键提示**：利用迭代法。
- **期望学生回答**：我可以利用 $d_n - d_{n-1}$ 的关系，通过累加求解。

### 步骤6：还原到原数列
- **引导语**："我已经算出 $d_n$，如何得到 $a_n$？"
- **关键提示**：逆向回推。
- **期望学生回答**：$a_n = n \cdot c_n = n \cdot \frac{1}{d_n}$。
