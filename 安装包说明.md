## 🚀 快速开始

### 方式一：一键启动（推荐）

1. 确保已安装 Python 3.9+、MySQL 8.0+、Redis 6.0+
2. 双击运行 `一键启动.bat`
3. 等待服务启动完成
4. 浏览器访问 http://localhost:8000

### 方式二：手动启动

#### 1. 安装依赖

```bash
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
venv\Scripts\activate

# 安装依赖
pip install -r backend\requirements.txt
```

#### 2. 配置数据库

```bash
# 在 MySQL 中创建数据库
mysql -u root -p
CREATE DATABASE ai_tutor CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

#### 3. 配置环境变量

复制 `backend\.env.example` 为 `backend\.env`，并修改以下配置：

```env
# MySQL 配置
MYSQL_HOST=localhost
MYSQL_PORT=3306
MYSQL_USER=root
MYSQL_PASSWORD=你的密码
MYSQL_DATABASE=ai_tutor

# Redis 配置
REDIS_HOST=localhost
REDIS_PORT=6379

# 硅基流动 API 配置
OPENAI_API_KEY=sk-你的API密钥
OPENAI_API_BASE=https://api.siliconflow.cn/v1

# JWT 密钥（随机生成）
SECRET_KEY=你的随机密钥
```

#### 4. 初始化数据库

```bash
cd backend
python -m database.init_db
```

#### 5. 启动服务

```bash
# 启动后端
cd backend
python -m uvicorn main:app --host 0.0.0.0 --port 8000

# 启动前端（新窗口）
cd frontend
npm install
npm run dev
```

## 🔧 功能特性

- ✅ **智能解题**：基于 DeepSeek-V3 的数学推理
- ✅ **图片识题**：支持上传图片识别数学题目
- ✅ **RAG 检索**：结合知识库提供精准解答
- ✅ **知识图谱**：数列知识点关联分析
- ✅ **个性化推荐**：根据学习情况推送练习题
