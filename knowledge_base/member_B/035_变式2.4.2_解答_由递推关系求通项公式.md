## 一、题目内容

> **原题位置**
> 《新高考你真的掌握了吗？·数列与不等式》 P35 变式2.4.2

**来源**
(2019华大高考联盟文16)

**题目：**
设数列 $\{a_n\}$ 的前 $n$ 项和 $S_n$ 满足 $S_{n+1} = 3S_n - 3S_{n-1} + S_{n-2} + 2$ ($n \geq 3$)，且 $a_1=3$，$a_2=8$，$a_3=15$，则 $a_n = $ ______。

## 二、答案与解析

### 答案
$a_n = n^2 + 2n$。

### 解析

**第一步：利用递推关系求出数列的前几项。**

根据递推关系 $S_{n+1} = 3S_n - 3S_{n-1} + S_{n-2} + 2$ ($n \geq 3$)，我们可以通过已知的 $a_1, a_2, a_3$ 计算出后续项。

已知 $a_1=3$, $a_2=8$, $a_3=15$。
- $S_1 = a_1 = 3$
- $S_2 = a_1 + a_2 = 3 + 8 = 11$
- $S_3 = a_1 + a_2 + a_3 = 3 + 8 + 15 = 26$

使用递推式计算 $S_4$ ($n=3$)：
$$
S_4 = 3S_3 - 3S_2 + S_1 + 2 = 3 \times 26 - 3 \times 11 + 3 + 2 = 78 - 33 + 3 + 2 = 50
$$

所以 $a_4 = S_4 - S_3 = 50 - 26 = 24$

计算 $S_5$ ($n=4$)：
$$
S_5 = 3S_4 - 3S_3 + S_2 + 2 = 3 \times 50 - 3 \times 26 + 11 + 2 = 150 - 78 + 11 + 2 = 85
$$

所以 $a_5 = S_5 - S_4 = 85 - 50 = 35$

计算 $S_6$ ($n=5$)：
$$
S_6 = 3S_5 - 3S_4 + S_3 + 2 = 3 \times 85 - 3 \times 50 + 26 + 2 = 255 - 150 + 26 + 2 = 133
$$

所以 $a_6 = S_6 - S_5 = 133 - 85 = 48$

**第二步：观察数列 $\{a_n\}$ 的规律。**

我们得到数列 $\{a_n\}$ 的前几项为：
- $a_1 = 3$
- $a_2 = 8$
- $a_3 = 15$
- $a_4 = 24$
- $a_5 = 35$
- $a_6 = 48$

观察相邻项的差值：
- $a_2 - a_1 = 8 - 3 = 5$
- $a_3 - a_2 = 15 - 8 = 7$
- $a_4 - a_3 = 24 - 15 = 9$
- $a_5 - a_4 = 35 - 24 = 11$
- $a_6 - a_5 = 48 - 35 = 13$

得到一个新的数列 $\{a_{n+1} - a_n\}$：$5, 7, 9, 11, 13, ...$，这是一个首项为5，公差为2的等差数列。

这表明数列 $\{a_n\}$ 是一个**二阶等差数列**，其通项公式 $a_n$ 是关于 $n$ 的一个二次函数。

**第三步：设二次函数模型求解通项公式。**

设 $a_n = An^2 + Bn + C$。

将已知的 $a_1, a_2, a_3$ 代入方程组：
- $n=1$: $A + B + C = 3$ ... (1)
- $n=2$: $4A + 2B + C = 8$ ... (2)
- $n=3$: $9A + 3B + C = 15$ ... (3)

解方程组：
- (2) - (1) 得：$3A + B = 5$ ... (4)
- (3) - (2) 得：$5A + B = 7$ ... (5)

- (5) - (4) 得：$2A = 2$，所以 $A = 1$。
- 将 $A=1$ 代入 (4) 得：$3 \times 1 + B = 5$，所以 $B = 2$。
- 将 $A=1, B=2$ 代入 (1) 得：$1 + 2 + C = 3$，所以 $C = 0$。

**第四步：得出通项公式并验证。**

因此，$a_n = n^2 + 2n$。

验证：
- $a_1 = 1^2 + 2 \times 1 = 1 + 2 = 3$ ✓
- $a_2 = 2^2 + 2 \times 2 = 4 + 4 = 8$ ✓
- $a_3 = 3^2 + 2 \times 3 = 9 + 6 = 15$ ✓
- $a_4 = 4^2 + 2 \times 4 = 16 + 8 = 24$ ✓
- $a_5 = 5^2 + 2 \times 5 = 25 + 10 = 35$ ✓
- $a_6 = 6^2 + 2 \times 6 = 36 + 12 = 48$ ✓

所有已知项均符合，故通项公式成立。

## 三、分步教学引导

### 步骤1：计算前几项
- **引导语**："题目给出了一个关于 $S_n$ 的复杂递推关系，以及 $a_1, a_2, a_3$ 的值。为了寻找 $\{a_n\}$ 的规律，我们首先应该做什么？"
- **关键提示**：利用 $a_n = S_n - S_{n-1}$ 和已知的递推关系，计算出 $a_4, a_5$ 等后续项。
- **期望学生回答**：我需要先算出 $S_1, S_2, S_3$，然后用递推关系算出 $S_4, S_5$，最后用 $a_n = S_n - S_{n-1}$ 求出 $a_4, a_5$。

### 步骤2：观察差值规律
- **引导语**："现在我们有了 $\{a_n\}$ 的前几项：3, 8, 15, 24, 35...。这些数字直接看很难看出规律。我们能否研究一下它们之间的变化？"
- **关键提示**：计算相邻两项的差 $a_{n+1} - a_n$。
- **期望学生回答**：我来算一下差值：$a_2-a_1=5$, $a_3-a_2=7$, $a_4-a_3=9$, $a_5-a_4=11$。我发现这些差值构成了一个等差数列。

### 步骤3：判断数列类型
- **引导语**："我们发现 $\{a_{n+1} - a_n\}$ 是一个等差数列。这说明原始数列 $\{a_n\}$ 有什么特点？"
- **关键提示**：如果一个数列的相邻项之差构成等差数列，那么这个数列被称为二阶等差数列。
- **期望学生回答**：这说明 $\{a_n\}$ 是一个二阶等差数列。对于二阶等差数列，它的通项公式 $a_n$ 应该是关于 $n$ 的二次函数。

### 步骤4：设二次函数模型
- **引导语**："既然 $\{a_n\}$ 很可能是二次函数，我们应该设怎样的通项公式来求解？"
- **关键提示**：设 $a_n = An^2 + Bn + C$，其中 $A, B, C$ 是待定系数。
- **期望学生回答**：我应该设 $a_n = An^2 + Bn + C$。

### 步骤5：建立并求解方程组
- **引导语**："设定了 $a_n = An^2 + Bn + C$，我们如何求出系数 $A, B, C$？"
- **关键提示**：将已知的 $a_1, a_2, a_3$ 的值代入设定的公式，建立一个关于 $A, B, C$ 的三元一次方程组。
- **期望学生回答**：将 $a_1=3$, $a_2=8$, $a_3=15$ 分别代入 $An^2 + Bn + C$，得到三个方程，然后解这个方程组。
