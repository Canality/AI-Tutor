# 【E_124】数列单调性与不等式证明

## 一、题目内容

> **原题位置**：《新高考你真的掌握了吗？》P124 变式5.9.2
> 
> **来源**：2017浙江模考20题

**题目：**

已知数列 $\{a_n\}$ 满足 $a_1 = 1$，$a_{n+1} = \frac{2a_n + 1}{a_n + 2}$（$n \in \mathbb{N}^*$）。

**(I)** 证明：数列 $\left\{\left|a_n - 1\right|\right\}$ 为单调递减数列；

**(II)** 记 $S_n$ 为数列 $\left\{\left|a_n - 1\right|\right\}$ 的前 $n$ 项和，证明：$S_n < \frac{2}{3}$（$n \in \mathbb{N}^*$）。

---

## 二、答案与解析

### (I) 证明数列 $\left\{\left|a_n - 1\right|\right\}$ 为单调递减数列

**答案：** 证明见解析

**解析：**

**步骤1：分析递推公式**
$a_{n+1} = \frac{2a_n + 1}{a_n + 2}$

**步骤2：计算 $a_{n+1} - 1$**
$$a_{n+1} - 1 = \frac{2a_n + 1}{a_n + 2} - 1 = \frac{2a_n + 1 - (a_n + 2)}{a_n + 2} = \frac{a_n - 1}{a_n + 2}$$

**步骤3：求 $|a_{n+1} - 1|$**
$$|a_{n+1} - 1| = \frac{|a_n - 1|}{|a_n + 2|}$$

**步骤4：分析分母**
由于 $a_1 = 1 > 0$，且递推公式保持正性，所以 $a_n > 0$ 对所有 $n$ 成立。
因此 $a_n + 2 > 2$，即 $|a_n + 2| = a_n + 2 > 2$。

**步骤5：建立不等式**
$$|a_{n+1} - 1| = \frac{|a_n - 1|}{a_n + 2} < \frac{|a_n - 1|}{2}$$

因为 $a_n + 2 > 2$，所以 $\frac{1}{a_n + 2} < \frac{1}{2}$。

**步骤6：证明单调递减**
由 $|a_{n+1} - 1| < \frac{1}{2}|a_n - 1| < |a_n - 1|$（当 $|a_n - 1| > 0$ 时）

且 $|a_n - 1| \geq 0$，等号仅当 $a_n = 1$ 时成立。

由于 $a_1 = 1$，计算 $a_2 = \frac{2 \times 1 + 1}{1 + 2} = 1$，所以 $a_n = 1$ 对所有 $n$ 成立？

> **修正**：重新计算
> $a_1 = 1$
> $a_2 = \frac{2 \times 1 + 1}{1 + 2} = \frac{3}{3} = 1$
> 
> 若 $a_1 = 1$，则所有 $a_n = 1$，此时 $|a_n - 1| = 0$，不是严格递减。
> 
> **可能原题条件不同**：请以原书为准，可能 $a_1 \neq 1$ 或有其他条件。

**正确分析**（假设 $a_1 \neq 1$）：
若 $a_1 \neq 1$，则 $|a_2 - 1| = \frac{|a_1 - 1|}{a_1 + 2} < |a_1 - 1|$（因为 $a_1 + 2 > 1$）

归纳可得 $|a_{n+1} - 1| < |a_n - 1|$，即数列单调递减。

---

### (II) 证明：$S_n < \frac{2}{3}$

**答案：** 证明见解析

**解析：**

**步骤1：利用(I)的结论**
由 (I) 知 $|a_{n+1} - 1| < \frac{1}{2}|a_n - 1|$（假设 $a_n > 0$）

**步骤2：建立等比关系**
设 $b_n = |a_n - 1|$，则 $b_{n+1} < \frac{1}{2}b_n$

这是一个等比递减关系，公比 $q = \frac{1}{2}$。

**步骤3：求 $b_n$ 的上界**
$$b_n < \frac{1}{2}b_{n-1} < \frac{1}{2^2}b_{n-2} < \cdots < \frac{1}{2^{n-1}}b_1$$

**步骤4：求 $S_n$**
$$S_n = \sum_{k=1}^{n} b_k < b_1 \sum_{k=1}^{n} \frac{1}{2^{k-1}} = b_1 \cdot \frac{1 - \frac{1}{2^n}}{1 - \frac{1}{2}} = 2b_1\left(1 - \frac{1}{2^n}\right) < 2b_1$$

**步骤5：确定 $b_1$ 的值**
若 $a_1 = 1$，则 $b_1 = 0$，$S_n = 0 < \frac{2}{3}$ 显然成立。

若 $a_1$ 为其他值（如 $a_1 = 2$），则 $b_1 = |a_1 - 1| = 1$，$S_n < 2$。

要证 $S_n < \frac{2}{3}$，需要更精确的放缩或不同的初始条件。

**更精确的放缩**：
实际上，由 $a_{n+1} - 1 = \frac{a_n - 1}{a_n + 2}$，当 $a_n > 0$ 时，$a_n + 2 > 2$。

若 $a_n \geq 1$，则 $a_n + 2 \geq 3$，所以 $|a_{n+1} - 1| \leq \frac{1}{3}|a_n - 1|$。

此时 $S_n < b_1 \cdot \frac{1}{1 - \frac{1}{3}} = \frac{3b_1}{2}$。

若 $b_1 = \frac{4}{9}$，则 $S_n < \frac{2}{3}$。

> **注意**：具体数值请以原书为准，上述为基于递推公式的一般性分析。
