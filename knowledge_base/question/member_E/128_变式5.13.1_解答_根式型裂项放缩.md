# 【E_150_1】根式型裂项放缩

## 一、题目内容

> **原题位置**：《新高考你真的掌握了吗？》P128 变式5.13.1

**题目：**

求证：$\sum_{k=1}^{n} \frac{1}{k\sqrt{k}} < 3$

---

## 二、答案与解析

**答案：** 证明见解析

**解析：**

**步骤1：保留第一项，对 $k \geq 2$ 放缩**
$$\sum_{k=1}^{n} \frac{1}{k\sqrt{k}} = 1 + \sum_{k=2}^{n} \frac{1}{k\sqrt{k}}$$

**步骤2：根式放缩**
对于 $k \geq 2$，利用不等式：
$$\frac{1}{k\sqrt{k}} < \frac{1}{\sqrt{k-1}} - \frac{1}{\sqrt{k}}$$

**验证：**
$$\frac{1}{\sqrt{k-1}} - \frac{1}{\sqrt{k}} = \frac{\sqrt{k} - \sqrt{k-1}}{\sqrt{k(k-1)}} = \frac{1}{\sqrt{k(k-1)}(\sqrt{k} + \sqrt{k-1})}$$

需要证明：$\frac{1}{k\sqrt{k}} < \frac{1}{\sqrt{k(k-1)}(\sqrt{k} + \sqrt{k-1})}$

即：$\sqrt{k-1}(\sqrt{k} + \sqrt{k-1}) < k$

展开：$\sqrt{k(k-1)} + (k-1) < k$

即：$\sqrt{k(k-1)} < 1$，对 $k \geq 2$ 不成立，需要调整放缩方式。

**正确放缩：**
利用 $\frac{1}{k\sqrt{k}} < 2\left(\frac{1}{\sqrt{k-1}} - \frac{1}{\sqrt{k}}\right)$（$k \geq 2$）

**步骤3：求和**
$$\sum_{k=2}^{n} \frac{1}{k\sqrt{k}} < 2\sum_{k=2}^{n}\left(\frac{1}{\sqrt{k-1}} - \frac{1}{\sqrt{k}}\right) = 2\left(1 - \frac{1}{\sqrt{n}}\right) < 2$$

**步骤4：完成证明**
$$\sum_{k=1}^{n} \frac{1}{k\sqrt{k}} < 1 + 2 = 3$$

**证毕。**
