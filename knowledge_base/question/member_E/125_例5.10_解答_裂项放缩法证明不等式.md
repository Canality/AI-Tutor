# 【E_125】裂项放缩法证明不等式

## 一、题目内容

> **原题位置**：《新高考你真的掌握了吗？》P125 例5.10
> 
> **来源**：经典例题

**题目：**

求证：$\sum_{k=1}^{n} \frac{1}{k^2} < 2$

---

## 二、答案与解析

**答案：** 证明见解析

**解析：**

**步骤1：分析通项**
需要证明 $\sum_{k=1}^{n} \frac{1}{k^2} < 2$

**步骤2：裂项放缩**
对于 $k \geq 2$，有：
$$\frac{1}{k^2} < \frac{1}{k(k-1)} = \frac{1}{k-1} - \frac{1}{k}$$

**关键技巧**：将分母 $k^2$ 放缩为 $k(k-1)$，使得分式可以裂项。

**步骤3：求和**
$$\sum_{k=1}^{n} \frac{1}{k^2} = 1 + \sum_{k=2}^{n} \frac{1}{k^2}$$

$$< 1 + \sum_{k=2}^{n} \left(\frac{1}{k-1} - \frac{1}{k}\right)$$

**步骤4：裂项相消**
$$= 1 + \left(1 - \frac{1}{2}\right) + \left(\frac{1}{2} - \frac{1}{3}\right) + \cdots + \left(\frac{1}{n-1} - \frac{1}{n}\right)$$

$$= 1 + 1 - \frac{1}{n} = 2 - \frac{1}{n} < 2$$

**证毕。**

---

## 三、分步教学引导（用于AI Agent）

**步骤1：理解目标**
- **引导语**："要证明 $\sum_{k=1}^{n} \frac{1}{k^2} < 2$，直接求和可以吗？"
- **关键提示**：$\frac{1}{k^2}$ 无法直接求和
- **期望学生回答**：需要放缩后再求和

**步骤2：寻找放缩方式**
- **引导语**："我们希望把 $\frac{1}{k^2}$ 变成可以裂项的形式，怎么放缩？"
- **关键提示**：让分母变成两个数的乘积
- **期望学生回答**：$\frac{1}{k^2} < \frac{1}{k(k-1)}$

**步骤3：验证放缩**
- **引导语**："为什么 $\frac{1}{k^2} < \frac{1}{k(k-1)}$？"
- **关键提示**：比较分母，$k^2 > k(k-1) = k^2 - k$
- **期望学生回答**：因为 $k^2 > k^2 - k$，所以 $\frac{1}{k^2} < \frac{1}{k^2-k} = \frac{1}{k(k-1)}$

**步骤4：裂项**
- **引导语**："$\frac{1}{k(k-1)}$ 可以裂项吗？"
- **期望学生回答**：$\frac{1}{k(k-1)} = \frac{1}{k-1} - \frac{1}{k}$

**步骤5：求和证明**
- **引导语**："现在可以求和了，写出过程"
- **期望学生回答**：裂项相消后得到 $2 - \frac{1}{n} < 2$

## 五、关联拓展

### 相关公式
**裂项公式：**
- $\frac{1}{k(k+1)} = \frac{1}{k} - \frac{1}{k+1}$
- $\frac{1}{k(k-1)} = \frac{1}{k-1} - \frac{1}{k}$（$k \geq 2$）

**更精确的放缩：**
- $\sum_{k=1}^{n} \frac{1}{k^2} < \frac{\pi^2}{6} \approx 1.645$（当 $n \to \infty$ 时）
