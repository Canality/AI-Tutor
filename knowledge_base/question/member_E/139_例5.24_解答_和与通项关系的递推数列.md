# 【E_139】和与通项关系的递推数列

## 一、题目内容

> **原题位置**：《新高考你真的掌握了吗？》P139 例5.24
> 
> **来源**：2012广东高考

**题目：**

已知 $2S_n = a_{n+1} - 2^{n+1} + 1$。

**(I)** 求 $a_n = 3^n - 2^n$；

**(II)** 证明：$\sum_{k=1}^{n} \frac{1}{a_k} < \frac{3}{2}$。

---

## 二、答案与解析

### (I) 求通项公式

**步骤1：建立递推**
由 $2S_n = a_{n+1} - 2^{n+1} + 1$ 和 $2S_{n-1} = a_n - 2^n + 1$（$n \geq 2$）

相减得：$2a_n = a_{n+1} - a_n - 2^n$

即：$a_{n+1} = 3a_n + 2^n$

**步骤2：求解递推**
利用待定系数法或变形：
$$a_{n+1} + 2^{n+1} = 3(a_n + 2^n)$$

所以 $a_n + 2^n = 3^{n-1}(a_1 + 2) = 3^n$

即 $a_n = 3^n - 2^n$。

### (II) 证明不等式

**伪等比放缩：**
$$\frac{1}{a_n} = \frac{1}{3^n - 2^n} = \frac{1}{3^n} \cdot \frac{1}{1-(\frac{2}{3})^n} < \frac{1}{3^n} \cdot 3 = \frac{1}{3^{n-1}}$$

**求和：**
$$\sum_{k=1}^{n} \frac{1}{a_k} < 1 + \frac{1}{3} + \frac{1}{3^2} + \cdots = \frac{1}{1-\frac{1}{3}} = \frac{3}{2}$$

---

## 三、分步教学引导

**步骤1：建立递推**
- **引导语**："利用 $a_n = S_n - S_{n-1}$ 建立递推"

**步骤2：求解递推**
- **引导语**："变形为 $a_{n+1} + 2^{n+1} = 3(a_n + 2^n)$"

**步骤3：伪等比放缩**
- **引导语**："证明 $\frac{1}{a_n} < \frac{1}{3^{n-1}}$"
