# 【E_142】奇偶性放缩法

## 一、题目内容

> **原题位置**：《新高考你真的掌握了吗？》P142 例5.27

**题目：**

已知 $S_n = 2a_n + (-1)^n$，证明：$\sum_{k=4}^{m} \frac{1}{a_k} < \frac{7}{8}$。

---

## 二、答案与解析

**步骤1：求通项**
由 $S_n = 2a_n + (-1)^n$ 和 $S_{n-1} = 2a_{n-1} + (-1)^{n-1}$ 相减得：
$$a_n = 2a_n - 2a_{n-1} + (-1)^n - (-1)^{n-1}$$

整理得：$a_n = 2a_{n-1} + 2(-1)^{n-1}$

解得：$a_n = \frac{2}{3}[2^{n-2} - (-1)^n]$（$n \geq 2$）

**步骤2：奇偶分类讨论**

**奇数项：** $a_{2n-1} = \frac{2}{3}(2^{2n-3} + 1)$

**偶数项：** $a_{2n} = \frac{2}{3}(2^{2n-2} - 1)$

**步骤3：配对放缩**
$$\frac{1}{a_{2n-1}} + \frac{1}{a_{2n}} < \frac{3}{2^{2n-2}}$$

**步骤4：等比求和**
$$\sum_{k=4}^{m} \frac{1}{a_k} < \sum_{n=2}^{\infty} \frac{3}{2^{2n-2}} = \frac{3}{4} \cdot \frac{1}{1-\frac{1}{4}} = 1$$

更精确放缩可得 $< \frac{7}{8}$。
