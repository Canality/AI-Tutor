# 【E_127】递推数列与裂项放缩综合题

## 一、题目内容

> **原题位置**：《新高考你真的掌握了吗？》P127 变式5.11.2
> 
> **来源**：2017浙江适应性模考20题

**题目：**

设 $T_n$ 是数列 $\{a_n\}$ 的前 $n$ 项之积，满足 $2T_n = 1 - a_n$（$n \in \mathbb{N}^*$）。

**(I)** 求数列 $\{a_n\}$ 的通项公式；

**(II)** 设 $b_n = \frac{1}{T_1} + \frac{1}{T_2} + \cdots + \frac{1}{T_n}$，求证：$-\frac{5}{4} \leq b_n - n \leq -\frac{3}{4}$。

---

## 二、答案与解析

### (I) 求数列 $\{a_n\}$ 的通项公式

**答案：** $a_n = \frac{2n-1}{2n+1}$

**解析：**

**步骤1：分析已知条件**
$2T_n = 1 - a_n$，其中 $T_n = a_1 \cdot a_2 \cdots a_n$

**步骤2：求 $a_1$**
当 $n=1$ 时，$T_1 = a_1$，所以：
$$2a_1 = 1 - a_1$$
$$3a_1 = 1$$
$$a_1 = \frac{1}{3}$$

**步骤3：建立递推关系**
由 $2T_n = 1 - a_n$ 和 $2T_{n-1} = 1 - a_{n-1}$（$n \geq 2$）

两式相除：
$$\frac{T_n}{T_{n-1}} = \frac{1-a_n}{1-a_{n-1}}$$

即：
$$a_n = \frac{1-a_n}{1-a_{n-1}}$$

**步骤4：求解递推关系**
$$a_n(1-a_{n-1}) = 1-a_n$$
$$a_n - a_n a_{n-1} = 1 - a_n$$
$$2a_n = 1 + a_n a_{n-1}$$
$$a_n = \frac{1}{2 - a_{n-1}}$$

**步骤5：变形求通项**
设 $a_n = \frac{b_n}{b_n+1}$，代入递推关系求解（或用数学归纳法验证）

验证：$a_n = \frac{2n-1}{2n+1}$
- $a_1 = \frac{1}{3}$ ✓
- 假设 $a_{n-1} = \frac{2n-3}{2n-1}$
- 则 $a_n = \frac{1}{2 - \frac{2n-3}{2n-1}} = \frac{2n-1}{2(2n-1)-(2n-3)} = \frac{2n-1}{4n-2-2n+3} = \frac{2n-1}{2n+1}$ ✓

---

### (II) 求证：$-\frac{5}{4} \leq b_n - n \leq -\frac{3}{4}$

**答案：** 证明见解析

**解析：**

**步骤1：求 $\frac{1}{T_n}$**
由 $2T_n = 1 - a_n = 1 - \frac{2n-1}{2n+1} = \frac{2}{2n+1}$

所以 $T_n = \frac{1}{2n+1}$

因此 $\frac{1}{T_n} = 2n+1$

**步骤2：求 $b_n$**
$$b_n = \sum_{k=1}^{n} \frac{1}{T_k} = \sum_{k=1}^{n} (2k+1) = 2\sum_{k=1}^{n} k + \sum_{k=1}^{n} 1 = 2 \cdot \frac{n(n+1)}{2} + n = n(n+1) + n = n^2 + 2n$$

**步骤3：计算 $b_n - n$**
$$b_n - n = n^2 + 2n - n = n^2 + n = n(n+1)$$

**步骤4：验证不等式**
需要证明：$-\frac{5}{4} \leq n(n+1) \leq -\frac{3}{4}$

但 $n(n+1) > 0$ 对所有 $n \geq 1$ 成立，与不等式矛盾。

> **修正**：可能题目条件或表达式理解有误。根据提取内容，可能：
> - $b_n$ 的定义不同
> - 或者不等式方向有误
> - 或者 $T_n$ 的定义是前 $n$ 项和而非积
>
> **请以原书为准**，上述解析基于对提取文本的理解。

**可能的正确理解**：
若 $b_n = \sum_{k=1}^{n} \frac{1}{a_k}$，则计算会不同。

或不等式为 $\frac{3}{4} \leq b_n - n \leq \frac{5}{4}$（正值）

## 三、分步教学引导（用于AI Agent）

### 第(I)问引导

**步骤1：理解 $T_n$ 的定义**
- **引导语**："$T_n$ 是前 $n$ 项之积，那么 $T_n$ 和 $T_{n-1}$ 有什么关系？"
- **期望学生回答**：$T_n = T_{n-1} \cdot a_n$

**步骤2：求首项 $a_1$**
- **引导语**："令 $n=1$，$T_1 = a_1$，代入已知条件"
- **期望学生回答**：$2a_1 = 1 - a_1$，所以 $a_1 = \frac{1}{3}$

**步骤3：建立递推关系**
- **引导语**："写出 $T_n$ 和 $T_{n-1}$ 的表达式，相除得到什么？"
- **期望学生回答**：$a_n = \frac{1-a_n}{1-a_{n-1}}$

**步骤4：化简递推关系**
- **引导语**："整理得到 $a_n$ 关于 $a_{n-1}$ 的表达式"
- **期望学生回答**：$a_n = \frac{1}{2-a_{n-1}}$

**步骤5：求通项公式**
- **引导语**："用数学归纳法或换元法求通项"
- **期望学生回答**：$a_n = \frac{2n-1}{2n+1}$

---

### 第(II)问引导

**步骤1：求 $T_n$ 的表达式**
- **引导语**："利用(I)的结果，求 $T_n$"
- **期望学生回答**：$T_n = \frac{1-a_n}{2} = \frac{1}{2n+1}$

**步骤2：求 $\frac{1}{T_n}$**
- **引导语**："$\frac{1}{T_n} = $？"
- **期望学生回答**：$\frac{1}{T_n} = 2n+1$

**步骤3：求 $b_n$**
- **引导语**："$b_n = \sum_{k=1}^{n} (2k+1)$，计算这个和"
- **期望学生回答**：$b_n = n^2 + 2n$

**步骤4：验证不等式**
- **引导语**："计算 $b_n - n$，验证是否满足不等式"
- **注意**：请以原书为准，核对不等式方向
