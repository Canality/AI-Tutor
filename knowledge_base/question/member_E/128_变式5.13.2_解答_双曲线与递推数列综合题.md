# 【E_128_3】双曲线与递推数列综合题

## 一、题目内容

> **原题位置**：《新高考你真的掌握了吗？》P128 变式5.13.2
> 
> **来源**：变式题

**题目：**

已知双曲线 $C: x^2 - y^2 = 1$。

点 $P$ 在曲线 $C$ 上，点 $A_n(0, a_n)$，$d_n$ 为点 $P$ 到 $A_n$ 的最小距离。

已知 $a_0 = 0$，$a_n = \sqrt{2}d_{n-1}$（$n \in \mathbb{N}^*$）。

**(I)** 求数列 $\{a_n\}$ 的通项公式；

**(II)** 判断是否存在常数 $M$，使得 $\sum_{k=1}^{n} \frac{1}{a_k^3} < M$ 对所有 $n \in \mathbb{N}^*$ 恒成立，并说明理由。

---

## 二、答案与解析

### (I) 求数列 $\{a_n\}$ 的通项公式

**答案：** $a_n = \sqrt{n}$（需根据实际推导验证）

**解析：**

**步骤1：建立距离函数**
设 $P(x, y)$ 在双曲线 $x^2 - y^2 = 1$ 上，则 $y^2 = x^2 - 1$（$|x| \geq 1$）

点 $P$ 到 $A_n(0, a_n)$ 的距离：
$$|PA_n|^2 = x^2 + (y - a_n)^2 = x^2 + y^2 - 2a_ny + a_n^2$$

**步骤2：利用双曲线方程消元**
由 $x^2 = y^2 + 1$，代入得：
$$|PA_n|^2 = 2y^2 + 1 - 2a_ny + a_n^2 = 2y^2 - 2a_ny + (a_n^2 + 1)$$

**步骤3：求最小值**
对 $y$ 求导或用配方法，当 $y = \frac{a_n}{2}$ 时取最小值。

**步骤4：建立递推关系**
$$d_n^2 = 2 \cdot \frac{a_n^2}{4} - 2a_n \cdot \frac{a_n}{2} + a_n^2 + 1 = \frac{a_n^2}{2} - a_n^2 + a_n^2 + 1 = \frac{a_n^2}{2} + 1$$

由 $a_{n+1} = \sqrt{2}d_n$，得 $a_{n+1}^2 = 2d_n^2 = a_n^2 + 2$

**步骤5：求解递推**
$a_n^2 = a_0^2 + 2n = 2n$，所以 $a_n = \sqrt{2n}$

> **注意**：具体推导请以原书为准

---

### (II) 判断是否存在常数 $M$

**答案：** 存在，$M = \frac{\sqrt{2}}{4}$（或其他值，需验证）

**解析：**

**步骤1：求通项**
由 (I) 知 $a_n = \sqrt{2n}$，则 $a_n^3 = (2n)^{3/2} = 2\sqrt{2}n^{3/2}$

**步骤2：放缩求和**
$$\sum_{k=1}^{n} \frac{1}{a_k^3} = \frac{1}{2\sqrt{2}}\sum_{k=1}^{n} \frac{1}{k^{3/2}}$$

**步骤3：利用积分或已知结论**
$$\sum_{k=1}^{n} \frac{1}{k^{3/2}} < 1 + \int_1^{n} \frac{1}{x^{3/2}}dx = 1 + \left[-\frac{2}{\sqrt{x}}\right]_1^n = 1 + 2 - \frac{2}{\sqrt{n}} < 3$$

或直接利用 $\sum_{k=1}^{\infty} \frac{1}{k^{3/2}} = \zeta(3/2) \approx 2.612$

**步骤4：确定 $M$**
$$\sum_{k=1}^{n} \frac{1}{a_k^3} < \frac{3}{2\sqrt{2}} = \frac{3\sqrt{2}}{4}$$

故存在 $M = \frac{3\sqrt{2}}{4}$（或其他更精确的上界）。

## 三、分步教学引导（用于AI Agent）

### 第(I)问引导

**步骤1：理解题意**
- **引导语**："$d_n$ 是什么？怎么求点到双曲线的最小距离？"
- **期望学生回答**：$d_n$ 是 $A_n$ 到双曲线的最短距离

**步骤2：建立距离函数**
- **引导语**："设 $P(x,y)$ 在双曲线上，写出 $|PA_n|^2$"
- **期望学生回答**：利用 $x^2 - y^2 = 1$ 消元

**步骤3：求最小值**
- **引导语**："对 $y$ 求导或用配方法求最小值"
- **期望学生回答**：当 $y = \frac{a_n}{2}$ 时取最小

**步骤4：建立递推**
- **引导语**："利用 $a_{n+1} = \sqrt{2}d_n$ 建立递推关系"
- **期望学生回答**：$a_{n+1}^2 = a_n^2 + 2$

**步骤5：求解通项**
- **引导语**："求解递推关系"
- **期望学生回答**：$a_n = \sqrt{2n}$
