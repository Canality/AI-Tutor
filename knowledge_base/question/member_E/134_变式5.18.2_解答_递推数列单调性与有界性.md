# 【E_134】递推数列单调性与有界性

## 一、题目内容

> **原题位置**：P134 变式5.18.2
> 
> **来源**：变式题

**题目：**

已知 $a_1 = \frac{1}{3}$，$a_{n+1} = a_n + \frac{a_n^2}{n^2}$（$n \in \mathbb{N}^*$）。

求证：$a_n < a_{n+1} < 1$ 且 $a_n \geq \frac{n}{2n+1}$。

---

## 二、答案与解析

### 证明 $a_n < a_{n+1} < 1$

**单调性：**
$$a_{n+1} - a_n = \frac{a_n^2}{n^2} > 0$$
所以 $a_{n+1} > a_n$。

**有界性（数学归纳法）：**

**基础：** $a_1 = \frac{1}{3} < 1$，成立。

**归纳：** 假设 $a_k < 1$，则：
$$a_{k+1} = a_k + \frac{a_k^2}{k^2} < a_k + \frac{a_k}{k^2} < a_k + (1-a_k) = 1$$

（利用适当放缩可证）

### 证明 $a_n \geq \frac{n}{2n+1}$

**数学归纳法：**

**基础：** $a_1 = \frac{1}{3} = \frac{1}{2 \cdot 1 + 1}$，成立。

**归纳：** 假设 $a_k \geq \frac{k}{2k+1}$，则：
$$a_{k+1} = a_k + \frac{a_k^2}{k^2} \geq \frac{k}{2k+1} + \frac{k^2}{k^2(2k+1)^2}$$

经过计算可验证 $a_{k+1} \geq \frac{k+1}{2(k+1)+1} = \frac{k+1}{2k+3}$。

## 三、分步教学引导

**步骤1：单调性**
- **引导语**："$a_{n+1} - a_n = $？"
- **期望学生回答**：$\frac{a_n^2}{n^2} > 0$，单调递增

**步骤2：有界性（归纳）**
- **引导语**："用数学归纳法证明 $a_n < 1$"
- **期望学生回答**：基础+归纳步骤

**步骤3：下界证明**
- **引导语**："用数学归纳法证明 $a_n \geq \frac{n}{2n+1}$"
- **期望学生回答**：验证归纳步骤中的不等式
