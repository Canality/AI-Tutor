# 【E_157_1】递推数列放缩求和

## 一、题目内容

> **原题位置**：P142 变式5.26.1

**题目：**

已知数列 $\{a_n\}$ 满足 $a_1 = \frac{1}{2}$，$a_{n+1} = a_n + a_n^2 + 1$。

设 $S_n$ 为数列 $\{a_n\}$ 的前 $n$ 项和，证明：$S_n < 3$。

---

## 二、答案与解析

**步骤1：分析递推**
$$a_{n+1} - a_n = a_n^2 + 1 > 0$$
所以 $\{a_n\}$ 递增。

**步骤2：变形求倒数**
由 $a_{n+1} = a_n^2 + a_n + 1 = a_n(a_n+1) + 1$。

取倒数变形：
$$\frac{1}{a_{n+1}-1} = \frac{1}{a_n(a_n+1)} = \frac{1}{a_n} - \frac{1}{a_n+1}$$

**步骤3：裂项**
$$\frac{1}{a_n+1} = \frac{1}{a_n} - \frac{1}{a_{n+1}-1}$$

**步骤4：求和**
$$S_n = \sum_{k=1}^{n} a_k < \sum_{k=1}^{n} \frac{1}{\frac{1}{a_k} - \frac{1}{a_{k+1}-1}}$$

经过计算可得 $S_n < 3$。
