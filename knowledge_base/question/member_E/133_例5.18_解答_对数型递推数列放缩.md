# 【E_133】对数型递推数列放缩

## 一、题目内容

> **原题位置**：P133 例5.18
> 
> **来源**：经典例题

**题目：**

已知 $a_1 = 1$，$a_{n+1} = \left(1 + \frac{1}{n^2+n}\right)a_n + \frac{1}{2^n}$（$n \in \mathbb{N}^*$）。

**(I)** 用数学归纳法证明：$a_n \geq 2$（$n \geq 2$）；

**(II)** 利用不等式 $\ln(x+1) < x$（$x > 0$）证明：$a_n < e^2$。

---

## 二、答案与解析

### (I) 用数学归纳法证明 $a_n \geq 2$（$n \geq 2$）

**证明：**

**基础步骤：**
当 $n=2$ 时：
$$a_2 = \left(1 + \frac{1}{2}\right)a_1 + \frac{1}{2} = \frac{3}{2} + \frac{1}{2} = 2 \geq 2$$
成立。

**归纳步骤：**
假设 $a_k \geq 2$（$k \geq 2$），则：
$$a_{k+1} = \left(1 + \frac{1}{k^2+k}\right)a_k + \frac{1}{2^k} > a_k \geq 2$$

所以 $a_{k+1} \geq 2$。

由数学归纳法，$a_n \geq 2$ 对所有 $n \geq 2$ 成立。

### (II) 利用 $\ln(x+1) < x$ 证明 $a_n < e^2$

**证明：**

**步骤1：取对数**
$$\ln a_{n+1} = \ln\left[\left(1 + \frac{1}{n^2+n}\right)a_n + \frac{1}{2^n}\right]$$

**步骤2：放缩**
由于 $a_n \geq 2$，有：
$$\ln a_{n+1} < \ln\left[\left(1 + \frac{1}{n^2+n}\right)a_n + \frac{a_n}{2^{n+1}}\right]$$

$$= \ln a_n + \ln\left(1 + \frac{1}{n^2+n} + \frac{1}{2^{n+1}}\right)$$

**步骤3：利用 $\ln(1+x) < x$**
$$\ln a_{n+1} - \ln a_n < \frac{1}{n^2+n} + \frac{1}{2^{n+1}} = \frac{1}{n(n+1)} + \frac{1}{2^{n+1}}$$

**步骤4：累加求和**
$$\ln a_n - \ln a_1 < \sum_{k=1}^{n-1}\frac{1}{k(k+1)} + \sum_{k=1}^{n-1}\frac{1}{2^{k+1}}$$

$$= \left(1 - \frac{1}{n}\right) + \frac{1}{2}\left(1 - \frac{1}{2^{n-1}}\right) < 1 + \frac{1}{2} = \frac{3}{2}$$

**步骤5：完成证明**
$$\ln a_n < \ln a_1 + \frac{3}{2} = 0 + \frac{3}{2} = \frac{3}{2}$$

所以 $a_n < e^{3/2} < e^2$。

**证毕。**

## 三、分步教学引导（用于AI Agent）

### 第(I)问引导

**步骤1：验证基础**
- **引导语**："计算 $a_2$，验证是否 $\geq 2$"
- **期望学生回答**：$a_2 = 2 \geq 2$，成立

**步骤2：归纳假设**
- **引导语**："假设 $a_k \geq 2$，证明 $a_{k+1} \geq 2$"
- **期望学生回答**：$a_{k+1} > a_k \geq 2$

**步骤3：完成归纳**
- **引导语**："由数学归纳法得证"

### 第(II)问引导

**步骤4：取对数**
- **引导语**："对递推式取对数，利用 $\ln(1+x) < x$"
- **期望学生回答**：$\ln a_{n+1} - \ln a_n < \frac{1}{n(n+1)} + \frac{1}{2^{n+1}}$

**步骤5：累加**
- **引导语**："累加后两个求和分别计算"
- **期望学生回答**：裂项求和 + 等比求和

**步骤6：完成证明**
- **引导语**："得到 $\ln a_n < \frac{3}{2}$，所以 $a_n < e^{3/2} < e^2$"
