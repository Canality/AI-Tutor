# 【E_141】分式型递推数列放缩

## 一、题目内容

> **原题位置**：P141 例5.26
> 
> **来源**：2016杭州模考

**题目：**

已知 $a_1 = 1$，$a_{n+1} = a_n + \frac{1}{a_n}$。

**(I)** 证明：$2 < a_{n+1}^2 - a_n^2 \leq 3$；

**(II)** 证明：$\frac{3n-1}{3n-2} \leq \frac{a_{n+1}}{a_n} < \frac{2n}{2n-1}$。

---

## 二、答案与解析

### (I) 证明 $2 < a_{n+1}^2 - a_n^2 \leq 3$

**计算：**
$$a_{n+1}^2 - a_n^2 = (a_{n+1}+a_n)(a_{n+1}-a_n) = (2a_n + \frac{1}{a_n}) \cdot \frac{1}{a_n} = 2 + \frac{1}{a_n^2}$$

由 $a_n \geq 1$（归纳可证），得 $2 < 2 + \frac{1}{a_n^2} \leq 3$。

### (II) 证明双向不等式

利用 (I) 的结果累加，结合适当放缩可证。
