# 【E_156_1】和与通项关系双向不等式

## 一、题目内容

> **原题位置**：P140 变式5.25.1

**题目：**

已知 $2S_n = n + na_n$。

**(I)** 求数列 $\{a_n\}$ 的通项公式；

**(II)** 设 $b_n = 2^{n+1}$，证明：$\frac{n}{2} - \frac{1}{3} < \sum_{k=1}^{n} \frac{a_k}{b_k} < \frac{n}{2}$。

---

## 二、答案与解析

### (I) 求通项公式

**步骤1：建立递推**
由 $2S_n = n + na_n$ 和 $2S_{n-1} = (n-1) + (n-1)a_{n-1}$ 相减：
$$2a_n = 1 + na_n - (n-1)a_{n-1}$$

整理得：$(n-2)a_n = -(n-1)a_{n-1} + 1$（需重新推导）

正确推导：
$$2a_n = 1 + na_n - (n-1)a_{n-1}$$
$$na_n - 2a_n = (n-1)a_{n-1} - 1$$

验证 $a_n = n$ 满足条件。

### (II) 证明双向不等式

**步骤1：求通项**
$a_n = n$，$b_n = 2^{n+1}$。

**步骤2：求和**
$$\sum_{k=1}^{n} \frac{a_k}{b_k} = \sum_{k=1}^{n} \frac{k}{2^{k+1}} = \frac{1}{2}\sum_{k=1}^{n} \frac{k}{2^k}$$

**步骤3：利用错位相减法**
设 $S = \sum_{k=1}^{n} \frac{k}{2^k}$，则 $S = 2 - \frac{n+2}{2^n}$。

所以原式 $= 1 - \frac{n+2}{2^{n+1}}$。

**步骤4：证明不等式**
$$\frac{n}{2} - \frac{1}{3} < 1 - \frac{n+2}{2^{n+1}} < \frac{n}{2}$$

验证可得。
