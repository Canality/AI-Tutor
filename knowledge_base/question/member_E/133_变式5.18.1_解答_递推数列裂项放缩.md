# 【E_152_2】递推数列裂项放缩

## 一、题目内容

> **原题位置**：《新高考你真的掌握了吗？》P133 变式5.18.1

**题目：**

已知数列 $\{a_n\}$ 满足 $a_1 = 2$，$a_{n+1} = a_n^2 + a_n$。

证明：$\sum_{k=1}^{n} \frac{1}{a_k+1} < \frac{1}{2}$。

---

## 二、答案与解析

**步骤1：变形递推式**
由 $a_{n+1} = a_n(a_n+1)$，取倒数：
$$\frac{1}{a_{n+1}} = \frac{1}{a_n(a_n+1)} = \frac{1}{a_n} - \frac{1}{a_n+1}$$

**步骤2：裂项**
$$\frac{1}{a_n+1} = \frac{1}{a_n} - \frac{1}{a_{n+1}}$$

**步骤3：求和**
$$\sum_{k=1}^{n} \frac{1}{a_k+1} = \sum_{k=1}^{n}\left(\frac{1}{a_k} - \frac{1}{a_{k+1}}\right) = \frac{1}{a_1} - \frac{1}{a_{n+1}}$$

**步骤4：完成证明**
$$= \frac{1}{2} - \frac{1}{a_{n+1}} < \frac{1}{2}$$
