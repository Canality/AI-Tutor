# 【E_143】含(-1)^n的递推数列

## 一、题目内容

> **原题位置**：《新高考你真的掌握了吗？》P143 例5.28

**题目：**

已知 $a_1 = 1$，$a_{n+1} = 2a_n + (-1)^{n+1}$。

**(I)** 求通项公式；

**(II)** 证明：$\sum_{k=1}^{n} \frac{1}{a_k} < \frac{7}{4}$。

---

## 二、答案与解析

### (I) 求通项公式

**待定系数法：**

设 $a_n + c(-1)^n = 2[a_{n-1} + c(-1)^{n-1}]$

解得 $c = \frac{1}{3}$，所以：
$$a_n + \frac{1}{3}(-1)^n = 2^{n-1}(a_1 + \frac{1}{3}) = \frac{4}{3} \cdot 2^{n-1} = \frac{2^{n+1}}{3}$$

即：$a_n = \frac{2^{n+1} - (-1)^{n+1}}{3}$

### (II) 证明不等式

**分类讨论：**

**当 $n = 2m-1$（奇数）：**
$$\sum_{k=1}^{2m-1} \frac{1}{a_k} = \sum_{k=1}^{m}\frac{1}{a_{2k-1}} + \sum_{k=1}^{m-1}\frac{1}{a_{2k}}$$

计算得 $< \frac{7}{4}$

**当 $n = 2m$（偶数）：**
类似计算得 $< \frac{7}{4}$

- **引导语**："设 $a_n + c(-1)^n$ 为等比数列"

- **引导语**："确定 $c$ 的值，写出通项"

**步骤3：分类讨论**
- **引导语**："分 $n$ 为奇数和偶数两种情况"

**步骤4：完成证明**
- **引导语**："分别计算两种情况下的和"
