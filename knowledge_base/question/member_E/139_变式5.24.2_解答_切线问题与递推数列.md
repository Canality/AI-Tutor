# 【E_155_2】切线问题与递推数列

## 一、题目内容

> **原题位置**：《新高考你真的掌握了吗？》P139 变式5.24.2

**题目：**

已知 $f(x) = x^2 - 4$，过曲线上一点 $(x_n, f(x_n))$ 的切线与 $x$ 轴交于 $(x_{n+1}, 0)$。

**(I)** 用 $x_n$ 表示 $x_{n+1}$；

**(II)** 若 $x_1 = 4$，记 $a_n = \lg\frac{x_n+2}{x_n-2}$，证明 $\{a_n\}$ 是等比数列，并求 $x_n$；

**(III)** 设 $b_n = x_n - 2$，$T_n = \sum_{k=1}^{n} b_k$，证明 $T_n < 3$。

---

## 二、答案与解析

### (I) 求递推关系

**步骤1：求切线方程**
$$f'(x) = 2x$$
切线：$y - (x_n^2-4) = 2x_n(x - x_n)$

**步骤2：求与x轴交点**
令 $y=0$：$-(x_n^2-4) = 2x_n(x_{n+1}-x_n)$

解得：$x_{n+1} = \frac{x_n}{2} + \frac{2}{x_n}$

### (II) 证明等比数列并求通项

**步骤1：变形递推**
$$x_{n+1} + 2 = \frac{(x_n+2)^2}{2x_n}$$
$$x_{n+1} - 2 = \frac{(x_n-2)^2}{2x_n}$$

**步骤2：求比值**
$$\frac{x_{n+1}+2}{x_{n+1}-2} = \left(\frac{x_n+2}{x_n-2}\right)^2$$

**步骤3：取对数**
$$a_{n+1} = \lg\frac{x_{n+1}+2}{x_{n+1}-2} = 2\lg\frac{x_n+2}{x_n-2} = 2a_n$$

所以 $\{a_n\}$ 是等比数列，公比为 2。

**步骤4：求 $x_n$**
$$a_n = a_1 \cdot 2^{n-1} = \lg 3 \cdot 2^{n-1}$$

解得：$x_n = \frac{2(3^{2^{n-1}}+1)}{3^{2^{n-1}}-1}$

### (III) 证明 $T_n < 3$

**步骤1：求 $b_n$**
$$b_n = x_n - 2 = \frac{4}{3^{2^{n-1}}-1}$$

**步骤2：放缩**
$$b_n = \frac{4}{3^{2^{n-1}}-1} < \frac{4}{3^{2^{n-1}} - 3^{2^{n-2}}} = \frac{4}{3^{2^{n-2}}(3^{2^{n-2}}-1)}$$

当 $n \geq 2$ 时，$b_n < \frac{1}{3^{n-1}}$。

**步骤3：求和**
$$T_n = b_1 + \sum_{k=2}^{n} b_k < 2 + \sum_{k=2}^{n}\frac{1}{3^{k-1}} < 2 + \frac{1/3}{1-1/3} = 2 + \frac{1}{2} = \frac{5}{2} < 3$$
