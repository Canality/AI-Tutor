# 【E_128】立方型裂项放缩证明

## 一、题目内容

> **原题位置**：《新高考你真的掌握了吗？》P128 例5.12
> 
> **来源**：经典例题

**题目：**

求证：$\sum_{k=1}^{n} \frac{1}{k^3} < \frac{29}{24}$

---

## 二、答案与解析

**答案：** 证明见解析

**解析：**

**步骤1：分析通项**
需要证明 $\sum_{k=1}^{n} \frac{1}{k^3} < \frac{29}{24}$

**步骤2：保留前两项，对 $k \geq 3$ 放缩**
$$\sum_{k=1}^{n} \frac{1}{k^3} = 1 + \frac{1}{8} + \sum_{k=3}^{n} \frac{1}{k^3} = \frac{9}{8} + \sum_{k=3}^{n} \frac{1}{k^3}$$

**步骤3：立方型放缩**
对于 $k \geq 3$，利用立方差公式：
$$k^3 - k = k(k^2-1) = k(k-1)(k+1)$$

所以：
$$\frac{1}{k^3} < \frac{1}{k^3 - k} = \frac{1}{k(k-1)(k+1)}$$

**步骤4：裂项**
$$\frac{1}{k(k-1)(k+1)} = \frac{1}{2}\left[\frac{1}{(k-1)k} - \frac{1}{k(k+1)}\right]$$

**验证：**
$$\frac{1}{(k-1)k} - \frac{1}{k(k+1)} = \frac{k+1 - (k-1)}{k(k-1)(k+1)} = \frac{2}{k(k-1)(k+1)}$$

所以：
$$\frac{1}{k(k-1)(k+1)} = \frac{1}{2}\left[\frac{1}{(k-1)k} - \frac{1}{k(k+1)}\right]$$

**步骤5：求和**
$$\sum_{k=3}^{n} \frac{1}{k^3} < \frac{1}{2}\sum_{k=3}^{n} \left[\frac{1}{(k-1)k} - \frac{1}{k(k+1)}\right]$$

**步骤6：裂项相消**
$$= \frac{1}{2}\left[\left(\frac{1}{2 \cdot 3} - \frac{1}{3 \cdot 4}\right) + \left(\frac{1}{3 \cdot 4} - \frac{1}{4 \cdot 5}\right) + \cdots + \left(\frac{1}{(n-1)n} - \frac{1}{n(n+1)}\right)\right]$$

$$= \frac{1}{2}\left[\frac{1}{6} - \frac{1}{n(n+1)}\right] < \frac{1}{2} \cdot \frac{1}{6} = \frac{1}{12}$$

**步骤7：完成证明**
$$\sum_{k=1}^{n} \frac{1}{k^3} < \frac{9}{8} + \frac{1}{12} = \frac{27}{24} + \frac{2}{24} = \frac{29}{24}$$

**证毕。**

## 三、分步教学引导（用于AI Agent）

**步骤1：分析难点**
- **引导语**："$\frac{1}{k^3}$ 比 $\frac{1}{k^2}$ 更难处理，直接放缩成 $\frac{1}{k(k-1)}$ 可以吗？"
- **关键提示**：需要保留前两项，从 $k=3$ 开始放缩
- **期望学生回答**：先计算前几项，对后面的项进行放缩

**步骤2：计算前几项**
- **引导语**："计算 $k=1,2$ 时的值"
- **期望学生回答**：$1 + \frac{1}{8} = \frac{9}{8}$

**步骤3：寻找放缩方式**
- **引导语**："对于 $k \geq 3$，怎么放缩 $\frac{1}{k^3}$？"
- **关键提示**：让分母可以因式分解
- **期望学生回答**：$\frac{1}{k^3} < \frac{1}{k^3 - k} = \frac{1}{k(k-1)(k+1)}$

**步骤4：裂项变形**
- **引导语**："$\frac{1}{k(k-1)(k+1)}$ 怎么裂项？"
- **关键提示**：拆成两个分式之差
- **期望学生回答**：$\frac{1}{2}\left[\frac{1}{(k-1)k} - \frac{1}{k(k+1)}\right]$

**步骤5：求和放缩**
- **引导语**："裂项相消后，剩余哪些项？"
- **期望学生回答**：只剩 $\frac{1}{6} - \frac{1}{n(n+1)} < \frac{1}{6}$

**步骤6：完成证明**
- **引导语**："加上前面的 $\frac{9}{8}$，验证是否小于 $\frac{29}{24}$"
- **期望学生回答**：$\frac{9}{8} + \frac{1}{12} = \frac{29}{24}$

## 五、关联拓展

### 相关公式
**立方差公式：**
- $a^3 - b^3 = (a-b)(a^2 + ab + b^2)$
- $k^3 - k = k(k^2-1) = k(k-1)(k+1)$

**裂项公式：**
- $\frac{1}{k(k-1)(k+1)} = \frac{1}{2}\left[\frac{1}{(k-1)k} - \frac{1}{k(k+1)}\right]$

**更精确的结论：**
- $\sum_{k=1}^{\infty} \frac{1}{k^3} = \zeta(3) \approx 1.202$（Apéry常数）
