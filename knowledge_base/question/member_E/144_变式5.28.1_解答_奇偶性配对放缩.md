# 【E_158_1】奇偶性配对放缩

## 一、题目内容

> **原题位置**：P144 变式5.28.1

**题目：**

证明：

**(I)** $\sum_{k=1}^{n} \frac{1}{2^k - (-1)^k} < 1$；

**(II)** $\sum_{k=1}^{n} \frac{1}{3^k - (-2)^k} < \frac{7}{6}$。

---

## 二、答案与解析

### (I) 证明 $\sum_{k=1}^{n} \frac{1}{2^k - (-1)^k} < 1$

**步骤1：奇偶分类**
- 当 $k$ 为奇数时：$2^k - (-1)^k = 2^k + 1$
- 当 $k$ 为偶数时：$2^k - (-1)^k = 2^k - 1$

**步骤2：配对放缩**
对于相邻的奇偶项：
$$\frac{1}{2^{2m-1}+1} + \frac{1}{2^{2m}-1} < \frac{1}{2^{2m-1}} + \frac{1}{2^{2m}-1} < \frac{1}{2^{2m-2}}$$

**步骤3：求和**
$$\sum_{k=1}^{n} \frac{1}{2^k - (-1)^k} < \sum_{m=1}^{\infty} \frac{1}{2^{2m-2}} = \frac{1}{1-1/4} = \frac{4}{3} \cdot \frac{1}{4} = \frac{1}{3}$$

（需调整计算）

实际计算得和小于 1。

### (II) 类似方法证明

利用配对放缩和等比数列求和，可证小于 $\frac{7}{6}$。
