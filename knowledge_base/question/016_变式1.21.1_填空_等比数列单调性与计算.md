# 【A_016_2】等比数列单调性与计算

## 一、题目内容

**题目：**

已知等比数列 $\{a_n\}$ 为单调递增数列，设其前 $n$ 项和为 $S_n$，若 $a_2 = 2$，$S_3 = 7$，则 $a_5$ 的值为 ______。

## 二、答案与解析

### 答案

$16$

### 解析

**步骤1：建立方程**

设首项为 $a_1$，公比为 $q$。

$$a_2 = a_1 q = 2$$

$$S_3 = a_1(1 + q + q^2) = 7$$

**步骤2：求解**

由 $a_1 = \frac{2}{q}$，代入：
$$\frac{2}{q}(1 + q + q^2) = 7$$
$$2(1 + q + q^2) = 7q$$
$$2q^2 - 5q + 2 = 0$$
$$(2q - 1)(q - 2) = 0$$

解得 $q = 2$ 或 $q = \frac{1}{2}$。

**步骤3：利用单调性确定 $q$**

若 $q = \frac{1}{2}$，则 $a_1 = 4$，数列为 $4, 2, 1, \ldots$，递减，舍去。

若 $q = 2$，则 $a_1 = 1$，数列为 $1, 2, 4, \ldots$，递增，符合。

**步骤4：计算 $a_5$**

$$a_5 = a_1 q^4 = 1 \cdot 16 = 16$$

## 三、分步教学引导

**步骤1：列方程**
- **引导语**："根据 $a_2 = 2$ 和 $S_3 = 7$，写出两个方程。"
- **期望学生回答**：$a_1 q = 2$，$a_1(1+q+q^2) = 7$

**步骤2：消元求解**
- **引导语**："消去 $a_1$，得到关于 $q$ 的方程。"
- **期望学生回答**：$2q^2 - 5q + 2 = 0$，$q = 2$ 或 $q = \frac{1}{2}$

**步骤3：利用单调性筛选**
- **引导语**："哪种情况下数列递增？"
- **期望学生回答**：$q = 2$（$q &gt; 1$ 且 $a_1 &gt; 0$ 时递增）

**步骤4：求 $a_5$**
- **期望学生回答**：$16$
