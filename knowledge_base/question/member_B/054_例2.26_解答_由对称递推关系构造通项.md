## 一、题目内容

> **原题位置**
> 《新高考你真的掌握了吗？·数列与不等式》 P54 例2.26

**来源**
(无明确来源，教材例题)

**题目：**
数列 $\{a_n\}$ 满足 $a_1 = 0$，$a_2 = 2$，对任意 $m, n$ 都有：
$$
a_{2m-1} + a_{2n-1} = 2a_{m+n-1} + 2(m-n)^2
$$
求 $\{a_n\}$ 的通项公式。

## 二、答案与解析

### 答案
数列 $\{a_n\}$ 的通项公式为 $a_n = n^2 - n$ ($n \in \mathbb{N}^*$)。

### 解析

已知：
$$
a_{2m-1} + a_{2n-1} = 2a_{m+n-1} + 2(m-n)^2
$$

令 $m = 2$, $n = 1$，代入得：
$$
a_3 + a_1 = 2a_2 + 2(2-1)^2 \Rightarrow a_3 + 0 = 2 \cdot 2 + 2 \cdot 1 = 4 + 2 = 6 \Rightarrow a_3 = 6
$$

令 $m = n+2$，则：
$$
a_{2(n+2)-1} + a_{2n-1} = 2a_{(n+2)+n-1} + 2((n+2)-n)^2 \\
a_{2n+3} + a_{2n-1} = 2a_{2n+1} + 2 \cdot 2^2 = 2a_{2n+1} + 8
$$

移项得：
$$
a_{2n+3} - a_{2n+1} = 2a_{2n+1} - a_{2n-1} + 8
$$

即：
$$
a_{2n+3} - a_{2n+1} = (a_{2n+1} - a_{2n-1}) + 8
$$

令 $b_n = a_{2n+1} - a_{2n-1}$，则：
$$
b_{n+1} = b_n + 8
$$

这是一个公差为 $8$ 的等差数列。

首项 $b_1 = a_3 - a_1 = 6 - 0 = 6$

所以：
$$
b_n = 6 + (n-1) \cdot 8 = 8n - 2
$$

即：
$$
a_{2n+1} - a_{2n-1} = 8n - 2
$$

累加得：
$$
a_{2n+1} = a_1 + \sum_{k=1}^{n} (8k - 2) = 0 + 8 \cdot \frac{n(n+1)}{2} - 2n = 4n(n+1) - 2n = 4n^2 + 4n - 2n = 4n^2 + 2n
$$

所以：
$$
a_{2n+1} = 4n^2 + 2n
$$

令 $m = 1$，代入原式：
$$
a_{2n-1} + a_1 = 2a_{n} + 2(1-n)^2 \Rightarrow a_{2n-1} = 2a_n + 2(n-1)^2
$$

又因为 $a_{2n-1} = 4(n-1)^2 + 2(n-1) = 4(n^2 - 2n + 1) + 2n - 2 = 4n^2 - 8n + 4 + 2n - 2 = 4n^2 - 6n + 2$

代入上式：
$$
4n^2 - 6n + 2 = 2a_n + 2(n-1)^2 = 2a_n + 2(n^2 - 2n + 1) = 2a_n + 2n^2 - 4n + 2
$$

解得：
$$
2a_n = 4n^2 - 6n + 2 - (2n^2 - 4n + 2) = 2n^2 - 2n \Rightarrow a_n = n^2 - n
$$

故数列 $\{a_n\}$ 的通项公式为 $a_n = n^2 - n$ ($n \in \mathbb{N}^*$)。

## 三、分步教学引导

### 步骤1：理解题意
- **引导语**："题目给出了一个数列 $\{a_n\}$，其中 $a_1 = 0$，$a_2 = 2$，以及一个关于奇数下标的对称递推关系。如何求通项公式？"
- **关键提示**：选取特殊值。
- **期望学生回答**：我可以令 $m = 2$, $n = 1$，算出 $a_3$。

### 步骤2：构造新数列
- **引导语**："现在我们有 $a_{2m-1} + a_{2n-1} = 2a_{m+n-1} + 2(m-n)^2$。如何化简？"
- **关键提示**：令 $m = n+2$。
- **期望学生回答**：我可以令 $m = n+2$，得到 $a_{2n+3} + a_{2n-1} = 2a_{2n+1} + 8$。

### 步骤3：再次构造
- **引导语**："现在我们有 $a_{2n+3} - a_{2n+1} = a_{2n+1} - a_{2n-1} + 8$。如何继续？"
- **关键提示**：令 $b_n = a_{2n+1} - a_{2n-1}$。
- **期望学生回答**：我可以令 $b_n = a_{2n+1} - a_{2n-1}$，那么 $b_{n+1} = b_n + 8$。

### 步骤4：求出通项
- **引导语**："现在我们知道 $b_n = 8n - 2$，如何求出 $a_n$？"
- **关键提示**：累加法。
- **期望学生回答**：我可以从 $n=1$ 到 $n$ 累加，得到 $a_{2n+1} = 4n^2 + 2n$。

### 步骤5：还原到原数列
- **引导语**："现在我们有 $a_{2n-1} = 4(n-1)^2 + 2(n-1)$，如何得到 $a_n$？"
- **关键提示**：代入原式。
- **期望学生回答**：我可以将 $a_{2n-1}$ 代入 $a_{2n-1} = 2a_n + 2(n-1)^2$，解出 $a_n = n^2 - n$。
