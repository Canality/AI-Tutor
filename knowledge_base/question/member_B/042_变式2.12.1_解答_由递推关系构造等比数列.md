## 一、题目内容

> **原题位置**
> ·数列与不等式 P42 变式2.12.1

**来源**
(无明确来源，教材例题)

**题目：**
在数列 $\{a_n\}$ 中，$a_1 = 1$，$a_{n+1} = \left(1 + \frac{1}{n}\right)a_n + \frac{n+1}{2^n}$。求数列 $\{a_n\}$ 的通项公式。

## 二、答案与解析

### 答案
$\{a_n\}$ 的通项公式为 $a_n = 2n - \frac{n}{2^{n-1}}$ ($n \in \mathbb{N}^*$)。

### 解析

**第一步：化简递推关系。**

已知：
$$
a_{n+1} = \left(1 + \frac{1}{n}\right)a_n + \frac{n+1}{2^n} = \frac{n+1}{n}a_n + \frac{n+1}{2^n}
$$

为了消除系数 $\frac{n+1}{n}$，我们等式两边同时除以 $n+1$：
$$
\frac{a_{n+1}}{n+1} = \frac{a_n}{n} + \frac{1}{2^n}
$$

**第二步：构造新数列。**

令 $b_n = \frac{a_n}{n}$，则：
$$
b_{n+1} = b_n + \frac{1}{2^n}
$$

这是一个累加型递推关系。

**第三步：求解 $b_n$ 的通项公式。**

从 $b_1 = \frac{a_1}{1} = 1$ 开始，利用累加法求通项：
$$
b_n = b_1 + \sum_{k=1}^{n-1} \frac{1}{2^k} = 1 + \left(\frac{1}{2} + \frac{1}{2^2} + \cdots + \frac{1}{2^{n-1}}\right)
$$

括号内是一个首项为 $\frac{1}{2}$，公比为 $\frac{1}{2}$ 的等比数列前 $n-1$ 项和：
$$
\sum_{k=1}^{n-1} \frac{1}{2^k} = \frac{\frac{1}{2}(1 - (\frac{1}{2})^{n-1})}{1 - \frac{1}{2}} = \frac{\frac{1}{2}(1 - \frac{1}{2^{n-1}})}{\frac{1}{2}} = 1 - \frac{1}{2^{n-1}}
$$

因此：
$$
b_n = 1 + \left(1 - \frac{1}{2^{n-1}}\right) = 2 - \frac{1}{2^{n-1}}
$$

**第四步：还原到原数列。**

由于 $b_n = \frac{a_n}{n}$，所以：
$$
a_n = n \cdot b_n = n \left(2 - \frac{1}{2^{n-1}}\right) = 2n - \frac{n}{2^{n-1}}
$$

**第五步：验证。**

- $n=1$: $a_1 = 2 \cdot 1 - \frac{1}{2^{0}} = 2 - 1 = 1$，符合。
- $n=2$: $a_2 = 2 \cdot 2 - \frac{2}{2^{1}} = 4 - 1 = 3$。
  由递推关系：$a_2 = \left(1 + \frac{1}{1}\right)a_1 + \frac{2}{2^1} = 2 \cdot 1 + 1 = 3$，符合。
- $n=3$: $a_3 = 2 \cdot 3 - \frac{3}{2^{2}} = 6 - \frac{3}{4} = \frac{21}{4}$。
  由递推关系：$a_3 = \left(1 + \frac{1}{2}\right)a_2 + \frac{3}{2^2} = \frac{3}{2} \cdot 3 + \frac{3}{4} = \frac{9}{2} + \frac{3}{4} = \frac{21}{4}$，符合。

综上所述，数列 $\{a_n\}$ 的通项公式为 $a_n = 2n - \frac{n}{2^{n-1}}$。

## 三、分步教学引导

### 步骤1：化简递推关系
- **引导语**："我们手头的递推关系是 $a_{n+1} = \frac{n+1}{n}a_n + \frac{n+1}{2^n}$。为了简化它，我们应该怎么做？"
- **关键提示**：将等式两边同时除以 $n+1$。
- **期望学生回答**：我可以将两边都除以 $n+1$，得到 $\frac{a_{n+1}}{n+1} = \frac{a_n}{n} + \frac{1}{2^n}$。

### 步骤2：构造新数列
- **引导语**："现在我们有了 $\frac{a_{n+1}}{n+1} = \frac{a_n}{n} + \frac{1}{2^n}$。这提示我们构造什么样的新数列？"
- **关键提示**：令 $b_n = \frac{a_n}{n}$。
- **期望学生回答**：我应该令 $b_n = \frac{a_n}{n}$，这样递推关系就变成了 $b_{n+1} = b_n + \frac{1}{2^n}$。

### 步骤3：求解新数列
- **引导语**："我们有 $b_{n+1} = b_n + \frac{1}{2^n}$ 和 $b_1 = 1$。这种类型的递推关系怎么求通项？"
- **关键提示**：这是累加型递推关系。
- **期望学生回答**：我可以使用累加法，$b_n = b_1 + \sum_{k=1}^{n-1} \frac{1}{2^k}$。

### 步骤4：计算等比数列和
- **引导语**："现在我要计算 $b_n = 1 + \sum_{k=1}^{n-1} \frac{1}{2^k}$。这个和是多少？"
- **关键提示**：这是一个首项为 $\frac{1}{2}$，公比为 $\frac{1}{2}$ 的等比数列的和。
- **期望学生回答**：$\sum_{k=1}^{n-1} \frac{1}{2^k} = \frac{\frac{1}{2}(1 - (\frac{1}{2})^{n-1})}{1 - \frac{1}{2}} = 1 - \frac{1}{2^{n-1}}$。

### 步骤5：求出 $b_n$ 并还原
- **引导语**："我已经算出 $\sum_{k=1}^{n-1} \frac{1}{2^k} = 1 - \frac{1}{2^{n-1}}$。那么 $b_n$ 是多少？"
- **关键提示**：$b_n = 1 + (1 - \frac{1}{2^{n-1}})$。
- **期望学生回答**：$b_n = 2 - \frac{1}{2^{n-1}}$。

### 步骤6：还原到原数列
- **引导语**："我们已经求出了 $b_n = 2 - \frac{1}{2^{n-1}}$，而 $b_n = \frac{a_n}{n}$。你能从中解出 $a_n$ 吗？"
- **关键提示**：移项即可。
- **期望学生回答**：$a_n = n \cdot b_n = n \left(2 - \frac{1}{2^{n-1}}\right) = 2n - \frac{n}{2^{n-1}}$。

## 七、备注

- **知识点应用难点**：本题的关键在于能够识别出递推关系的特点，并通过适当的变量替换（令 $b_n = \frac{a_n}{n}$）将其转化为易于处理的累加型递推关系。
- **易错点提醒**：在构造新数列时，要注意选择合适的变换，确保变形后能消去常数项。在验证时，要特别注意递推关系中的下标 $n$。
- **解题技巧总结**：对于形如 $a_{n+1} = \frac{n+k}{n}a_n + g(n)$ 的递推关系，常常可以通过令 $b_n = \frac{a_n}{n}$ 来构造新数列。
