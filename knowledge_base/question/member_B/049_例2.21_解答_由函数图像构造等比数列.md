## 一、题目内容

> **原题位置**
> ·数列与不等式 P49 例2.21

**来源**
(2018 湖南六校联考理 17)

**题目：**
已知函数 $f(x) = x^2 + x - \frac{1}{4}$，数列 $\{a_n\}$ 的首项 $a_1 = \frac{3}{2}$，点 $(a_n, a_{n+1})$ 在函数 $f(x)$ 的图像上，其中 $n \geq 1$，$n \in \mathbb{N}^*$。证明：数列 $\left\{\lg\left(a_n + \frac{1}{2}\right)\right\}$ 是等比数列，并求出 $\{a_n\}$ 的通项公式。


## 二、答案与解析

### 答案
数列 $\left\{\lg\left(a_n + \frac{1}{2}\right)\right\}$ 是以 $\lg 2$ 为首项，$2$ 为公比的等比数列。  
数列 $\{a_n\}$ 的通项公式为 $a_n = 2^{2^{n-1}} - \frac{1}{2}$ ($n \in \mathbb{N}^*$)。

### 解析

因为点 $(a_n, a_{n+1})$ 在函数 $f(x)$ 的图像上，所以：
$$
a_{n+1} = f(a_n) = a_n^2 + a_n - \frac{1}{4}
$$

要证明数列 $\left\{\lg\left(a_n + \frac{1}{2}\right)\right\}$ 是等比数列，只需验证其相邻两项的比值为常数。

计算：
$$
\frac{\lg\left(a_{n+1} + \frac{1}{2}\right)}{\lg\left(a_n + \frac{1}{2}\right)} = \frac{\lg\left(a_n^2 + a_n - \frac{1}{4} + \frac{1}{2}\right)}{\lg\left(a_n + \frac{1}{2}\right)} = \frac{\lg\left(a_n^2 + a_n + \frac{1}{4}\right)}{\lg\left(a_n + \frac{1}{2}\right)}
$$

注意到：
$$
a_n^2 + a_n + \frac{1}{4} = \left(a_n + \frac{1}{2}\right)^2
$$

所以：
$$
\frac{\lg\left(a_{n+1} + \frac{1}{2}\right)}{\lg\left(a_n + \frac{1}{2}\right)} = \frac{\lg\left(\left(a_n + \frac{1}{2}\right)^2\right)}{\lg\left(a_n + \frac{1}{2}\right)} = \frac{2 \lg\left(a_n + \frac{1}{2}\right)}{\lg\left(a_n + \frac{1}{2}\right)} = 2
$$

因此，数列 $\left\{\lg\left(a_n + \frac{1}{2}\right)\right\}$ 是公比为 $2$ 的等比数列。

其首项为：
$$
\lg\left(a_1 + \frac{1}{2}\right) = \lg\left(\frac{3}{2} + \frac{1}{2}\right) = \lg 2
$$

所以：
$$
\lg\left(a_n + \frac{1}{2}\right) = \lg 2 \cdot 2^{n-1} = \lg(2^{2^{n-1}})
$$

因此：
$$
a_n + \frac{1}{2} = 2^{2^{n-1}} \Rightarrow a_n = 2^{2^{n-1}} - \frac{1}{2}
$$

故数列 $\{a_n\}$ 的通项公式为 $a_n = 2^{2^{n-1}} - \frac{1}{2}$ ($n \in \mathbb{N}^*$)。

## 三、分步教学引导

### 步骤1：理解题意
- **引导语**："题目给出了一个函数 $f(x)$ 和一个数列 $\{a_n\}$，并且告诉我们点 $(a_n, a_{n+1})$ 在函数图像上。这说明了什么？"
- **关键提示**：$a_{n+1} = f(a_n)$。
- **期望学生回答**：我可以写出 $a_{n+1} = a_n^2 + a_n - \frac{1}{4}$。

### 步骤2：构造新数列
- **引导语**："我们要证明 $\left\{\lg\left(a_n + \frac{1}{2}\right)\right\}$ 是等比数列。如何开始？"
- **关键提示**：利用等比数列的定义。
- **期望学生回答**：我需要计算 $\frac{\lg\left(a_{n+1} + \frac{1}{2}\right)}{\lg\left(a_n + \frac{1}{2}\right)}$ 是否为常数。

### 步骤3：化简表达式
- **引导语**："现在我们有 $a_{n+1} = a_n^2 + a_n - \frac{1}{4}$，如何代入 $\lg\left(a_{n+1} + \frac{1}{2}\right)$？"
- **关键提示**：合并常数项。
- **期望学生回答**：我可以把 $a_{n+1} + \frac{1}{2} = a_n^2 + a_n + \frac{1}{4}$。

### 步骤4：应用对数性质
- **引导语**："注意到 $a_n^2 + a_n + \frac{1}{4} = \left(a_n + \frac{1}{2}\right)^2$，这对对数有什么帮助？"
- **关键提示**：$\lg(a^2) = 2 \lg a$。
- **期望学生回答**：我可以把 $\lg\left(\left(a_n + \frac{1}{2}\right)^2\right)$ 化为 $2 \lg\left(a_n + \frac{1}{2}\right)$。

### 步骤5：求出通项
- **引导语**："现在我们知道 $\left\{\lg\left(a_n + \frac{1}{2}\right)\right\}$ 是等比数列，如何求出 $a_n$？"
- **关键提示**：先写出 $\lg\left(a_n + \frac{1}{2}\right)$ 的通项，再取指数。
- **期望学生回答**：$\lg\left(a_n + \frac{1}{2}\right) = \lg 2 \cdot 2^{n-1} = \lg(2^{2^{n-1}})$，所以 $a_n + \frac{1}{2} = 2^{2^{n-1}}$。
