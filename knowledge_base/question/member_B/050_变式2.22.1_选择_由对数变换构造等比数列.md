## 一、题目内容

> **原题位置**
> ·数列与不等式 P50 变式2.22.1

**来源**
(2018 天津六校联考)

**题目：**
数列 $\{a_n\}$ 满足 $a_1 = 2$，$a_{n+1} = a_n^2$。则 $a_n =$ （ ）。

A. $10^{n-2}$  
B. $10^{n-1}$  
C. $10^{2^{n-1}}$  
D. $2^{2^{n-1}}$

## 二、答案与解析

### 答案
D. $2^{2^{n-1}}$

### 解析

已知：
$$
a_1 = 2, \quad a_{n+1} = a_n^2
$$

我们计算前几项：

- $a_1 = 2$
- $a_2 = a_1^2 = 2^2 = 4$
- $a_3 = a_2^2 = 4^2 = 16$
- $a_4 = a_3^2 = 16^2 = 256$

观察发现：
- $a_1 = 2 = 2^{2^0}$
- $a_2 = 4 = 2^{2^1}$
- $a_3 = 16 = 2^{2^2}$
- $a_4 = 256 = 2^{2^3}$

所以可以归纳出：
$$
a_n = 2^{2^{n-1}}
$$

验证选项：
- A. $10^{n-2}$：当 $n=1$ 时，$10^{-1} = 0.1 \neq 2$，排除。
- B. $10^{n-1}$：当 $n=1$ 时，$10^0 = 1 \neq 2$，排除。
- C. $10^{2^{n-1}}$：当 $n=1$ 时，$10^{2^0} = 10^1 = 10 \neq 2$，排除。
- D. $2^{2^{n-1}}$：当 $n=1$ 时，$2^{2^0} = 2^1 = 2$，符合；当 $n=2$ 时，$2^{2^1} = 2^2 = 4$，符合。

故正确答案为 D。

## 三、分步教学引导

### 步骤1：理解题意
- **引导语**："题目给出了一个数列 $\{a_n\}$，其中 $a_1 = 2$，$a_{n+1} = a_n^2$。你能想到什么方法来求通项公式吗？"
- **关键提示**：计算前几项。
- **期望学生回答**：我可以先算出 $a_2, a_3, a_4$，看看有没有规律。

### 步骤2：计算前几项
- **引导语**："现在我们有 $a_1 = 2$，$a_2 = a_1^2 = 4$，$a_3 = a_2^2 = 16$。接下来呢？"
- **关键提示**：继续计算。
- **期望学生回答**：$a_4 = a_3^2 = 256$。

### 步骤3：观察规律
- **引导语**："现在我们有 $a_1 = 2$，$a_2 = 4$，$a_3 = 16$，$a_4 = 256$。这些数有什么共同点？"
- **关键提示**：写成幂的形式。
- **期望学生回答**：$2 = 2^1$，$4 = 2^2$，$16 = 2^4$，$256 = 2^8$，指数是 $1, 2, 4, 8$，即 $2^0, 2^1, 2^2, 2^3$。

### 步骤4：归纳通项
- **引导语**："所以 $a_n = 2^{2^{n-1}}$。如何验证这个结论？"
- **关键提示**：代入选项。
- **期望学生回答**：我可以把 $n=1, 2, 3, 4$ 代入各个选项，看哪个符合。

### 步骤5：验证选项
- **引导语**："现在我们已经得出 $a_n = 2^{2^{n-1}}$，如何判断哪个选项正确？"
- **关键提示**：逐个代入。
- **期望学生回答**：A、B、C 都不符合，只有 D 符合。

## 七、备注

- **知识点应用难点**：本题的关键在于能够通过计算前几项来发现规律，并正确地写出通项公式。
- **易错点提醒**：在计算 $a_n$ 时，要注意指数的层级，避免混淆 $2^{2^{n-1}}$ 和 $(2^2)^{n-1}$。
- **解题技巧总结**：对于形如 $a_{n+1} = a_n^p$ 的递推关系，常常可以通过计算前几项来发现规律，并利用指数运算法则来求解通项公式。
