# 【A_033_2】系数递推求通项（累乘法）

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P33 变式2.3.2
&gt; 
&gt; **来源**：2018年广州调研文17

**题目：**

已知数列 $\{a_n\}$ 满足 $a_1 + 4a_2 + 4^2a_3 + \cdots + 4^{n-1}a_n = \frac{n}{4}$（$n \in \mathbb{N}^*$），求其通项公式。

---

## 二、答案与解析

### 答案

$$a_n = \frac{1}{4^n} \quad (n \in \mathbb{N}^*)$$

### 解析

**步骤1：设和式为$T_n$**

设$T_n = a_1 + 4a_2 + 4^2a_3 + \cdots + 4^{n-1}a_n = \frac{n}{4}$。

**步骤2：求首项$a_1$**

当$n = 1$时：
$$T_1 = a_1 = \frac{1}{4}$$

**步骤3：建立递推关系**

当$n \geq 2$时：
$$T_{n-1} = a_1 + 4a_2 + \cdots + 4^{n-2}a_{n-1} = \frac{n-1}{4}$$

两式相减：
$$T_n - T_{n-1} = 4^{n-1}a_n = \frac{n}{4} - \frac{n-1}{4} = \frac{1}{4}$$

**步骤4：求解$a_n$**

由$4^{n-1}a_n = \frac{1}{4}$，得：
$$a_n = \frac{1}{4} \cdot \frac{1}{4^{n-1}} = \frac{1}{4^n} = \left(\frac{1}{4}\right)^n \quad (n \geq 2)$$

**步骤5：验证$n=1$的情况**

当$n = 1$时，$\left(\frac{1}{4}\right)^1 = \frac{1}{4} = a_1$，符合。

因此，通项公式为$a_n = \frac{1}{4^n}$（$n \in \mathbb{N}^*$）。

## 三、分步教学引导（用于AI Agent）

**步骤1：识别和式结构**
- **引导语**："观察等式左边$a_1 + 4a_2 + 4^2a_3 + \cdots + 4^{n-1}a_n$，这是数列的什么？我们可以设什么？"
- **关键提示**：这是数列$\{4^{n-1}a_n\}$的前$n$项和
- **期望学生回答**：设$T_n = \sum_{k=1}^n 4^{k-1}a_k = \frac{n}{4}$

**步骤2：求首项**
- **引导语**："当$n=1$时，左边只有第一项，等于多少？"
- **关键提示**：$n=1$时，左边$= a_1$，右边$= \frac{1}{4}$
- **期望学生回答**：$a_1 = \frac{1}{4}$

**步骤3：利用作差法**
- **引导语**："写出$T_{n-1}$的表达式（$n \geq 2$），然后计算$T_n - T_{n-1}$。"
- **关键提示**：$T_{n-1} = \frac{n-1}{4}$
- **期望学生回答**：$T_n - T_{n-1} = 4^{n-1}a_n = \frac{1}{4}$

**步骤4：解出$a_n$**
- **引导语**："由$4^{n-1}a_n = \frac{1}{4}$，解出$a_n$。"
- **关键提示**：$a_n = \frac{1}{4} \cdot 4^{-(n-1)} = 4^{-n}$
- **期望学生回答**：$a_n = \frac{1}{4^n}$（$n \geq 2$）

**步骤5：验证一致性**
- **引导语**："当$n=1$时，这个公式是否成立？"
- **关键提示**：代入$n=1$验证
- **期望学生回答**：成立，所以$a_n = \frac{1}{4^n}$对所有$n \in \mathbb{N}^*$成立

## 四、标签信息（用于分类和检索）

### 解题方法标签
- 累差法（作差法）
- 构造和式法（构造$T_n$）
- 指数运算
- 验证法

### 易错点标签
- 混淆$4^{n-1}a_n$的指数运算（如错误地认为$\frac{1}{4} \div 4^{n-1} = \frac{1}{4^{n-2}}$）
- 忘记验证$n=1$的情况（虽然本题可合并，但需养成习惯）
- 错误地将左边看作$\{a_n\}$的前$n$项和（忽略系数$4^{k-1}$）
- 计算$T_n - T_{n-1}$时分数运算错误（$\frac{n}{4} - \frac{n-1}{4} = \frac{1}{4}$）

### 题目特征标签
- 调研题（广州调研）
- 系数递推型（$4^{n-1}a_n$）
- 等比数列通项（结果为等比数列）
- 累差法典型题

## 五、关联拓展

### 相关方法总结

**形如$\sum_{k=1}^n f(k) \cdot a_k = g(n)$的求通项方法：**
1. 设$T_n = \sum_{k=1}^n f(k) \cdot a_k = g(n)$
2. $n=1$时，求$a_1$
3. $n \geq 2$时，$f(n) \cdot a_n = T_n - T_{n-1} = g(n) - g(n-1)$
4. 解出$a_n = \frac{g(n) - g(n-1)}{f(n)}$
5. 验证$n=1$是否满足

**常见变形：**
- $\sum_{k=1}^n k \cdot a_k = g(n)$ → $n \cdot a_n = g(n) - g(n-1)$
- $\sum_{k=1}^n 2^k \cdot a_k = g(n)$ → $2^n \cdot a_n = g(n) - g(n-1)$
