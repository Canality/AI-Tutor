# 【A_016】等比数列性质巧算

## 一、题目内容

**题目：**

在等比数列 $\{a_n\}$ 中，若 $a_7 + a_8 + a_9 + a_{10} = \frac{15}{8}$，$a_8 a_9 = -\frac{9}{8}$，则 $\frac{1}{a_7} + \frac{1}{a_8} + \frac{1}{a_9} + \frac{1}{a_{10}} =$ ______。

## 二、答案与解析

### 答案

$-\frac{5}{3}$

### 解析

**步骤1：利用等比数列性质**

注意到 $a_7 a_{10} = a_8 a_9 = -\frac{9}{8}$（因为 $7+10 = 8+9$）。

**步骤2：化简所求式**

$$\frac{1}{a_7} + \frac{1}{a_{10}} = \frac{a_7 + a_{10}}{a_7 a_{10}} = \frac{a_7 + a_{10}}{-\frac{9}{8}}$$

$$\frac{1}{a_8} + \frac{1}{a_9} = \frac{a_8 + a_9}{a_8 a_9} = \frac{a_8 + a_9}{-\frac{9}{8}}$$

**步骤3：合并**

原式 $= \frac{a_7 + a_8 + a_9 + a_{10}}{-\frac{9}{8}} = \frac{\frac{15}{8}}{-\frac{9}{8}} = -\frac{15}{9} = -\frac{5}{3}$。

## 三、分步教学引导

**步骤1：观察下标关系**
- **引导语**："观察 $7,8,9,10$ 的下标，有什么性质可用？"
- **关键提示**：$7+10 = 8+9$
- **期望学生回答**：$a_7 a_{10} = a_8 a_9$

**步骤2：通分合并**
- **引导语**："将所求式两两组合通分。"
- **期望学生回答**：$\frac{a_7+a_{10}}{a_7a_{10}} + \frac{a_8+a_9}{a_8a_9}$

**步骤3：代入计算**
- **引导语**："利用 $a_7a_{10} = a_8a_9 = -\frac{9}{8}$ 代入。"
- **期望学生回答**：$-\frac{5}{3}$
