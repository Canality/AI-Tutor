# 【A_007_3】含平方项的递推关系（2015新课标I）

## 一、题目内容

&gt; **原题位置**：P7 变式1.10.2
&gt; 
&gt; **来源**：2015年新课标I卷理科第17题

**题目：**

$S_n$ 是数列 $\{a_n\}$ 的前 $n$ 项和，已知 $a_n &gt; 0$，$a_n^2 + 2a_n = 4S_n + 3$，则 $\{a_n\}$ 的通项公式为 ______。

---

## 二、答案与解析

### 答案

$$a_n = 2n + 1$$

### 解析

**步骤1：求首项 $a_1$**

当 $n = 1$ 时，$S_1 = a_1$，代入已知条件：
$$a_1^2 + 2a_1 = 4a_1 + 3$$
$$a_1^2 - 2a_1 - 3 = 0$$
$$(a_1 - 3)(a_1 + 1) = 0$$

因为 $a_n &gt; 0$，所以 $a_1 = 3$。

**步骤2：建立递推关系**

当 $n \geq 2$ 时，有：
$$a_n^2 + 2a_n = 4S_n + 3 \quad \text{①}$$
$$a_{n-1}^2 + 2a_{n-1} = 4S_{n-1} + 3 \quad \text{②}$$

① - ② 得：
$$a_n^2 - a_{n-1}^2 + 2a_n - 2a_{n-1} = 4(S_n - S_{n-1}) = 4a_n$$

**步骤3：化简并判定数列类型**

左边因式分解：
$$(a_n - a_{n-1})(a_n + a_{n-1}) + 2(a_n - a_{n-1}) = 4a_n$$
$$(a_n - a_{n-1})(a_n + a_{n-1} + 2) = 4a_n$$

由于 $a_n &gt; 0$，所以 $a_n + a_{n-1} + 2 &gt; 0$。

进一步分析（或尝试假设等差）：

由 $a_n^2 + 2a_n = 4S_n + 3$ 和 $a_{n-1}^2 + 2a_{n-1} = 4S_{n-1} + 3$ 相减：

$$a_n^2 - a_{n-1}^2 + 2(a_n - a_{n-1}) = 4a_n$$
$$(a_n - a_{n-1})(a_n + a_{n-1}) + 2(a_n - a_{n-1}) = 4a_n$$
$$(a_n - a_{n-1})(a_n + a_{n-1} + 2) = 4a_n$$

注意到如果 $\{a_n\}$ 是等差数列，设公差为 $d$，则 $a_n - a_{n-1} = d$：
$$d(a_n + a_{n-1} + 2) = 4a_n$$
$$d(2a_{n-1} + d + 2) = 4(a_{n-1} + d)$$

对比系数可得 $d = 2$。

验证：若 $d = 2$，则 $2(2a_{n-1} + 2 + 2) = 4(a_{n-1} + 2)$，即 $4a_{n-1} + 8 = 4a_{n-1} + 8$，成立。

**步骤4：求通项公式**

$\{a_n\}$ 是首项 $a_1 = 3$，公差 $d = 2$ 的等差数列。

$$a_n = a_1 + (n-1)d = 3 + 2(n-1) = 2n + 1$$

验证：当 $n=1$ 时，$a_1 = 3$，符合。

故 $a_n = 2n + 1$。

## 三、分步教学引导（用于AI Agent）

### 整体思路引导

**步骤1：求首项**
- **引导语**："当 $n=1$ 时，$S_1$ 等于什么？代入已知条件能得到什么？"
- **关键提示**：$S_1 = a_1$，解关于 $a_1$ 的二次方程，注意 $a_n &gt; 0$ 的条件
- **期望学生回答**：$a_1^2 + 2a_1 = 4a_1 + 3$，解得 $a_1 = 3$（舍去 $-1$）

**步骤2：建立递推关系**
- **引导语**："对于 $n \geq 2$，我们有两个方程（n和n-1），如何处理？"
- **关键提示**：写出 $n$ 和 $n-1$ 时的等式，相减消去 $S_n$
- **期望学生回答**：$a_n^2 - a_{n-1}^2 + 2a_n - 2a_{n-1} = 4a_n$

**步骤3：化简与判定**
- **引导语**："左边可以因式分解吗？分解后能看到什么规律？"
- **关键提示**：$(a_n - a_{n-1})(a_n + a_{n-1} + 2) = 4a_n$，尝试假设等差数列
- **期望学生回答**：假设 $a_n - a_{n-1} = d$（常数），代入验证发现 $d = 2$ 满足

**步骤4：写出通项**
- **引导语**："现在知道是等差数列，首项和公差分别是多少？"
- **关键提示**：$a_1 = 3$，$d = 2$
- **期望学生回答**：$a_n = 3 + (n-1) \times 2 = 2n + 1$

## 四、标签信息（用于分类和检索）

### 解题方法标签
- 类讨论型（由 $S_n$ 求 $a_n$）
- 递推法（$a_n$ 与 $S_n$ 混合关系）
- 因式分解（平方差公式）
- 等差数列判定（假设-验证法）

### 易错点标签
- 未舍去负根（解 $a_1$ 时得到 $3$ 和 $-1$，忘记 $a_n &gt; 0$）
- 相减时符号错误（$a_n^2 - a_{n-1}^2$ 分解错误）
- 无法识别等差数列（化简后不会处理 $(a_n - a_{n-1})(a_n + a_{n-1} + 2) = 4a_n$）
- 计算公差错误（假设等差后代入计算错误）

### 题目特征标签
- 高考真题（2015新课标I理17）
- 中等偏难（递推关系复杂）
- 含 $a_n^2$ 的递推
- 等差数列证明与求解

## 五、关联拓展

### 相关方法总结

**处理 $a_n^2$ 与 $S_n$ 混合关系的策略：**
1. 先求 $a_1$（令 $n=1$，$S_1 = a_1$）
2. 写出 $n$ 和 $n-1$ 时的关系式，相减消去 $S_n$
3. 利用 $a_n = S_n - S_{n-1}$ 建立纯 $a_n$ 的递推关系
4. 通过因式分解或假设特殊数列（如等差、等比）求解

**常见因式分解技巧：**
- $a_n^2 - a_{n-1}^2 = (a_n - a_{n-1})(a_n + a_{n-1})$
- 分组分解法
