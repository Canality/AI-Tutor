# 【A_016_2】等比数列多项式整体求值

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P16 变式1.23.1
&gt; 
&gt; **来源**：2018年广州模考

**题目：**

已知数列 $\{a_n\}$ 为等比数列，若 $a_4 + a_6 = 10$，则 $a_7(a_1 + 2a_3) + a_3 a_9 = $ ______。

---

## 二、答案与解析

### 答案

$100$

### 解析

**步骤1：展开表达式**

$$a_7(a_1 + 2a_3) + a_3 a_9 = a_1 a_7 + 2a_3 a_7 + a_3 a_9$$

**步骤2：利用等比数列性质转化各项**

设公比为 $q$，则：
- $a_1 a_7 = a_1 \cdot a_1 q^6 = a_1^2 q^6 = (a_1 q^3)^2 = a_4^2$
- $a_3 a_7 = a_1 q^2 \cdot a_1 q^6 = a_1^2 q^8 = (a_1 q^4)^2 = a_5^2$？不对，重新检查下标。

正确利用性质：$m+n=p+q \Rightarrow a_m a_n = a_p a_q$。

- $a_1 a_7$：$1+7=8=4+4$，所以 $a_1 a_7 = a_4^2$
- $a_3 a_7$：$3+7=10=5+5$，所以 $a_3 a_7 = a_5^2$？但原式有$a_3a_7$。

重新观察：目标是凑出$a_4+a_6$的形式。

- $a_1 a_7 = a_4^2$（因为$1+7=4+4$）
- $a_3 a_9 = a_6^2$（因为$3+9=6+6$）
- $a_3 a_7$：$3+7=10=4+6$，所以 $a_3 a_7 = a_4 a_6$

因此：
原式 $= a_4^2 + 2a_4 a_6 + a_6^2 = (a_4 + a_6)^2$

**步骤3：代入计算**

$$= 10^2 = 100$$

---

## 三、分步教学引导（用于AI Agent）

**步骤1：展开多项式**
- **引导语**："先将括号展开，得到几项？"
- **期望学生回答**：$a_1a_7 + 2a_3a_7 + a_3a_9$

**步骤2：分析下标配对**
- **引导语**："观察各项的下标乘积：$a_1a_7$、$a_3a_7$、$a_3a_9$，利用等比性质转化为含$a_4$和$a_6$的形式。"
- **关键提示**：$1+7=4+4$，$3+7=4+6$，$3+9=6+6$
- **期望学生回答**：$a_1a_7=a_4^2$，$a_3a_7=a_4a_6$，$a_3a_9=a_6^2$

**步骤3：配方计算**
- **引导语**："现在表达式变成什么形式？"
- **关键提示**：$a_4^2 + 2a_4a_6 + a_6^2 = (a_4+a_6)^2$
- **期望学生回答**：完全平方，结果为$100$

## 四、标签信息（用于分类和检索）

### 解题方法标签
- 等比数列性质（下标和相等则积相等）
- 多项式展开
- 完全平方公式
- 整体代换

### 易错点标签
- 下标配对错误（如误认为$a_3a_7=a_5^2$）
- 展开时漏项（忘记$2a_3$乘$a_7$）
- 计算$a_4+a_6$的平方时直接得20（未平方）

### 题目特征标签
- 模考题（广州）
- 整体代换思想
- 性质灵活运用

## 五、关联拓展

**等比数列求值技巧：**
当已知$a_m+a_n$的值，求关于$a_k a_l$的表达式时，尝试将积转化为和的形式，通常可以凑成$(a_m+a_n)^2$或$(a_m+a_n)(a_p+a_q)$。
