# 【A_014】等比数列性质与计算

## 一、题目内容

**题目：**

已知 $\{a_n\}$ 为等比数列，$a_4 + a_7 = 2$，$a_5 a_6 = -8$，则 $a_1 + a_{10} =$（ ）。

A. $7$  
B. $5$  
C. $-5$  
D. $-7$

## 二、答案与解析

### 答案

**D**（$-7$）

### 解析

**步骤1：利用等比数列性质**

由等比数列性质 $a_5 a_6 = a_4 a_7 = -8$。

**步骤2：解方程组**

设 $a_4 = x$，$a_7 = y$，则：
$$\begin{cases} x + y = 2 \\ xy = -8 \end{cases}$$

解得：
$$\begin{cases} x = 4 \\ y = -2 \end{cases} \quad \text{或} \quad \begin{cases} x = -2 \\ y = 4 \end{cases}$$

**步骤3：情况一：$a_4 = 4$，$a_7 = -2$**

$$q^3 = \frac{a_7}{a_4} = -\frac{1}{2}$$

$$a_1 = \frac{a_4}{q^3} = \frac{4}{-1/2} = -8$$

$$a_{10} = a_7 q^3 = -2 \cdot \left(-\frac{1}{2}\right) = 1$$

$$a_1 + a_{10} = -8 + 1 = -7$$

**步骤4：情况二：$a_4 = -2$，$a_7 = 4$**

$$q^3 = \frac{4}{-2} = -2$$

$$a_1 = \frac{-2}{-2} = 1$$

$$a_{10} = 4 \cdot (-2) = -8$$

$$a_1 + a_{10} = 1 - 8 = -7$$

综上，$a_1 + a_{10} = -7$。

## 三、分步教学引导

**步骤1：利用性质转化**
- **引导语**："$a_5 a_6$ 可以用 $a_4$ 和 $a_7$ 表示吗？"
- **关键提示**：$a_5 a_6 = a_4 a_7$（下标和相等：$5+6=4+7$）
- **期望学生回答**：$a_4 a_7 = -8$

**步骤2：解方程组**
- **引导语**："已知 $a_4 + a_7 = 2$ 和 $a_4 a_7 = -8$，求 $a_4$ 和 $a_7$。"
- **期望学生回答**：$(4, -2)$ 或 $(-2, 4)$

**步骤3：计算 $a_1 + a_{10}$**
- **引导语**："在两种情况下分别计算 $a_1 + a_{10}$。"
- **关键提示**：$a_{10} = a_7 q^3 = a_7 \cdot \frac{a_7}{a_4} = \frac{a_7^2}{a_4}$，或直接求 $q^3$
- **期望学生回答**：两种情况都得到 $-7$
