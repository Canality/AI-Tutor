# 【A_008_2】Sn/n为等差数列的应用（2018合肥三模）

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P8 例1.12
&gt; 
&gt; **来源**：2018年合肥三模文科第16题

**题目：**

已知数列 $\{a_n\}$ 的前 $n$ 项和为 $S_n$，且 $\{\frac{S_n}{n}\}$ 为等差数列，若 $S_2 = 1$，$S_{2018} - S_{2016} = 5$，则 $S_{2018} = $ ______。

---

## 二、答案与解析

### 答案

$$3027$$

### 解析

**步骤1：设 $\frac{S_n}{n}$ 的通项**

设 $\{\frac{S_n}{n}\}$ 是首项为 $b_1$，公差为 $d$ 的等差数列，则：
$$\frac{S_n}{n} = b_1 + (n-1)d$$

**步骤2：利用 $S_2 = 1$ 求关系**

$$\frac{S_2}{2} = b_1 + d = \frac{1}{2}$$
所以 $b_1 = \frac{1}{2} - d$。

因此：
$$\frac{S_n}{n} = \frac{1}{2} - d + (n-1)d = \frac{1}{2} + (n-2)d$$

即 $S_n = n[\frac{1}{2} + (n-2)d] = \frac{n}{2} + n(n-2)d$。

**步骤3：利用 $S_{2018} - S_{2016} = 5$**

计算 $S_{2018}$ 和 $S_{2016}$：
$$S_{2018} = 1009 + 2018 \times 2016d$$
$$S_{2016} = 1008 + 2016 \times 2014d$$

相减：
$$S_{2018} - S_{2016} = 1 + 2018 \times 2016d - 2016 \times 2014d$$
$$= 1 + 2016(2018 - 2014)d$$
$$= 1 + 2016 \times 4d$$
$$= 1 + 8064d = 5$$

解得：
$$8064d = 4$$
$$d = \frac{4}{8064} = \frac{1}{2016}$$

**步骤4：计算 $S_{2018}$**

$$S_{2018} = 1009 + 2018 \times 2016 \times \frac{1}{2016}$$
$$= 1009 + 2018$$
$$= 3027$$

故填 **3027**。

## 五、关联拓展

### 相关方法总结

**处理 $\{\frac{S_n}{n}\}$ 为等差数列的套路：**
1. 设 $\frac{S_n}{n} = b_1 + (n-1)d$
2. 则 $S_n = n(b_1 + (n-1)d) = dn^2 + (b_1-d)n$
3. 对比 $S_n = \frac{d'}{2}n^2 + (a_1 - \frac{d'}{2})n$，可知原数列公差 $d' = 2d$

**快速计算技巧：**
$S_{2018} - S_{2016} = a_{2017} + a_{2018}$（后两项和），但本题用 $\frac{S_n}{n}$ 构造更直接。
