# 【A_008_2】等差数列基本量与最值（2018新课标II）

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P8 例1.15
&gt; 
&gt; **来源**：2018年新课标II卷文理科第17题

**题目：**

记 $S_n$ 为等差数列 $\{a_n\}$ 的前 $n$ 项和，已知 $a_1=-7, S_3=-15$。

(I) 求 $\{a_n\}$ 的通项公式；

(II) 求 $S_n$，并求 $S_n$ 的最小值。

---

## 二、答案与解析

### 答案

**(I)** $a_n = 2n-9$

**(II)** $S_n = n^2-8n$，最小值为 $-16$（当 $n=4$ 时取得）

### 解析

**步骤1：求公差 $d$（第I问）**

由 $S_3 = 3a_1 + \frac{3\times 2}{2}d = 3a_1 + 3d = -15$。

代入 $a_1=-7$：
$$3(-7) + 3d = -15$$
$$-21 + 3d = -15$$
$$3d = 6 \Rightarrow d=2$$

**步骤2：求通项公式（第I问）**

$$a_n = a_1 + (n-1)d = -7 + 2(n-1) = 2n-9$$

**步骤3：求 $S_n$ 并配方（第II问）**

$$S_n = \frac{n(a_1+a_n)}{2} = \frac{n(-7+2n-9)}{2} = \frac{n(2n-16)}{2} = n^2-8n$$

配方得：
$$S_n = (n-4)^2 - 16$$

**步骤4：确定最小值（第II问）**

由于 $n \in \mathbb{N}^*$，当 $n=4$ 时，$S_n$ 取得最小值 $-16$。

## 三、分步教学引导（用于AI Agent）

### 整体思路引导

**步骤1：建立方程**
- **引导语**："已知 $a_1$ 和 $S_3$，如何利用 $S_3$ 的公式建立关于 $d$ 的方程？"
- **关键提示**：$S_3 = 3a_1 + 3d$
- **期望学生回答**：代入得 $-21+3d=-15$，解得 $d=2$

**步骤2：写通项公式**
- **引导语**："已知 $a_1=-7, d=2$，写出 $a_n$ 的表达式。"
- **关键提示**：$a_n = a_1 + (n-1)d$
- **期望学生回答**：$a_n = 2n-9$

**步骤3：求 $S_n$ 表达式**
- **引导语**："利用 $S_n = \frac{n(a_1+a_n)}{2}$ 或 $S_n = na_1 + \frac{n(n-1)}{2}d$ 计算。"
- **关键提示**：代入 $a_n=2n-9$ 或直接用 $n$ 和 $d$ 的公式
- **期望学生回答**：$S_n = n^2-8n$

**步骤4：求最值**
- **引导语**："$S_n = n^2-8n$ 是关于 $n$ 的二次函数，如何求最小值？注意 $n$ 的取值范围。"
- **关键提示**：配方得 $(n-4)^2-16$，对称轴 $n=4$ 恰为整数
- **期望学生回答**：当 $n=4$ 时，$S_{\min}=-16$

## 五、关联拓展

### 核心方法总结

**等差数列最值的两种求法：**
1. **函数法**：$S_n = \frac{d}{2}n^2 + (a_1-\frac{d}{2})n$，视为二次函数求最值，注意 $n \in \mathbb{N}^*$。
2. **邻项法**：找使 $a_n \leq 0$ 且 $a_{n+1} \geq 0$（最小值）或反之的 $n$。

**本题邻项法验证：**
$a_n = 2n-9 \leq 0 \Rightarrow n \leq 4.5$
$a_4 = -1 &lt; 0, a_5 = 1 &gt; 0$，故 $n=4$ 时 $S_n$ 最小，$S_4 = 16-32 = -16$。
