# 【A_012_2】数列最值与求和

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P12 例1.18
&gt; 
&gt; **来源**：2019年四川模拟第16题

**题目：**

记 $S_n$ 为等差数列 $a_n = 2n - 1$ 的前 $n$ 项和，则 $\{4n - S_n\}$ 的前 $n$ 项和的最大值为 ______。

---

## 二、答案与解析

### 答案

$10$

### 解析

**步骤1：求 $S_n$**

数列 $a_n = 2n - 1$ 是首项为 $1$，公差为 $2$ 的等差数列。

$$S_n = \frac{n(1 + 2n - 1)}{2} = \frac{n \cdot 2n}{2} = n^2$$

**步骤2：构造新数列**

设 $b_n = 4n - S_n = 4n - n^2$。

**步骤3：分析 $b_n$ 的符号**

令 $b_n \geq 0$：
$$4n - n^2 \geq 0$$
$$n(4 - n) \geq 0$$
$$0 \leq n \leq 4$$

由于 $n \in \mathbb{N}^*$，所以 $n = 1, 2, 3, 4$ 时 $b_n \geq 0$。

当 $n \geq 5$ 时，$b_n &lt; 0$。

**步骤4：求前 $n$ 项和的最大值**

设 $T_n$ 为 $\{b_n\}$ 的前 $n$ 项和。

由于 $b_1, b_2, b_3, b_4 &gt; 0$，$b_5, b_6, \ldots &lt; 0$，所以 $T_n$ 在 $n=4$ 时取得最大值。

$$T_4 = b_1 + b_2 + b_3 + b_4 = (4-1) + (8-4) + (12-9) + (16-16) = 3 + 4 + 3 + 0 = 10$$

## 三、分步教学引导

**步骤1：计算 $S_n$**
- **引导语**："已知 $a_n = 2n - 1$，求前 $n$ 项和 $S_n$。"
- **关键提示**：等差数列求和，首项1，末项 $2n-1$
- **期望学生回答**：$S_n = n^2$

**步骤2：构造函数**
- **引导语**："写出 $b_n = 4n - S_n$ 的表达式。"
- **关键提示**：代入 $S_n = n^2$
- **期望学生回答**：$b_n = 4n - n^2$

**步骤3：确定正项范围**
- **引导语**："解不等式 $b_n \geq 0$，找出所有正整数 $n$。"
- **关键提示**：$n(4-n) \geq 0$
- **期望学生回答**：$n = 1, 2, 3, 4$

**步骤4：计算最大和**
- **引导语**："由于第5项开始为负，前4项和最大，计算 $T_4$。"
- **关键提示**：$b_1=3, b_2=4, b_3=3, b_4=0$
- **期望学生回答**：$3+4+3+0=10$

## 五、关联拓展

### 方法总结
对于数列 $\{c_n\}$，若 $c_n &gt; 0$ 对 $n \leq k$ 成立，$c_n &lt; 0$ 对 $n &gt; k$ 成立，则前 $n$ 项和在 $n=k$ 时最大。
