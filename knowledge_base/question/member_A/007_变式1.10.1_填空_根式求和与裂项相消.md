# 【A_007_2】根式求和与裂项相消（2018宝鸡质检）

## 一、题目内容

&gt; **原题位置**：P7 变式1.10.1
&gt; 
&gt; **来源**：2018年陕西宝鸡质检

**题目：**

若数列 $\{a_n\}$ 是正项数列，且 $\sqrt{a_1} + \sqrt{a_2} + \sqrt{a_3} + \cdots + \sqrt{a_n} = n^2 + n$，则 $a_1 + \frac{a_2}{2} + \cdots + \frac{a_n}{n} = $ ______。

---

## 二、答案与解析

### 答案

$$2n^2 + 2n$$

### 解析

**步骤1：设辅助数列求 $\sqrt{a_n}$**

设 $T_n = \sqrt{a_1} + \sqrt{a_2} + \cdots + \sqrt{a_n} = n^2 + n$。

当 $n = 1$ 时：
$$\sqrt{a_1} = T_1 = 1^2 + 1 = 2$$
所以 $a_1 = 4$。

当 $n \geq 2$ 时：
$$\sqrt{a_n} = T_n - T_{n-1} = (n^2 + n) - [(n-1)^2 + (n-1)]$$
$$= n^2 + n - (n^2 - 2n + 1 + n - 1)$$
$$= n^2 + n - n^2 + n$$
$$= 2n$$

验证 $n=1$：$\sqrt{a_1} = 2 \times 1 = 2$，与上式一致。

故 $\sqrt{a_n} = 2n$，即 $a_n = 4n^2$。

**步骤2：计算 $\frac{a_n}{n}$**

$$\frac{a_n}{n} = \frac{4n^2}{n} = 4n$$

**步骤3：求和**

$$a_1 + \frac{a_2}{2} + \cdots + \frac{a_n}{n} = \sum_{k=1}^{n} 4k = 4 \sum_{k=1}^{n} k = 4 \cdot \frac{n(n+1)}{2} = 2n(n+1) = 2n^2 + 2n$$

故答案为 $2n^2 + 2n$。

---

## 三、分步教学引导（用于AI Agent）

### 整体思路引导

**步骤1：识别数列类型**
- **引导语**："题目左边是 $\sqrt{a_n}$ 的前 $n$ 项和，我们可以把它看作什么？"
- **关键提示**：设新数列 $b_n = \sqrt{a_n}$，则左边就是 $b_n$ 的前 $n$ 项和 $T_n$
- **期望学生回答**：设 $T_n = \sum_{k=1}^n \sqrt{a_k} = n^2 + n$

**步骤2：求 $\sqrt{a_n}$ 的通项**
- **引导语**："知道了 $T_n = n^2 + n$，如何求 $\sqrt{a_n}$？"
- **关键提示**：利用 $b_n = T_n - T_{n-1}$（$n \geq 2$），且 $b_1 = T_1$
- **期望学生回答**：$\sqrt{a_n} = 2n$（$n \geq 1$）

**步骤3：求 $a_n$ 并计算 $\frac{a_n}{n}$**
- **引导语**："现在知道了 $\sqrt{a_n} = 2n$，那么 $a_n$ 是多少？$\frac{a_n}{n}$ 又是多少？"
- **关键提示**：两边平方，然后除以 $n$
- **期望学生回答**：$a_n = 4n^2$，所以 $\frac{a_n}{n} = 4n$

**步骤4：求最终和式**
- **引导语**："现在要求的是 $4 + 8 + 12 + \cdots + 4n$，这是什么数列？如何求和？"
- **关键提示**：等差数列求和，或者提取公因数后用自然数和公式
- **期望学生回答**：$4(1+2+\cdots+n) = 4 \cdot \frac{n(n+1)}{2} = 2n^2 + 2n$

## 四、标签信息（用于分类和检索）

### 解题方法标签
- 类讨论型（由和求通项）
- 换元法（设 $b_n = \sqrt{a_n}$）
- 公式法（自然数求和公式）
- 裂项相消思想（求差过程）

### 易错点标签
- 混淆 $a_n$ 和 $\sqrt{a_n}$（直接认为 $a_n = n^2 + n$）
- 忘记平方（求出 $\sqrt{a_n} = 2n$ 后直接作为答案）
- 计算 $\frac{a_n}{n}$ 时错误（误认为 $\frac{4n^2}{n} = 4n^2$）
- 求和公式记错（$1+2+\cdots+n = \frac{n(n+1)}{2}$ 记错为 $\frac{n(n-1)}{2}$）

### 题目特征标签
- 质检题（陕西宝鸡）
- 中等偏难（多步转化）
- 根式与多项式混合
- 双重求和结构

## 五、关联拓展

### 相关方法总结

**处理 $\sqrt{a_n}$ 型数列的策略：**
1. 设 $b_n = \sqrt{a_n}$，将问题转化为 $b_n$ 的求和问题
2. 求出 $b_n$ 通项后，通过平方得到 $a_n$
3. 注意定义域：$a_n &gt; 0$ 保证 $\sqrt{a_n}$ 有意义

**常见求和公式：**
- $\sum_{k=1}^n k = \frac{n(n+1)}{2}$
- $\sum_{k=1}^n k^2 = \frac{n(n+1)(2n+1)}{6}$
- $\sum_{k=1}^n k^3 = [\frac{n(n+1)}{2}]^2$
