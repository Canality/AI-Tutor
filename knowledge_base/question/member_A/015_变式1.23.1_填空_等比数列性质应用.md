# 【A_015_3】等比数列性质应用

## 一、题目内容

**题目：**

已知数列 $\{a_n\}$ 为等比数列，若 $a_4 + a_6 = 10$，则 $a_7(a_1 + 2a_3) + a_3 a_9 =$ ______。

## 二、答案与解析

### 答案

$100$

### 解析

**步骤1：展开并利用性质**

$$a_7(a_1 + 2a_3) + a_3 a_9 = a_7 a_1 + 2a_7 a_3 + a_3 a_9$$

利用 $a_m a_n = a_p a_q$（下标和相等）：
- $a_7 a_1 = a_4^2$（因为 $7+1 = 4+4$）
- $a_7 a_3 = a_5^2$（因为 $7+3 = 5+5$）或 $a_4 a_6$
- $a_3 a_9 = a_6^2$（因为 $3+9 = 6+6$）

更优路径：
$$a_7 a_1 = a_4^2$$
$$a_7 a_3 = a_4 a_6$$
$$a_3 a_9 = a_6^2$$

原式 $= a_4^2 + 2a_4 a_6 + a_6^2 = (a_4 + a_6)^2 = 10^2 = 100$。

## 三、分步教学引导

**步骤1：展开式子**
- **引导语**："先展开原式。"
- **期望学生回答**：$a_7 a_1 + 2a_7 a_3 + a_3 a_9$

**步骤2：转化各项**
- **引导语**："将各项转化为 $a_4$ 和 $a_6$ 的表达式。"
- **关键提示**：$a_7 a_1 = a_4^2$，$a_7 a_3 = a_4 a_6$，$a_3 a_9 = a_6^2$
- **期望学生回答**：完全平方形式 $(a_4 + a_6)^2$

**步骤3：计算结果**
- **期望学生回答**：$100$
