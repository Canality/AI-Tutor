# 【A_016_5】等比数列奇偶项和（珠海质检）

## 一、题目内容

&gt; **原题位置**：P16 变式1.25.2
&gt; 
&gt; **来源**：2015年广东珠海质量检测

**题目：**

等比数列 $\{a_n\}$ 共有奇数项，所有奇数项和 $S_{\text{奇}} = 255$，所有偶数项和 $S_{\text{偶}} = -126$，末项是192，则首项 $a_1 =$（ ）。

A. 1  
B. 2  
C. 3  
D. 4

---

## 二、答案与解析

### 答案

**C**

### 解析

**步骤1：设定变量**
设等比数列共有 $2k+1$ 项（奇数项），公比为 $q$。

**步骤2：分析奇偶项结构**
- 奇数项：$a_1, a_3, a_5, \ldots, a_{2k+1}$，共 $k+1$ 项，首项 $a_1$，公比 $q^2$
- 偶数项：$a_2, a_4, a_6, \ldots, a_{2k}$，共 $k$ 项，首项 $a_2 = a_1q$，公比 $q^2$

**步骤3：建立方程**

奇数项和：
$$S_{\text{奇}} = \frac{a_1[1-(q^2)^{k+1}]}{1-q^2} = \frac{a_1(1-q^{2k+2})}{1-q^2} = 255$$

偶数项和：
$$S_{\text{偶}} = \frac{a_2[1-(q^2)^k]}{1-q^2} = \frac{a_1q(1-q^{2k})}{1-q^2} = -126$$

末项：$a_{2k+1} = a_1 q^{2k} = 192$

**步骤4：寻找关系**

注意到 $S_{\text{奇}} = a_1 + a_3 + \cdots + a_{2k+1}$，$S_{\text{偶}} = a_2 + a_4 + \cdots + a_{2k}$。

有 $S_{\text{偶}} = q(a_1 + a_3 + \cdots + a_{2k-1}) = q(S_{\text{奇}} - a_{2k+1})$。

所以：
$$-126 = q(255 - 192) = 63q$$

解得 $q = -2$。

**步骤5：求首项 $a_1$**

由 $S_{\text{奇}} - S_{\text{偶}} = \frac{a_1(1-q^{2k+2}) - a_1q(1-q^{2k})}{1-q^2}$... 较复杂。

改用：$S_{\text{奇}} + S_{\text{偶}} = S_{2k+1} = \frac{a_1(1-q^{2k+1})}{1-q}$。

或利用末项：$a_{2k+1} = a_1 q^{2k} = a_1 \cdot (-2)^{2k} = 192$。

由 $S_{\text{偶}} = q \cdot (\text{奇数项除去最后一项的和})$...

**更简洁的方法：**

$S_{\text{奇}} = a_1 + a_3 + \cdots + a_{2k-1} + a_{2k+1} = (a_1 + a_3 + \cdots + a_{2k-1}) + 192 = 255$

所以 $a_1 + a_3 + \cdots + a_{2k-1} = 63$。

而 $S_{\text{偶}} = q(a_1 + a_3 + \cdots + a_{2k-1}) = -2 \times 63 = -126$，符合。

所以奇数项（除去末项）和为63，末项为192。

这些奇数项（除去末项）是首项 $a_1$，公比 $q^2=4$，共 $k$ 项的等比数列。

$$\frac{a_1(4^k-1)}{4-1} = 63 \Rightarrow \frac{a_1(4^k-1)}{3} = 63 \Rightarrow a_1(4^k-1) = 189$$

又 $a_{2k+1} = a_1 \cdot 4^k = 192$，所以 $a_1 \cdot 4^k = 192$。

两式相减：$a_1 \cdot 4^k - a_1(4^k-1) = 192 - 189 = 3$，所以 $a_1 = 3$。

## 三、分步教学引导（用于AI Agent）

**步骤1：理解奇偶项的构成**
- **引导语**："数列有奇数项 $2k+1$ 项，奇数项有多少项？偶数项有多少项？"
- **关键提示**：奇数项 $k+1$ 个，偶数项 $k$ 个
- **期望学生回答**：奇数项 $k+1$ 项，偶数项 $k$ 项

**步骤2：建立奇偶项关系**
- **引导语**："观察偶数项与奇数项的关系，$a_2$ 与 $a_1$ 什么关系？$a_4$ 与 $a_3$ 什么关系？"
- **关键提示**：$a_{2i} = q \cdot a_{2i-1}$
- **期望学生回答**：$S_{\text{偶}} = q \times (\text{前}k\text{个奇数项的和})$

**步骤3：利用已知条件列方程**
- **引导语**："前 $k$ 个奇数项的和加上末项等于 $S_{\text{奇}} = 255$，所以前 $k$ 个奇数项和是多少？"
- **关键提示**：$255 - 192 = 63$
- **期望学生回答**：前 $k$ 个奇数项和为63

**步骤4：求公比 $q$**
- **引导语**："$S_{\text{偶}} = q \times 63 = -126$，求 $q$。"
- **期望学生回答**：$q = -2$

**步骤5：利用末项与部分和关系**
- **引导语**："奇数项构成公比为 $q^2=4$ 的等比数列。前 $k$ 个奇数项和为 $\frac{a_1(4^k-1)}{3} = 63$，且第 $k+1$ 个奇数项（末项）$a_1 \cdot 4^k = 192$。如何求 $a_1$？"
- **关键提示**：$a_1 \cdot 4^k - a_1(4^k-1) = a_1$
- **期望学生回答**：$192 - 189 = 3$，所以 $a_1 = 3$

## 四、标签信息（用于分类和检索）

### 解题方法标签
- 等比数列奇偶项和性质（$S_{\text{偶}} = q \cdot S_{\text{奇}}'$）
- 方程思想（建立关于 $a_1$ 和 $k$ 的方程）
- 整体代换（将 $4^k$ 作为整体处理）

### 易错点标签
- 混淆项数（奇数项比偶数项多一项）
- 奇数项公比错误（奇数项公比是 $q^2$ 不是 $q$）
- 符号错误（$q=-2$ 时负数运算出错）
- 不能建立 $S_{\text{偶}}$ 与部分奇数项和的关系

### 题目特征标签
- 地方质检题（珠海）
- 等比数列性质深度应用
- 奇偶项和分类讨论
- 中等偏难

## 五、关联拓展

### 奇偶项和的重要结论

对于等比数列（公比 $q \neq \pm 1$）：
1. 若共有 $2n$ 项：$\frac{S_{\text{偶}}}{S_{\text{奇}}} = q$
2. 若共有 $2n+1$ 项：$\frac{S_{\text{偶}}}{S_{\text{奇}} - a_{2n+1}} = q$（或 $S_{\text{偶}} = q(S_{\text{奇}} - a_{\text{末}})$）

**推导：**
$S_{\text{偶}} = a_2 + a_4 + \cdots = q(a_1 + a_3 + \cdots) = q \cdot S_{\text{奇对应项}}$
