# 【A_011】等差数列最值问题（配方法）

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P11 例1.15
&gt; 
&gt; **来源**：2018年新课标II卷文理科第17题

**题目：**

记 $S_n$ 为等差数列 $\{a_n\}$ 的前 $n$ 项和。已知 $a_1 = -7$，$S_3 = -15$。

(I) 求 $\{a_n\}$ 的通项公式；

(II) 求 $S_n$，并求 $S_n$ 的最小值。

---

## 二、答案与解析

### 答案

(I) $a_n = 2n - 9$

(II) $S_n = n^2 - 8n$，最小值为 $-16$（当 $n=4$ 时取得）

### 解析

**步骤1：求通项公式（第I问）**

设等差数列 $\{a_n\}$ 的公差为 $d$。

由 $S_3 = -15$，根据求和公式：
$$S_3 = 3a_1 + \frac{3 \times 2}{2}d = 3a_1 + 3d = -15$$

代入 $a_1 = -7$：
$$3(-7) + 3d = -15$$
$$-21 + 3d = -15$$
$$3d = 6$$
$$d = 2$$

因此通项公式为：
$$a_n = a_1 + (n-1)d = -7 + 2(n-1) = 2n - 9$$

**步骤2：求前n项和并配方（第II问）**

$$S_n = na_1 + \frac{n(n-1)}{2}d = -7n + \frac{n(n-1)}{2} \times 2 = -7n + n(n-1) = n^2 - 8n$$

配方得：
$$S_n = n^2 - 8n = (n-4)^2 - 16$$

**步骤3：确定最小值**

由于 $n \in \mathbb{N}^*$，且二次函数开口向上，对称轴为 $n=4$。

当 $n=4$ 时，$S_n$ 取得最小值 $-16$。

## 三、分步教学引导

**步骤1：分析已知条件**
- **引导语**："题目给出了等差数列的哪两个具体数值？"
- **关键提示**：首项 $a_1 = -7$，前三项和 $S_3 = -15$
- **期望学生回答**：$a_1 = -7$，$S_3 = -15$

**步骤2：建立方程求公差**
- **引导语**："利用 $S_3$ 的公式，写出关于 $d$ 的方程并求解。"
- **关键提示**：$S_3 = 3a_1 + 3d$
- **期望学生回答**：$3(-7) + 3d = -15$，解得 $d = 2$

**步骤3：写出通项公式**
- **引导语**："现在我们知道 $a_1 = -7$，$d = 2$，写出 $a_n$ 的表达式。"
- **关键提示**：$a_n = a_1 + (n-1)d$
- **期望学生回答**：$a_n = -7 + 2(n-1) = 2n - 9$

**步骤4：求 $S_n$ 的表达式**
- **引导语**："利用求和公式写出 $S_n$，并尝试配方。"
- **关键提示**：$S_n = n^2 - 8n$，配方为 $(n-4)^2 - 16$
- **期望学生回答**：$S_n = (n-4)^2 - 16$

**步骤5：确定最值**
- **引导语**："由于 $n$ 必须是正整数，当 $n$ 取何值时 $S_n$ 最小？最小值是多少？"
- **关键提示**：对称轴 $n=4$ 恰好是正整数
- **期望学生回答**：$n=4$ 时，最小值为 $-16$

## 六、备注

- **核心思想**：等差数列前 $n$ 项和是关于 $n$ 的二次函数（无常数项），可通过配方或求导求最值。
- **教学重点**：强调 $n \in \mathbb{N}^*$ 的约束条件，若对称轴非整数（如 $n=4.5$），需比较 $n=4$ 和 $n=5$ 的函数值。
