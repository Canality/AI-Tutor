# 【A_002】等差数列与根式构造（镇江一模）

## 一、题目内容

&gt; **原题位置**：P2 例1.2
&gt; 
&gt; **来源**：2019年镇江一模第11题

**题目：**

已知等差数列 $\{a_n\}$ 的公差为 $d(d \neq 0)$，前 $n$ 项和为 $S_n$，且数列 $\{\sqrt{S_n + n}\}$ 也是公差为 $d$ 的等差数列，则 $d = $ ______。

---

## 二、答案与解析

### 答案

$$\frac{1}{2}$$

### 解析

**步骤1：表示 $S_n$**

设等差数列 $\{a_n\}$ 的首项为 $a_1$，公差为 $d(d \neq 0)$。

根据等差数列前 $n$ 项和公式：
$$S_n = na_1 + \frac{n(n-1)}{2}d$$

**步骤2：分析新数列的结构**

令 $b_n = \sqrt{S_n + n}$，则 $\{b_n\}$ 也是公差为 $d$ 的等差数列。

因此：
$$b_n = b_1 + (n-1)d$$

**步骤3：建立等式关系**

由 $b_n = \sqrt{S_n + n}$，两边平方得：
$$S_n + n = [b_1 + (n-1)d]^2$$

将 $S_n = na_1 + \frac{n(n-1)}{2}d$ 代入：
$$na_1 + \frac{n(n-1)}{2}d + n = [b_1 + (n-1)d]^2$$

**步骤4：确定 $b_1$ 的值**

当 $n = 1$ 时：
$$b_1 = \sqrt{S_1 + 1} = \sqrt{a_1 + 1}$$

所以 $b_1^2 = a_1 + 1$，即 $a_1 = b_1^2 - 1$。

**步骤5：展开并比较系数**

右边展开：
$$[b_1 + (n-1)d]^2 = d^2n^2 + (2b_1d - 2d^2)n + (b_1^2 - 2b_1d + d^2)$$

左边整理：
$$\frac{d}{2}n^2 + \left(a_1 - \frac{d}{2} + 1\right)n$$

**步骤6：对应系数相等**

比较 $n^2$ 的系数：
$$\frac{d}{2} = d^2$$

解得 $d = 0$（舍去，因为 $d \neq 0$）或 $d = \frac{1}{2}$。

验证：当 $d = \frac{1}{2}$ 时，可以解得 $a_1 = -\frac{3}{4}$，满足所有条件。

故 $d = \frac{1}{2}$。

## 三、分步教学引导（用于AI Agent）

### 第(I)问引导（分析结构）

**步骤1：识别已知条件**
- **引导语**："题目告诉我们哪些关键信息？特别是关于两个等差数列的关系。"
- **关键提示**：原数列公差为 $d$，新数列 $\{\sqrt{S_n+n}\}$ 公差也为 $d$
- **期望学生回答**：$S_n$ 是等差数列前 $n$ 项和，$\sqrt{S_n+n}$ 也是等差数列且公差为 $d$

**步骤2：设新数列简化问题**
- **引导语**："根号看起来比较复杂，我们可以设一个新数列来简化，设什么比较好？"
- **关键提示**：令 $b_n = \sqrt{S_n + n}$
- **期望学生回答**：设 $b_n = \sqrt{S_n + n}$，则 $\{b_n\}$ 是公差为 $d$ 的等差数列

**步骤3：写出 $b_n$ 的通项**
- **引导语**："既然 $\{b_n\}$ 是等差数列，它的通项公式怎么写？"
- **关键提示**：$b_n = b_1 + (n-1)d$，其中 $b_1 = \sqrt{a_1 + 1}$
- **期望学生回答**：$b_n = b_1 + (n-1)d$

**步骤4：建立方程**
- **引导语**："由 $b_n = \sqrt{S_n + n}$，两边平方后得到什么？把 $S_n$ 的公式代进去。"
- **关键提示**：$S_n + n = [b_1 + (n-1)d]^2$
- **期望学生回答**：得到 $na_1 + \frac{n(n-1)}{2}d + n = [b_1 + (n-1)d]^2$

**步骤5：比较系数**
- **引导语**："右边展开后是关于 $n$ 的二次多项式，左边也是，比较 $n^2$ 的系数能得到什么？"
- **关键提示**：左边 $n^2$ 系数是 $\frac{d}{2}$，右边是 $d^2$
- **期望学生回答**：$\frac{d}{2} = d^2$，解得 $d = \frac{1}{2}$（舍去0）

## 五、关联拓展

### 相关方法总结

**构造法等差数列的常见类型：**
1. $\{a_n + c\}$ 是等差数列（平移构造）
2. $\{ka_n\}$ 是等差数列（伸缩构造）
3. $\{a_n^2\}$ 或 $\{\sqrt{a_n}\}$ 等特殊形式（本题属于此类）

**多项式恒等定理：**
若两个多项式对所有 $n$ 都相等，则对应次数的系数必须相等。
