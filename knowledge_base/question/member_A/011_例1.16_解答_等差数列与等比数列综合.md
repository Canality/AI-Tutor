# 【A_011_2】等差数列与等比数列综合（北京卷）

## 一、题目内容

&gt; **原题位置**：P11 例1.16
&gt; 
&gt; **来源**：2019年北京卷理科第16题

**题目：**

设 $\{a_n\}$ 是等差数列，$a_1 = -10$，且 $a_2 + 10, a_3 + 8, a_4 + 6$ 成等比数列。

(I) 求 $\{a_n\}$ 的通项公式；

(II) 记 $\{a_n\}$ 的前 $n$ 项和为 $S_n$，求 $S_n$ 的最小值。

---

## 二、答案与解析

### 答案

(I) $a_n = 2n - 12$

(II) $S_n$ 的最小值为 $-30$

### 解析

**(I) 求通项公式**

**步骤1：设公差并表示三项**

设等差数列 $\{a_n\}$ 的公差为 $d$。

- $a_2 + 10 = (a_1 + d) + 10 = -10 + d + 10 = d$
- $a_3 + 8 = (a_1 + 2d) + 8 = -10 + 2d + 8 = 2d - 2$  
- $a_4 + 6 = (a_1 + 3d) + 6 = -10 + 3d + 6 = 3d - 4$

**步骤2：利用等比中项列方程**

因为 $a_2 + 10, a_3 + 8, a_4 + 6$ 成等比数列，所以：
$$(a_3 + 8)^2 = (a_2 + 10)(a_4 + 6)$$

即：
$$(2d - 2)^2 = d(3d - 4)$$

**步骤3：解方程**

展开得：
$$4d^2 - 8d + 4 = 3d^2 - 4d$$

整理：
$$d^2 - 4d + 4 = 0$$

$$(d - 2)^2 = 0$$

解得 $d = 2$。

**步骤4：写出通项**

$$a_n = a_1 + (n-1)d = -10 + 2(n-1) = 2n - 12$$

**(II) 求 $S_n$ 的最小值**

**步骤1：建立 $S_n$ 表达式**

$$S_n = \frac{n(a_1 + a_n)}{2} = \frac{n(-10 + 2n - 12)}{2} = \frac{n(2n - 22)}{2} = n^2 - 11n$$

**步骤2：配方求最值**

$$S_n = n^2 - 11n = \left(n - \frac{11}{2}\right)^2 - \frac{121}{4}$$

**步骤3：确定整数最值点**

由于 $n \in \mathbb{N}^*$，且 $\frac{11}{2} = 5.5$，所以 $n = 5$ 或 $n = 6$ 时 $S_n$ 取最小值。

$$S_5 = 25 - 55 = -30$$
$$S_6 = 36 - 66 = -30$$

故 $S_n$ 的最小值为 $-30$。

## 三、分步教学引导（用于AI Agent）

### 第(I)问引导

**步骤1：表示新三项**
- **引导语**："设公差为 $d$，用 $a_1$ 和 $d$ 表示 $a_2+10, a_3+8, a_4+6$。"
- **关键提示**：$a_2+10 = d$，$a_3+8 = 2d-2$，$a_4+6 = 3d-4$
- **期望学生回答**：得到关于 $d$ 的三个表达式

**步骤2：列等比条件**
- **引导语**："三项成等比数列，中间项的平方等于什么？"
- **关键提示**：等比中项性质 $b^2 = ac$
- **期望学生回答**：$(2d-2)^2 = d(3d-4)$

**步骤3：解方程**
- **引导语**："展开整理这个方程，注意可能有重根。"
- **关键提示**：完全平方公式
- **期望学生回答**：$d = 2$（重根）

**步骤4：写通项**
- **引导语**："代入 $a_1 = -10, d = 2$ 写出 $a_n$。"
- **关键提示**：$a_n = -10 + 2(n-1)$
- **期望学生回答**：$a_n = 2n - 12$

### 第(II)问引导

**步骤1：建立 $S_n$ 函数**
- **引导语**："用公式 $S_n = \frac{n(a_1+a_n)}{2}$ 或 $S_n = na_1 + \frac{n(n-1)}{2}d$ 建立表达式。"
- **关键提示**：代入 $a_1 = -10, d = 2$ 或 $a_n = 2n-12$
- **期望学生回答**：$S_n = n^2 - 11n$

**步骤2：配方或求对称轴**
- **引导语**："这是一个关于 $n$ 的二次函数，开口向上，对称轴在哪里？"
- **关键提示**：$n = \frac{11}{2} = 5.5$
- **期望学生回答**：对称轴 $n = 5.5$

**步骤3：取整求最值**
- **引导语**："离 $5.5$ 最近的正整数是哪两个？分别计算 $S_5$ 和 $S_6$。"
- **关键提示**：$n$ 必须是正整数，$5$ 和 $6$ 距离相等
- **期望学生回答**：$n = 5$ 或 $6$，最小值 $-30$

## 四、标签信息（用于分类和检索）

### 解题方法标签
- 基本量法（设 $a_1, d$）
- 等比中项性质（$b^2 = ac$）
- 配方法求二次函数最值
- 邻项法验证（$a_n \leq 0, a_{n+1} \geq 0$）

### 易错点标签
- 计算 $a_2+10$ 时忘记 $a_1=-10$ 已经包含在内（误写为 $-10+d+10$ 是对的，但要化简）
- 解方程时展开错误 $(2d-2)^2 = 4d^2 - 4d + 4$（漏乘或符号错）
- 最值点取 $n=5.5$ 而非整数 $5$ 或 $6$
- 未验证 $n=5,6$ 时 $S_n$ 相等

- 等差与等比综合
- 双问解答题
- 最值问题（配方法）

## 五、关联拓展

### 相关公式总结

**等比中项：**
若 $a, b, c$ 成等比数列，则 $b^2 = ac$（注意 $b = \pm\sqrt{ac}$，但本题通过方程解确定符号）

**等差数列 $S_n$ 最值判定：**
- 配方法：$S_n = \frac{d}{2}n^2 + (a_1 - \frac{d}{2})n$，对称轴 $n = \frac{1}{2} - \frac{a_1}{d}$
- 邻项法：找到 $a_n \leq 0$ 且 $a_{n+1} \geq 0$ 的 $n$

**验证邻项法：**
$a_n = 2n - 12 \leq 0 \Rightarrow n \leq 6$
$a_{n+1} = 2(n+1) - 12 \geq 0 \Rightarrow n \geq 5$
所以 $n = 5$ 或 $6$（$a_6 = 0$，所以 $S_5 = S_6$）
