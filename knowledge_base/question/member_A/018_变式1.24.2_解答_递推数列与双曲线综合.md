# 【A_018_3】递推数列与双曲线综合（四川卷）

## 一、题目内容

&gt; **原题位置**：P18 变式1.24.2
&gt; 
&gt; **来源**：2016年四川文第19题

**题目：**

已知数列 $\{a_n\}$ 的首项为 $1$，$S_n$ 为数列 $\{a_n\}$ 的前 $n$ 项和，$S_{n+1} = qS_n + 1$，其中 $q &gt; 0$，$n \in \mathbb{N}^*$。设双曲线 $x^2 - \frac{y^2}{a_n^2} = 1$ 的离心率为 $e_n$，且 $e_2 = 2$，求 $e_1^2 + e_2^2 + \cdots + e_n^2$。

---

## 二、答案与解析

### 答案

$$e_1^2 + e_2^2 + \cdots + e_n^2 = n + \frac{1}{2}(3^n - 1)$$

### 解析

**步骤1：求数列 $\{a_n\}$ 的通项公式**

由 $S_{n+1} = qS_n + 1$ ①

当 $n \geq 2$ 时，$S_n = qS_{n-1} + 1$ ②

①-②得：$a_{n+1} = qa_n$（$n \geq 2$）。

又 $S_2 = qS_1 + 1$，即 $a_1 + a_2 = qa_1 + 1$。

因为 $a_1 = 1$，所以 $1 + a_2 = q + 1$，得 $a_2 = q$。

所以 $\frac{a_2}{a_1} = q$，满足 $a_{n+1} = qa_n$。

因此 $\{a_n\}$ 是首项为 $1$，公比为 $q$ 的等比数列，$a_n = q^{n-1}$。

**步骤2：利用离心率条件求 $q$**

双曲线 $x^2 - \frac{y^2}{a_n^2} = 1$ 中，$a^2 = 1$，$b^2 = a_n^2$。

离心率 $e_n = \sqrt{1 + \frac{b^2}{a^2}} = \sqrt{1 + a_n^2} = \sqrt{1 + q^{2(n-1)}}$。

由 $e_2 = 2$，得 $\sqrt{1 + q^2} = 2$，即 $1 + q^2 = 4$，$q^2 = 3$。

因为 $q &gt; 0$，所以 $q = \sqrt{3}$。

**步骤3：计算 $e_n^2$ 的表达式**

$$e_n^2 = 1 + q^{2(n-1)} = 1 + 3^{n-1}$$

**步骤4：求和**

$$e_1^2 + e_2^2 + \cdots + e_n^2 = \sum_{k=1}^n (1 + 3^{k-1}) = n + (1 + 3 + 3^2 + \cdots + 3^{n-1})$$

$$= n + \frac{1 \cdot (3^n - 1)}{3 - 1} = n + \frac{3^n - 1}{2}$$

## 三、分步教学引导（用于AI Agent）

### 整体思路引导

**步骤1：识别递推类型**
- **引导语**："$S_{n+1} = qS_n + 1$ 是什么类型的递推？如何求 $a_n$？"
- **关键提示**：再写一式 $S_n = qS_{n-1} + 1$ 相减，得 $a_{n+1} = qa_n$
- **期望学生回答**：$\{a_n\}$ 是等比数列，$a_n = q^{n-1}$

**步骤2：验证首项**
- **引导语**："需要验证 $n=1$ 时是否满足等比关系吗？"
- **关键提示**：计算 $a_2$ 与 $a_1$ 的关系
- **期望学生回答**：$a_2 = q$，所以 $\frac{a_2}{a_1} = q$，满足等比数列定义

**步骤3：建立离心率表达式**
- **引导语**："双曲线 $x^2 - \frac{y^2}{a_n^2} = 1$ 的离心率公式是什么？"
- **关键提示**：$e = \sqrt{1 + \frac{b^2}{a^2}}$，这里 $a^2=1$，$b^2=a_n^2$
- **期望学生回答**：$e_n = \sqrt{1 + a_n^2} = \sqrt{1 + q^{2(n-1)}}$

**步骤4：求参数 $q$**
- **引导语**："利用 $e_2 = 2$ 建立方程。"
- **期望学生回答**：$\sqrt{1 + q^2} = 2$，$q = \sqrt{3}$（舍去负值）

**步骤5：分组求和**
- **引导语**："$e_n^2 = 1 + 3^{n-1}$，如何求前 $n$ 项和？"
- **关键提示**：拆分为常数列和等比数列分别求和
- **期望学生回答**：$n + \frac{3^n - 1}{2}$

## 五、关联拓展

### 重要公式
**双曲线离心率：**
对于 $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$，有 $e = \sqrt{1 + \frac{b^2}{a^2}} &gt; 1$。

**递推式 $S_{n+1} = pS_n + q$ 的处理：**
通过作差转化为 $a_{n+1} = pa_n$（$n \geq 2$），需验证 $a_2/a_1$ 是否等于 $p$。
