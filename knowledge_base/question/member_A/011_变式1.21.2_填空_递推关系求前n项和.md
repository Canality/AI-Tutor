# 【A_011_4】递推关系求前n项和（2018合肥质检）

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P11 变式1.21.2
&gt; 
&gt; **来源**：2018年合肥质检

**题目：**

已知数列 $\{a_n\}$ 中，$a_1=2$，且 $\frac{a_{n+1}^2}{a_n} = 4(a_{n+1}-a_n)$（$n \in \mathbb{N}^*$），则其前9项和 $S_9 =$ ______。

---

## 二、答案与解析

### 答案

**1022**

### 解析

**步骤1：变形递推关系**

由 $\frac{a_{n+1}^2}{a_n} = 4(a_{n+1}-a_n)$，整理得：
$$a_{n+1}^2 = 4a_n(a_{n+1}-a_n)$$
$$a_{n+1}^2 = 4a_na_{n+1} - 4a_n^2$$
$$a_{n+1}^2 - 4a_na_{n+1} + 4a_n^2 = 0$$
$$(a_{n+1} - 2a_n)^2 = 0$$

**步骤2：确定数列类型**

所以 $a_{n+1} = 2a_n$。

这说明 $\{a_n\}$ 是等比数列，首项 $a_1 = 2$，公比 $q = 2$。

**步骤3：计算 $S_9$**

$$S_9 = \frac{a_1(1-q^9)}{1-q} = \frac{2(1-2^9)}{1-2} = \frac{2(1-512)}{-1} = 2 \times 511 = 1022$$

故填 **1022**。

## 三、分步教学引导（用于AI Agent）

### 整体思路引导

**步骤1：分析递推式结构**
- **引导语**："观察递推式 $\frac{a_{n+1}^2}{a_n} = 4(a_{n+1}-a_n)$，尝试去分母并整理成关于 $a_{n+1}$ 和 $a_n$ 的方程。"
- **关键提示**：两边同乘 $a_n$
- **期望学生回答**：$a_{n+1}^2 = 4a_na_{n+1} - 4a_n^2$

**步骤2：因式分解**
- **引导语**："将上式看作关于 $a_{n+1}$ 的二次方程，尝试配方或因式分解。"
- **关键提示**：完全平方公式
- **期望学生回答**：$(a_{n+1}-2a_n)^2 = 0$，即 $a_{n+1}=2a_n$

**步骤3：识别数列类型**
- **引导语**："由 $a_{n+1}=2a_n$ 和 $a_1=2$，这是什么数列？"
- **期望学生回答**：首项为2，公比为2的等比数列

**步骤4：计算前9项和**
- **引导语**："使用等比数列求和公式计算 $S_9$，注意 $2^9=512$。"
- **期望学生回答**：$S_9 = \frac{2(1-512)}{-1} = 1022$

### 递推关系变形技巧

遇到分式型递推关系 $\frac{a_{n+1}^2}{a_n} = f(a_n, a_{n+1})$，通常：
1. 去分母化为整式方程
2. 整理为齐次方程或完全平方形式
3. 识别特殊数列（等差、等比）
