# 【A_020_1】数列证明与通项求解（2019全国II卷）

## 一、题目内容

&gt; **原题位置**：P20 例1.26
&gt; 
&gt; **来源**：2019年全国II卷理科第19题

**题目：**

已知数列 $\{a_n\}$ 和 $\{b_n\}$ 满足 $a_1=1$，$b_1=0$，且
$$4a_{n+1}=3a_n-b_n+4, \quad 4b_{n+1}=3b_n-a_n-4$$

(I) 证明：$\{a_n+b_n\}$ 是等比数列，$\{a_n-b_n\}$ 是等差数列；

(II) 求 $\{a_n\}$ 和 $\{b_n\}$ 的通项公式。

---

## 二、答案与解析

### 答案

**(I)** 证明见解析；**(II)** $a_n=\frac{1}{2^n}+n-\frac{1}{2}$，$b_n=\frac{1}{2^n}-n+\frac{1}{2}$

### 解析

**步骤1：构造和数列证明等比**

将两个递推关系式相加：
$$4(a_{n+1}+b_{n+1}) = 3a_n-b_n+4 + 3b_n-a_n-4 = 2(a_n+b_n)$$

整理得：
$$\frac{a_{n+1}+b_{n+1}}{a_n+b_n} = \frac{1}{2}$$

又 $a_1+b_1=1+0=1$，故 $\{a_n+b_n\}$ 是**首项为1，公比为 $\frac{1}{2}$ 的等比数列**。

**步骤2：构造差数列证明等差**

将两个递推关系式相减：
$$4(a_{n+1}-b_{n+1}) = (3a_n-b_n+4) - (3b_n-a_n-4) = 4(a_n-b_n)+8$$

整理得：
$$(a_{n+1}-b_{n+1}) - (a_n-b_n) = 2$$

又 $a_1-b_1=1-0=1$，故 $\{a_n-b_n\}$ 是**首项为1，公差为2的等差数列**。

**步骤3：求通项公式**

由(I)可知：
- $a_n+b_n = 1 \cdot \left(\frac{1}{2}\right)^{n-1} = \frac{1}{2^{n-1}}$
- $a_n-b_n = 1 + 2(n-1) = 2n-1$

联立求解：
$$a_n = \frac{(a_n+b_n)+(a_n-b_n)}{2} = \frac{\frac{1}{2^{n-1}} + 2n-1}{2} = \frac{1}{2^n} + n - \frac{1}{2}$$

$$b_n = \frac{(a_n+b_n)-(a_n-b_n)}{2} = \frac{\frac{1}{2^{n-1}} - (2n-1)}{2} = \frac{1}{2^n} - n + \frac{1}{2}$$

验证 $n=1$：$a_1=\frac{1}{2}+1-\frac{1}{2}=1$，$b_1=\frac{1}{2}-1+\frac{1}{2}=0$，符合题意。

## 三、分步教学引导（用于AI Agent）

**步骤1：观察递推结构**
- **引导语**："题目给出了两个数列的交叉递推关系，直接求通项比较困难。观察系数特点，思考如何通过加减消元简化问题？"
- **关键提示**：系数对称（3和-1），考虑构造 $a_n+b_n$ 和 $a_n-b_n$
- **期望学生回答**：两式相加和相减可以消去交叉项

**步骤2：证明等比数列**
- **引导语**："将两式相加后，能得到关于 $a_n+b_n$ 的什么关系？计算相邻两项的比值。"
- **关键提示**：$4(a_{n+1}+b_{n+1})=2(a_n+b_n)$
- **期望学生回答**：比值为 $\frac{1}{2}$，是等比数列

**步骤3：证明等差数列**
- **引导语**："将两式相减后，观察 $a_{n+1}-b_{n+1}$ 与 $a_n-b_n$ 的差是否为常数。"
- **关键提示**：差为2
- **期望学生回答**：公差为2的等差数列

**步骤4：求解通项**
- **引导语**："现在我们有了 $a_n+b_n$ 和 $a_n-b_n$ 的表达式，如何解出 $a_n$ 和 $b_n$？"
- **关键提示**：联立方程组，加减消元
- **期望学生回答**：$a_n=\frac{1}{2}[(a_n+b_n)+(a_n-b_n)]$
