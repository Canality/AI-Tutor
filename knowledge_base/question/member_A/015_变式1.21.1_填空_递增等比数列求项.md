# 【A_015】递增等比数列基本量计算

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P15 变式1.21.1
&gt; 
&gt; **来源**：2019年南京一模第8题

**题目：**

已知等比数列 $\{a_n\}$ 为单调递增数列，设其前 $n$ 项和为 $S_n$，若 $a_2 = 2$，$S_3 = 7$，则 $a_5$ 的值为 ______。

---

## 二、答案与解析

### 答案

$16$

### 解析

**步骤1：设定基本量建立方程**

设等比数列 $\{a_n\}$ 的首项为 $a_1$，公比为 $q$。

由 $a_2 = 2$ 得：
$$a_1 q = 2 \quad \text{①}$$

由 $S_3 = 7$ 得：
$$a_1 + a_2 + a_3 = 7$$
即：
$$a_1 + 2 + a_1 q^2 = 7$$
$$a_1(1 + q^2) = 5 \quad \text{②}$$

**步骤2：求解公比 $q$**

由①得 $a_1 = \frac{2}{q}$，代入②：
$$\frac{2}{q}(1 + q^2) = 5$$
$$2(1 + q^2) = 5q$$
$$2q^2 - 5q + 2 = 0$$

解得：
$$(2q - 1)(q - 2) = 0$$
所以 $q = 2$ 或 $q = \frac{1}{2}$。

**步骤3：利用单调性确定唯一解**

当 $q = 2$ 时，$a_1 = 1$，数列为 $1, 2, 4, 8, \ldots$，单调递增，符合条件。

当 $q = \frac{1}{2}$ 时，$a_1 = 4$，数列为 $4, 2, 1, \frac{1}{2}, \ldots$，单调递减，不符合条件，舍去。

因此 $a_1 = 1$，$q = 2$。

**步骤4：计算 $a_5$**

$$a_5 = a_1 q^4 = 1 \times 2^4 = 16$$

## 三、分步教学引导（用于AI Agent）

**步骤1：翻译条件**
- **引导语**："题目告诉我们等比数列是单调递增的，还知道哪些具体数值？"
- **关键提示**：$a_2=2$，$S_3=7$
- **期望学生回答**：第二项为2，前三项和为7

**步骤2：设量列方程**
- **引导语**："设首项$a_1$和公比$q$，用它们表示$a_2$和$S_3$。"
- **关键提示**：$a_2=a_1q=2$，$S_3=a_1(1+q+q^2)=7$或$a_1+2+a_1q^2=7$
- **期望学生回答**：建立方程组$\begin{cases} a_1q=2 \\ a_1(1+q^2)=5 \end{cases}$

**步骤3：解方程**
- **引导语**："消去$a_1$，得到关于$q$的方程并求解。"
- **关键提示**：代入得$2q^2-5q+2=0$
- **期望学生回答**：$q=2$或$q=\frac{1}{2}$

**步骤4：利用单调性筛选**
- **引导语**："两个$q$值都满足方程，但题目还有哪个限制条件？"
- **关键提示**：单调递增数列
- **期望学生回答**：$q=2$时递增，$q=\frac{1}{2}$时递减，故取$q=2$

**步骤5：计算结果**
- **引导语**："现在计算$a_5$的值。"
- **关键提示**：$a_5=a_1q^4=1\times 16$
- **期望学生回答**：$16$

- 方程思想

### 易错点标签
- 忽略单调性条件（得到两个答案未舍去）
- 混淆$S_3$与$a_3$（误将$S_3=7$当作$a_3=7$）

### 题目特征标签
- 模拟题（南京一模）
- 等比数列单调性
- 基础计算

## 五、关联拓展

**等比数列单调性条件：**
- $a_1&gt;0$且$q&gt;1$：递增
- $a_1&gt;0$且$0&lt;q&lt;1$：递减
- $a_1&lt;0$且$q&gt;1$：递减
- $a_1&lt;0$且$0&lt;q&lt;1$：递增
