# 【A_019_2】等比数列片段和比例（苏州一模）

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P19 变式1.25.1
&gt; 
&gt; **来源**：2019年苏州一模第11题

**题目：**

设 $S_n$ 是等比数列 $\{a_n\}$ 的前 $n$ 项和，若 $\frac{S_5}{S_{10}} = \frac{1}{3}$，则 $\frac{S_5}{S_{20}+S_{10}} = $ ______。

---

## 二、答案与解析

### 答案

$$\frac{1}{18}$$

### 解析

**步骤1：利用片段和性质**

在等比数列中，$S_5$，$S_{10}-S_5$，$S_{15}-S_{10}$，$S_{20}-S_{15}$ 成等比数列。

设 $S_5 = k$，由 $\frac{S_5}{S_{10}} = \frac{1}{3}$，得 $S_{10} = 3k$。

所以 $S_{10} - S_5 = 2k$。

**步骤2：求后续片段**

公比 $q' = \frac{S_{10}-S_5}{S_5} = \frac{2k}{k} = 2$。

所以：
- $S_{15} - S_{10} = 2k \cdot 2 = 4k$，所以 $S_{15} = 3k + 4k = 7k$
- $S_{20} - S_{15} = 4k \cdot 2 = 8k$，所以 $S_{20} = 7k + 8k = 15k$

**步骤3：计算目标比值**

$$\frac{S_5}{S_{20}+S_{10}} = \frac{k}{15k + 3k} = \frac{k}{18k} = \frac{1}{18}$$

## 三、分步教学引导（用于AI Agent）

### 整体思路引导

**步骤1：设定变量**
- **引导语**："设 $S_5 = k$，根据已知条件，$S_{10}$ 是多少？"
- **期望学生回答**：$S_{10} = 3k$

**步骤2：识别片段和关系**
- **引导语**："$S_5, S_{10}-S_5, S_{15}-S_{10}, S_{20}-S_{15}$ 构成什么数列？"
- **关键提示**：等比数列，首项 $k$，第二项 $2k$
- **期望学生回答**：等比数列，公比为2

**步骤3：递推计算**
- **引导语**："依次计算 $S_{15}$ 和 $S_{20}$。"
- **期望学生回答**：$S_{15}=7k$，$S_{20}=15k$

**步骤4：代入求值**
- **引导语**："计算 $\frac{S_5}{S_{20}+S_{10}}$。"
- **期望学生回答**：$\frac{k}{15k+3k} = \frac{1}{18}$

## 四、标签信息（用于分类和检索）

- 片段和性质法

### 易错点标签
- 直接认为 $S_5, S_{10}, S_{20}$ 成等比（错解为 $\frac{1}{3+9}=\frac{1}{12}$ 等）

- 混淆 $S_{20}+S_{10}$ 与 $S_{20}-S_{10}$
