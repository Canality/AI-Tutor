# 【A_009】等差数列基本量与最值计算（2019北京卷）

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P9 变式1.16.1
&gt; 
&gt; **来源**：2019年北京卷理科第10题

**题目：**

设等差数列 $\{a_n\}$ 的前 $n$ 项和为 $S_n$，若 $a_2=-3, S_5=-10$，则 $a_5=$ ______，$S_n$ 的最小值为 ______。

---

## 二、答案与解析

### 答案

$$0; \quad -10$$

### 解析

**步骤1：建立方程组求解基本量**

设等差数列首项为 $a_1$，公差为 $d$。

根据已知条件：
- $a_2 = a_1 + d = -3$  ……①
- $S_5 = 5a_1 + \frac{5 \times 4}{2}d = 5a_1 + 10d = -10$

化简第二个方程得：
- $a_1 + 2d = -2$  ……②

**步骤2：求解首项和公差**

由② - ①得：
$$(a_1 + 2d) - (a_1 + d) = -2 - (-3)$$
$$d = 1$$

将 $d = 1$ 代入①得：
$$a_1 = -3 - 1 = -4$$

**步骤3：计算 $a_5$**

$$a_5 = a_1 + 4d = -4 + 4 \times 1 = 0$$

**步骤4：求 $S_n$ 的最小值**

$$S_n = na_1 + \frac{n(n-1)}{2}d = -4n + \frac{n(n-1)}{2} = \frac{n^2 - 9n}{2}$$

配方得：
$$S_n = \frac{1}{2}\left(n - \frac{9}{2}\right)^2 - \frac{81}{8}$$

由于 $n \in \mathbb{N}^*$，且二次函数对称轴为 $n = 4.5$，因此当 $n = 4$ 或 $n = 5$ 时，$S_n$ 取得最小值。

计算验证：
- $S_4 = \frac{16 - 36}{2} = -10$
- $S_5 = \frac{25 - 45}{2} = -10$

故 $S_n$ 的最小值为 $-10$。

## 三、分步教学引导（用于AI Agent）

### 整体思路引导

**步骤1：识别已知条件**
- **引导语**："题目给出了等差数列的哪些信息？$a_2$ 和 $S_5$ 分别代表什么？"
- **关键提示**：$a_2 = a_1 + d = -3$，$S_5 = 5a_1 + 10d = -10$
- **期望学生回答**：可以列出关于 $a_1$ 和 $d$ 的方程组

**步骤2：解方程组**
- **引导语**："联立这两个方程，如何快速消元求解 $d$ 和 $a_1$？"
- **关键提示**：用第二个方程减去第一个方程的适当倍数
- **期望学生回答**：$d = 1$，$a_1 = -4$

**步骤3：计算 $a_5$**
- **引导语**："已知首项和公差，如何求第5项？使用什么公式？"
- **关键提示**：$a_n = a_1 + (n-1)d$
- **期望学生回答**：$a_5 = -4 + 4 \times 1 = 0$

**步骤4：分析和的最小值**
- **引导语**："$S_n$ 是关于 $n$ 的什么函数？如何求这个函数的最小值？"
- **关键提示**：二次函数 $S_n = \frac{1}{2}n^2 - \frac{9}{2}n$，注意 $n$ 必须是正整数
- **期望学生回答**：对称轴 $n = 4.5$，所以 $n = 4$ 或 $5$ 时取最小值，$S_{\min} = -10$

**步骤5：验证结果**
- **引导语**："$S_5$ 已知为 $-10$，这与我们求出的最小值有什么关系？"
- **关键提示**：验证 $S_4$ 是否也等于 $-10$
- **期望学生回答**：确认最小值确实是 $-10$

## 五、关联拓展

### 核心方法总结

**等差数列最值问题的一般解法：**
1. **通项分析法**：若 $a_n \geq 0$ 且 $a_{n+1} \leq 0$，则 $S_n$ 最大；若 $a_n \leq 0$ 且 $a_{n+1} \geq 0$，则 $S_n$ 最小。
2. **函数法**：将 $S_n = \frac{d}{2}n^2 + (a_1 - \frac{d}{2})n$ 视为二次函数，通过配方求最值，注意 $n \in \mathbb{N}^*$。

**本题关键发现：**
- 当公差 $d &gt; 0$ 时，数列递增，$S_n$ 先减后增，存在最小值。
- 当 $S_n$ 关于 $n$ 的二次函数对称轴恰好为半整数（如 $4.5$）时，离对称轴最近的两个整数（4和5）同时取得最值。
