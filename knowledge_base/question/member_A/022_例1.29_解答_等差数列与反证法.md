# 【A_022_1】等差数列与反证法证明

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P22 例1.29

**题目：**

等差数列 $\{a_n\}$ 的前 $n$ 项和为 $S_n$，$a_1=1+\sqrt{2}$，$S_3=9+3\sqrt{2}$。

(I) 求数列 $\{a_n\}$ 的通项公式与前 $n$ 项和 $S_n$；

(II) 设 $b_n=\frac{S_n}{n}$（$n \in \mathbb{N}^*$），求证：$\{b_n\}$ 中任意不同的三项都不可能为等比数列。

---

## 二、答案与解析

### 答案

**(I)** $a_n=2n-1+\sqrt{2}$，$S_n=n(n+\sqrt{2})$；**(II)** 证明见解析

### 解析

**步骤1：求基本量**

设公差为 $d$。
$$S_3 = 3a_1 + \frac{3 \times 2}{2}d = 3a_1 + 3d = 9+3\sqrt{2}$$

即 $a_1+d = 3+\sqrt{2}$。

由 $a_1=1+\sqrt{2}$，得：
$$d = (3+\sqrt{2}) - (1+\sqrt{2}) = 2$$

**步骤2：求通项与和**

通项：
$$a_n = a_1 + (n-1)d = 1+\sqrt{2} + 2(n-1) = 2n-1+\sqrt{2}$$

前 $n$ 项和：
$$S_n = \frac{n(a_1+a_n)}{2} = \frac{n(1+\sqrt{2}+2n-1+\sqrt{2})}{2} = \frac{n(2n+2\sqrt{2})}{2} = n(n+\sqrt{2})$$

**步骤3：反证法证明（第II问）**

由(I)知：
$$b_n = \frac{S_n}{n} = n+\sqrt{2}$$

假设 $\{b_n\}$ 中存在三项 $b_k, b_m, b_p$（$k&lt;m&lt;p$，$k,m,p \in \mathbb{N}^*$）成等比数列，则：
$$b_m^2 = b_k \cdot b_p$$

即：
$$(m+\sqrt{2})^2 = (k+\sqrt{2})(p+\sqrt{2})$$

展开：
$$m^2+2m\sqrt{2}+2 = kp + (k+p)\sqrt{2} + 2$$

整理：
$$(m^2-kp) + (2m-k-p)\sqrt{2} = 0$$

**步骤4：利用无理数性质**

因为 $m^2-kp$ 和 $2m-k-p$ 都是整数，而 $\sqrt{2}$ 是无理数，要使上式成立，必须：
$$\begin{cases} m^2-kp = 0 \\ 2m-k-p = 0 \end{cases}$$

由第二式得 $k+p=2m$，代入第一式：
$$m^2 = kp = k(2m-k) = 2mk - k^2$$
$$(m-k)^2 = 0$$

得 $m=k$，与 $k&lt;m$ 矛盾。

故假设不成立，$\{b_n\}$ 中任意不同的三项都不可能为等比数列。

## 三、分步教学引导（用于AI Agent）

**步骤1：计算基本量**
- **引导语**："已知 $a_1$ 和 $S_3$，如何求公差 $d$？"
- **关键提示**：$S_3=3a_1+3d$
- **期望学生回答**：$d=2$

**步骤2：写出 $b_n$ 表达式**
- **引导语**："计算 $b_n=\frac{S_n}{n}$，观察其形式。"
- **关键提示**：$b_n=n+\sqrt{2}$
- **期望学生回答**：$n+\sqrt{2}$，一个整数加无理数

**步骤3：反证法假设**
- **引导语**："假设存在三项成等比，写出等比中项关系式。"
- **关键提示**：$b_m^2=b_k \cdot b_p$
- **期望学生回答**：$(m+\sqrt{2})^2=(k+\sqrt{2})(p+\sqrt{2})$

**步骤4：展开整理**
- **引导语**："展开后，将有理数部分和无理数部分分别整理。"
- **关键提示**：$(m^2-kp) + (2m-k-p)\sqrt{2} = 0$
- **期望学生回答**：有理数部分=0，无理数系数=0

**步骤5：导出矛盾**
- **引导语**："由 $2m=k+p$ 和 $m^2=kp$ 能推出什么？"
- **关键提示**：基本不等式或消元得 $(m-k)^2=0$
- **期望学生回答**：$m=k$，与假设 $k&lt;m$ 矛盾

## 五、关联拓展

### 相关方法总结

**证明"不可能"类问题的策略：**
1. 反证法：假设存在，推出矛盾
2. 极端原理：考察最大最小项
3. 整数性质：利用整数的离散性、奇偶性等

**无理数方程处理：**
若 $a+b\sqrt{2}=0$ 且 $a,b \in \mathbb{Q}$，则 $a=b=0$（$\sqrt{2}$ 是无理数）
