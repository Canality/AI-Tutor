# 【A_012_4】等比数列对数求和（2014广东理13）

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P12 变式1.23.2
&gt; 
&gt; **来源**：2014年广东理科第13题

**题目：**

若等比数列 $\{a_n\}$ 的各项均为正数，且 $a_{10}a_{11}+a_9a_{12}=2e^5$，则 $\ln a_1+\ln a_2+\cdots+\ln a_{20}=$ ______。

---

## 二、答案与解析

### 答案

**50**

### 解析

**步骤1：利用等比数列性质化简已知**

由 $a_{10}a_{11} = a_9a_{12}$（因为 $10+11=9+12=21$），所以：
$$a_{10}a_{11} + a_9a_{12} = 2a_{10}a_{11} = 2e^5$$

得 $a_{10}a_{11} = e^5$。

**步骤2：转化对数求和**

设 $S = \ln a_1 + \ln a_2 + \cdots + \ln a_{20} = \ln(a_1a_2\cdots a_{20})$

**步骤3：利用倒序相乘**

设 $P = a_1a_2\cdots a_{20}$，则：
$$P = a_1 \cdot a_2 \cdots a_{20}$$
$$P = a_{20} \cdot a_{19} \cdots a_1$$

两式相乘：
$$P^2 = (a_1a_{20})(a_2a_{19})\cdots(a_{20}a_1)$$

共20项，每对 $a_ia_{21-i} = a_{10}a_{11} = e^5$（因为 $i+(21-i)=21=10+11$）。

所以 $P^2 = (e^5)^{20} = e^{100}$，得 $P = e^{50}$（因为各项为正）。

**步骤4：计算结果**

$$S = \ln P = \ln e^{50} = 50$$

故填 **50**。

## 三、分步教学引导（用于AI Agent）

### 整体思路引导

**步骤1：分析已知条件**
- **引导语**："已知 $a_{10}a_{11}+a_9a_{12}=2e^5$，利用等比数列性质，$a_{10}a_{11}$ 与 $a_9a_{12}$ 有什么关系？"
- **关键提示**：$10+11=9+12$
- **期望学生回答**：$a_{10}a_{11} = a_9a_{12}$，所以 $2a_{10}a_{11}=2e^5$，即 $a_{10}a_{11}=e^5$

**步骤2：转化求和式**
- **引导语**："要求的式子可以转化为对数里面的什么？"
- **期望学生回答**：$\ln(a_1a_2\cdots a_{20})$

**步骤3：利用对称性**
- **引导语**："设 $P = a_1a_2\cdots a_{20}$，考虑 $P^2 = (a_1a_2\cdots a_{20})(a_{20}a_{19}\cdots a_1)$，每一对 $a_i \cdot a_{21-i}$ 等于多少？"
- **关键提示**：$i+(21-i)=21=10+11$
- **期望学生回答**：$a_ia_{21-i} = a_{10}a_{11} = e^5$

**步骤4：计算乘积**
- **引导语**："共有20对，所以 $P^2 = (e^5)^{20} = e^{100}$，那么 $P$ 是多少？"
- **期望学生回答**：$P = e^{50}$（取正）

**步骤5：得出结果**
- **引导语**："所以 $\ln P = ?$"
- **期望学生回答**：$50$

## 五、关联拓展

### 倒序相乘法

对于等比数列求积 $P = a_1a_2\cdots a_n$，常用技巧：
$$P^2 = (a_1a_n)(a_2a_{n-1})\cdots$$

若 $n$ 为偶数，共 $\frac{n}{2}$ 对；若 $n$ 为奇数，中间项为 $a_{(n+1)/2}$。
