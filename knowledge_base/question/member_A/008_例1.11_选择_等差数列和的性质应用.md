# 【A_008】等差数列和的性质应用

## 一、题目内容

&gt; **原题位置**：P8 例1.11

**题目：**

已知等差数列 $\{a_n\}$ 的前 $n$ 项和为 $S_n$，且满足 $\frac{S_{n+2}}{n+2} - \frac{S_n}{n} = 2$，则数列 $\{a_n\}$ 的公差是（ ）。

A. 1  
B. 2  
C. 3  
D. 4

---

## 二、答案与解析

### 答案

**B**

### 解析

**步骤1：利用 $\frac{S_n}{n}$ 的性质**

对于等差数列，$\frac{S_n}{n} = \frac{a_1 + a_n}{2}$。

因此：
$$\frac{S_{n+2}}{n+2} = \frac{a_1 + a_{n+2}}{2}$$

**步骤2：代入已知条件**

$$\frac{a_1 + a_{n+2}}{2} - \frac{a_1 + a_n}{2} = 2$$

$$\frac{a_{n+2} - a_n}{2} = 2$$

$$a_{n+2} - a_n = 4$$

**步骤3：求公差**

在等差数列中，$a_{n+2} - a_n = 2d$。

所以 $2d = 4$，解得 $d = 2$。

故选 **B**。

## 三、分步教学引导（用于AI Agent）

### 整体思路引导

**步骤1：识别关键性质**
- **引导语**："题目中出现了 $\frac{S_n}{n}$，回忆一下等差数列中 $\frac{S_n}{n}$ 等于什么？"
- **关键提示**：$\frac{S_n}{n} = \frac{a_1 + a_n}{2}$，即首项与第n项的等差中项
- **期望学生回答**：$\frac{S_n}{n} = \frac{a_1 + a_n}{2}$

**步骤2：代入化简**
- **引导语**："将 $\frac{S_{n+2}}{n+2}$ 和 $\frac{S_n}{n}$ 都用这个公式表示，然后相减。"
- **关键提示**：注意分子相减后 $a_1$ 消去
- **期望学生回答**：$\frac{a_{n+2} - a_n}{2} = 2$

**步骤3：利用公差定义**
- **引导语**："$a_{n+2} - a_n$ 与公差 $d$ 有什么关系？"
- **关键提示**：隔两项的差是 $2d$
- **期望学生回答**：$a_{n+2} - a_n = 2d = 4$，所以 $d = 2$

## 五、关联拓展

### 相关结论总结

**重要性质：**
数列 $\{a_n\}$ 是等差数列 $\Leftrightarrow$ $\frac{S_n}{n}$ 是等差数列 $\Leftrightarrow$ $S_n = An^2 + Bn$

且 $\{\frac{S_n}{n}\}$ 的公差为 $\frac{d}{2}$，其中 $d$ 是 $\{a_n\}$ 的公差。

**验证：**
$$\frac{S_n}{n} = na_1 + \frac{n(n-1)}{2}d \cdot \frac{1}{n} = a_1 + \frac{(n-1)d}{2} = \frac{d}{2}n + (a_1 - \frac{d}{2})$$

这是关于 $n$ 的一次函数，公差为 $\frac{d}{2}$。
