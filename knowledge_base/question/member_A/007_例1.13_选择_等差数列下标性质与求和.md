# 【A_007】等差数列下标性质与求和（2015兰州诊断考试）

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P7 例1.13
&gt; 
&gt; **来源**：2015年兰州诊断考试

**题目：**

在等差数列 $\{a_n\}$ 中，$3(a_3+a_5)+2(a_7+a_{10}+a_{13})=24$，则该数列前 $13$ 项的和是 ( )。

A. $13$

B. $26$

C. $52$

D. $156$

---

## 二、答案与解析

### 答案

**B**

### 解析

**步骤1：利用下标性质化简已知条件**

根据等差数列下标性质：若 $m+n=p+q$，则 $a_m+a_n=a_p+a_q$。

对于 $a_3+a_5$：
- $3+5=8=4+4$，所以 $a_3+a_5=2a_4$

对于 $a_7+a_{10}+a_{13}$：
- $7+13=20=10+10$，所以 $a_7+a_{13}=2a_{10}$
- 因此 $a_7+a_{10}+a_{13}=2a_{10}+a_{10}=3a_{10}$

**步骤2：建立关于 $a_4$ 和 $a_{10}$ 的方程**

将上述结果代入已知条件：
$$3 \times 2a_4 + 2 \times 3a_{10} = 24$$
$$6a_4 + 6a_{10} = 24$$
$$a_4 + a_{10} = 4$$

**步骤3：转化为首末项关系**

再次利用下标性质：
- $4+10=14=1+13$，所以 $a_4+a_{10}=a_1+a_{13}=4$

**步骤4：计算前13项和**

$$S_{13} = \frac{13(a_1+a_{13})}{2} = \frac{13 \times 4}{2} = 26$$

故选 **B**。

## 三、分步教学引导（用于AI Agent）

### 整体思路引导

**步骤1：观察下标特征**
- **引导语**："观察等式中的各项下标：3和5，7、10和13。根据等差数列下标性质，它们分别可以转化成哪个中间项？"
- **关键提示**：$3+5=8=4+4$，$7+13=20=10+10$
- **期望学生回答**：$a_3+a_5=2a_4$，$a_7+a_{13}=2a_{10}$

**步骤2：化简方程**
- **引导语**："将转化后的式子代入原方程，能得到什么？"
- **关键提示**：$6a_4+6a_{10}=24$，即 $a_4+a_{10}=4$
- **期望学生回答**：$a_4+a_{10}=4$

**步骤3：联系首末项**
- **引导语**："要求前13项和 $S_{13}$，需要知道 $a_1+a_{13}$。$a_4+a_{10}$ 与 $a_1+a_{13}$ 有什么关系？"
- **关键提示**：$4+10=14=1+13$，下标和相等则对应项和相等
- **期望学生回答**：$a_1+a_{13}=a_4+a_{10}=4$

**步骤4：计算结果**
- **引导语**："利用 $S_{13}=\frac{13(a_1+a_{13})}{2}$ 计算最终结果。"
- **关键提示**：代入 $a_1+a_{13}=4$
- **期望学生回答**：$S_{13}=26$

## 五、关联拓展

### 核心结论总结

**等差数列下标性质的应用技巧：**
当条件中出现多个项的和时，优先考虑利用 $a_m+a_n=a_p+a_q$（其中 $m+n=p+q$）进行合并，目标是凑出与问题相关的首末项或中间项。

**本题模式：**
- 条件：多个不同下标的项 → 利用性质合并为 $a_4+a_{10}$
- 目标：求 $S_{13}$ → 需要 $a_1+a_{13}$ → 利用 $4+10=1+13$ 建立联系
