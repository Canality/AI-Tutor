# 【A_013】等比数列通项与求和（2018新课标III）

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P13 例1.24
&gt; 
&gt; **来源**：2018年新课标III卷文理第17题

**题目：**

等比数列 $\{a_n\}$ 中，$a_1=1$，$a_5=4a_3$。

(I) 求 $\{a_n\}$ 的通项公式；

(II) 记 $S_n$ 为 $\{a_n\}$ 的前 $n$ 项和，若 $S_m=63$，求 $m$。

---

## 二、答案与解析

### 答案

(I) $a_n = (-2)^{n-1}$ 或 $a_n = 2^{n-1}$；

(II) $m=6$。

### 解析

**(I) 求通项公式**

设公比为 $q$。

由 $a_5 = 4a_3$，得 $a_1q^4 = 4a_1q^2$。

因为 $a_1 = 1 \neq 0$，所以 $q^4 = 4q^2$，即 $q^2(q^2-4) = 0$。

解得 $q = 0$（舍去，等比数列公比不为0），$q = 2$ 或 $q = -2$。

当 $q = 2$ 时，$a_n = 2^{n-1}$；

当 $q = -2$ 时，$a_n = (-2)^{n-1}$。

**(II) 求 $m$**

**情况1**：$a_n = (-2)^{n-1}$

$$S_n = \frac{1-(-2)^n}{1-(-2)} = \frac{1-(-2)^n}{3}$$

令 $S_m = 63$，得 $\frac{1-(-2)^m}{3} = 63$，即 $1-(-2)^m = 189$，$(-2)^m = -188$。

无整数解（左边为偶数，右边为偶数，但 $(-2)^m$ 的绝对值为2的幂次，188不是2的幂次），舍去。

**情况2**：$a_n = 2^{n-1}$

$$S_n = \frac{1-2^n}{1-2} = 2^n - 1$$

令 $S_m = 63$，得 $2^m - 1 = 63$，即 $2^m = 64 = 2^6$，所以 $m = 6$。

综上，$m = 6$。

## 三、分步教学引导（用于AI Agent）

### 整体思路引导

**步骤1：建立方程**
- **引导语**："由 $a_5=4a_3$ 和 $a_1=1$，写出关于 $q$ 的方程。"
- **关键提示**：$a_5=q^4$，$a_3=q^2$
- **期望学生回答**：$q^4=4q^2$

**步骤2：求解并取舍**
- **引导语**："解得 $q=0, 2, -2$，哪个要舍去？为什么？"
- **关键提示**：等比数列定义
- **期望学生回答**：$q=0$ 舍去，因为等比数列公比不能为0

**步骤3：写出两种通项**
- **引导语**："分别写出 $q=2$ 和 $q=-2$ 时的通项公式。"
- **期望学生回答**：$a_n=2^{n-1}$ 或 $a_n=(-2)^{n-1}$

**步骤4：分类求 $m$（第一种）**
- **引导语**："对于 $a_n=(-2)^{n-1}$，$S_m=63$ 是否有解？"
- **关键提示**：$S_m = \frac{1-(-2)^m}{3} = 63$ 得 $(-2)^m = -188$
- **期望学生回答**：无解，因为188不是2的整数幂

**步骤5：分类求 $m$（第二种）**
- **引导语**："对于 $a_n=2^{n-1}$，解方程 $2^m-1=63$。"
- **期望学生回答**：$2^m=64$，$m=6$

## 五、关联拓展

### 等比数列求和注意点

1. 必须讨论 $q=1$ 和 $q \neq 1$ 两种情况（本题 $q=1$ 不满足 $a_5=4a_3$）
2. 当 $q&lt;0$ 时，$S_n$ 公式中 $(-2)^n$ 的符号随 $n$ 奇偶变化
3. 验证解的合理性（如 $(-2)^m = -188$ 无整数解）
