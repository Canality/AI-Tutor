# 【A_011_2】等差数列最值与项计算

## 一、题目内容

&gt; **原题位置**：P12 变式1.16.1
&gt; 
&gt; **来源**：2019年北京卷理科第10题

**题目：**

设等差数列 $\{a_n\}$ 的前 $n$ 项和为 $S_n$，若 $a_2 = -3$，$S_5 = -10$，则 $a_5 =$ ______，$S_n$ 的最小值为 ______。

---

## 二、答案与解析

### 答案

$a_5 = 0$；$S_n$ 的最小值为 $-10$

### 解析

**步骤1：建立方程组求基本量**

设首项为 $a_1$，公差为 $d$。

由 $a_2 = -3$ 得：
$$a_1 + d = -3 \quad \text{①}$$

由 $S_5 = -10$ 得：
$$5a_1 + \frac{5 \times 4}{2}d = -10$$
$$5a_1 + 10d = -10$$
$$a_1 + 2d = -2 \quad \text{②}$$

**步骤2：求解基本量**

② - ① 得：
$$(a_1 + 2d) - (a_1 + d) = -2 - (-3)$$
$$d = 1$$

代入①得：
$$a_1 = -3 - 1 = -4$$

**步骤3：求 $a_5$**

$$a_5 = a_1 + 4d = -4 + 4 \times 1 = 0$$

**步骤4：求 $S_n$ 的最小值**

$$S_n = na_1 + \frac{n(n-1)}{2}d = -4n + \frac{n(n-1)}{2} = \frac{n^2 - 9n}{2}$$

配方：
$$S_n = \frac{1}{2}\left(n - \frac{9}{2}\right)^2 - \frac{81}{8}$$

由于 $n \in \mathbb{N}^*$，且对称轴 $n = 4.5$，故 $n=4$ 或 $n=5$ 时取得最小值。

计算：
$$S_4 = \frac{16 - 36}{2} = -10$$
$$S_5 = -10$$

故最小值为 $-10$。

---

## 三、分步教学引导

**步骤1：列方程组**
- **引导语**："根据 $a_2 = -3$ 和 $S_5 = -10$，写出关于 $a_1$ 和 $d$ 的两个方程。"
- **关键提示**：$a_2 = a_1 + d$，$S_5 = 5a_1 + 10d$
- **期望学生回答**：$\begin{cases} a_1 + d = -3 \\ a_1 + 2d = -2 \end{cases}$

**步骤2：解方程组**
- **引导语**："用消元法解这个方程组。"
- **关键提示**：②式减①式
- **期望学生回答**：$d = 1$，$a_1 = -4$

**步骤3：计算 $a_5$**
- **引导语**："利用通项公式计算第5项。"
- **关键提示**：$a_5 = a_1 + 4d$
- **期望学生回答**：$a_5 = -4 + 4 = 0$

**步骤4：配方求最值**
- **引导语**："写出 $S_n$ 的表达式并配方，注意对称轴位置。"
- **关键提示**：$S_n = \frac{1}{2}n^2 - \frac{9}{2}n$，对称轴 $n = 4.5$
- **期望学生回答**：由于 $n$ 必须为整数，比较 $n=4$ 和 $n=5$，最小值为 $-10$
