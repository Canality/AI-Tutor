# 【A_003】等差数列与裂项相消求和（2017新课标II）

## 一、题目内容

&gt; **原题位置**：P3 变式1.2.2
&gt; 
&gt; **来源**：2017年新课标II卷理科第15题

**题目：**

等差数列 $\{a_n\}$ 的前 $n$ 项和为 $S_n$，$a_3 = 3$，$S_4 = 10$，则 $\sum\limits_{k=1}^{n}\frac{1}{S_k} = $ ______。

---

## 二、答案与解析

### 答案

$$\frac{2n}{n+1}$$

### 解析

**步骤1：求首项和公差**

设等差数列 $\{a_n\}$ 的首项为 $a_1$，公差为 $d$。

由已知条件：
$$\begin{cases} a_3 = a_1 + 2d = 3 \\ S_4 = 4a_1 + \frac{4 \times 3}{2}d = 10 \end{cases}$$

即：
$$\begin{cases} a_1 + 2d = 3 \\ 4a_1 + 6d = 10 \end{cases}$$

由第一个方程得 $a_1 = 3 - 2d$，代入第二个方程：
$$4(3-2d) + 6d = 10$$
$$12 - 8d + 6d = 10$$
$$-2d = -2 \Rightarrow d = 1$$

回代得 $a_1 = 3 - 2 = 1$。

所以 $\begin{cases} a_1 = 1 \\ d = 1 \end{cases}$。

**步骤2：求$S_n$的表达式**

$$S_n = na_1 + \frac{n(n-1)}{2}d = n + \frac{n(n-1)}{2} = \frac{2n + n^2 - n}{2} = \frac{n(n+1)}{2}$$

**步骤3：裂项处理**

$$\frac{1}{S_k} = \frac{2}{k(k+1)} = 2\left(\frac{1}{k} - \frac{1}{k+1}\right)$$

**步骤4：求和（裂项相消）**

$$\sum_{k=1}^{n}\frac{1}{S_k} = \sum_{k=1}^{n}2\left(\frac{1}{k} - \frac{1}{k+1}\right)$$

$$= 2\left[\left(1-\frac{1}{2}\right) + \left(\frac{1}{2}-\frac{1}{3}\right) + \left(\frac{1}{3}-\frac{1}{4}\right) + \cdots + \left(\frac{1}{n}-\frac{1}{n+1}\right)\right]$$

中间项全部抵消，得：
$$= 2\left(1 - \frac{1}{n+1}\right) = 2 \cdot \frac{n}{n+1} = \frac{2n}{n+1}$$

故填 $\frac{2n}{n+1}$。

## 三、分步教学引导（用于AI Agent）

### 整体思路引导

**步骤1：识别题型**
- **引导语**："这道题给出了等差数列的两个条件，但最后要求的是倒数和，我们应该先做什么？"
- **关键提示**：先求出 $S_n$ 的具体表达式
- **期望学生回答**：先通过 $a_3=3$ 和 $S_4=10$ 求出 $a_1$ 和 $d$

**步骤2：列方程求解基本量**
- **引导语**："写出关于 $a_1$ 和 $d$ 的方程组并求解。"
- **关键提示**：$a_1+2d=3$，$4a_1+6d=10$
- **期望学生回答**：$a_1=1$，$d=1$

**步骤3：求前n项和公式**
- **引导语**："现在知道了 $a_1=1$，$d=1$，$S_n$ 等于什么？"
- **关键提示**：$S_n = \frac{n(n+1)}{2}$
- **期望学生回答**：$S_n = \frac{n(n+1)}{2}$

**步骤4：裂项相消**
- **引导语**："看到 $\frac{1}{S_k} = \frac{2}{k(k+1)}$，这种形式怎么处理？"
- **关键提示**：分式裂项 $\frac{1}{k(k+1)} = \frac{1}{k} - \frac{1}{k+1}$
- **期望学生回答**：$\frac{2}{k(k+1)} = 2(\frac{1}{k} - \frac{1}{k+1})$

**步骤5：求和并化简**
- **引导语**："写出前几项，看看哪些项会消掉？"
- **关键提示**：相邻项正负抵消
- **期望学生回答**：原式 $= 2(1 - \frac{1}{n+1}) = \frac{2n}{n+1}$

## 四、标签信息（用于分类和检索）

### 解题方法标签
- 基本量法（求 $a_1, d$）
- 裂项相消法（$\frac{1}{k(k+1)} = \frac{1}{k} - \frac{1}{k+1}$）
- 累加求和

### 易错点标签
- $S_n$ 公式记错（忘记除以2）
- 裂项系数错误（忘记前面的系数2）
- 裂项符号错误（$\frac{1}{k} - \frac{1}{k+1}$ 写成 $\frac{1}{k+1} - \frac{1}{k}$）
- 求和时漏乘系数2
- 最后化简错误（$1-\frac{1}{n+1}$ 算错）

### 题目特征标签
- 高考真题（2017新课标II卷）
- 中高档题（裂项相消综合）
- 等差数列与数列求和结合
- 经典裂项模型

## 五、关联拓展

### 核心公式总结

**裂项相消常见模型：**
1. $\frac{1}{n(n+1)} = \frac{1}{n} - \frac{1}{n+1}$
2. $\frac{1}{n(n+2)} = \frac{1}{2}(\frac{1}{n} - \frac{1}{n+2})$
3. $\frac{1}{\sqrt{n}+\sqrt{n+1}} = \sqrt{n+1} - \sqrt{n}$

**等差数列求和公式变形：**
当 $a_1=d$ 时，$S_n = \frac{n(n+1)}{2}d$（本题 $d=1$ 的特殊情况）
