# 【A_012_3】等比数列性质综合（2018广州模考）

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P12 变式1.23.1
&gt; 
&gt; **来源**：2018年广州模考

**题目：**

已知数列 $\{a_n\}$ 为等比数列，若 $a_4+a_6=10$，则 $a_7(a_1+2a_3)+a_3a_9=$ ______。

---

## 二、答案与解析

### 答案

**100**

### 解析

**步骤1：展开表达式**

$$a_7(a_1+2a_3)+a_3a_9 = a_1a_7 + 2a_3a_7 + a_3a_9$$

**步骤2：利用等比数列性质**

在等比数列中：
- $a_1a_7 = a_4^2$（因为 $1+7=4+4$）
- $a_3a_7 = a_4a_6$（因为 $3+7=4+6$）
- $a_3a_9 = a_6^2$（因为 $3+9=6+6$）

**步骤3：代换化简**

原式 $= a_4^2 + 2a_4a_6 + a_6^2 = (a_4+a_6)^2$

**步骤4：代入已知**

因为 $a_4+a_6 = 10$，所以原式 $= 10^2 = 100$。

故填 **100**。

## 三、分步教学引导（用于AI Agent）

### 整体思路引导

**步骤1：展开括号**
- **引导语**："先将表达式 $a_7(a_1+2a_3)+a_3a_9$ 展开，看看有哪些项。"
- **期望学生回答**：$a_1a_7 + 2a_3a_7 + a_3a_9$

**步骤2：寻找等比关系**
- **引导语**："观察这三项中的下标，利用等比数列性质 $a_ma_n=a_pa_q$（当 $m+n=p+q$），分别转化为与 $a_4$ 和 $a_6$ 相关的形式。"
- **关键提示**：$1+7=8=4+4$，$3+7=10=4+6$，$3+9=12=6+6$
- **期望学生回答**：$a_1a_7=a_4^2$，$a_3a_7=a_4a_6$，$a_3a_9=a_6^2$

**步骤3：识别完全平方**
- **引导语**："现在表达式变成了 $a_4^2 + 2a_4a_6 + a_6^2$，这是什么公式？"
- **期望学生回答**：$(a_4+a_6)^2$

**步骤4：代入求值**
- **引导语**："已知 $a_4+a_6=10$，所以结果是多少？"
- **期望学生回答**：$100$

## 五、关联拓展

### 等比数列积的性质推广

对于等比数列，若 $m+n=p+q=r+s$，则 $a_ma_n = a_pa_q = a_ra_s$。

特别适用于将分散的下标集中，便于利用已知条件。
