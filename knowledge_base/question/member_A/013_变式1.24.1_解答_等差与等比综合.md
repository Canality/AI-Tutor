# 【A_013_2】等差与等比综合（2018北京文15）

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P13 变式1.24.1
&gt; 
&gt; **来源**：2018年北京文科第15题

**题目：**

设 $\{a_n\}$ 是等差数列，且 $a_1=\ln 2$，$a_2+a_3=5\ln 2$。

(I) 求 $\{a_n\}$ 的通项公式；

(II) 求 $e^{a_1}+e^{a_2}+\cdots+e^{a_n}$。

---

## 二、答案与解析

### 答案

(I) $a_n = n\ln 2$；

(II) $2^{n+1}-2$。

### 解析

**(I) 求等差数列通项**

设公差为 $d$。

由 $a_2+a_3 = (a_1+d)+(a_1+2d) = 2a_1+3d = 5\ln 2$。

代入 $a_1=\ln 2$：
$$2\ln 2 + 3d = 5\ln 2$$
$$3d = 3\ln 2$$
$$d = \ln 2$$

所以 $a_n = a_1+(n-1)d = \ln 2 + (n-1)\ln 2 = n\ln 2$。

**(II) 求指数和**

由 (I) 知 $a_n = n\ln 2 = \ln 2^n$。

所以 $e^{a_n} = e^{\ln 2^n} = 2^n$。

因此 $e^{a_1}+e^{a_2}+\cdots+e^{a_n} = 2^1+2^2+\cdots+2^n = \frac{2(1-2^n)}{1-2} = 2(2^n-1) = 2^{n+1}-2$。

---

## 三、分步教学引导（用于AI Agent）

### 整体思路引导

**步骤1：求公差**
- **引导语**："利用 $a_2+a_3=5\ln 2$ 和 $a_1=\ln 2$，建立关于 $d$ 的方程。"
- **关键提示**：$a_2=a_1+d$，$a_3=a_1+2d$
- **期望学生回答**：$2\ln 2 + 3d = 5\ln 2$，解得 $d=\ln 2$

**步骤2：写出通项**
- **引导语**："有了 $a_1$ 和 $d$，写出 $a_n$ 并化简。"
- **期望学生回答**：$a_n = \ln 2 + (n-1)\ln 2 = n\ln 2$

**步骤3：化简 $e^{a_n}$**
- **引导语**："根据对数性质，$e^{n\ln 2}$ 等于什么？"
- **关键提示**：$e^{\ln x} = x$，$n\ln 2 = \ln 2^n$
- **期望学生回答**：$e^{a_n} = 2^n$

**步骤4：识别数列类型并求和**
- **引导语**："所以要求的和是什么数列的前 $n$ 项和？"
- **期望学生回答**：首项为2，公比为2的等比数列，$S_n = 2^{n+1}-2$

## 五、关联拓展

### 等差与等比转换

若 $\{a_n\}$ 是等差数列，则 $\{e^{a_n}\}$ 是等比数列，公比为 $e^d$。

证明：$\frac{e^{a_{n+1}}}{e^{a_n}} = e^{a_{n+1}-a_n} = e^d$（常数）。

本题中 $d=\ln 2$，所以公比 $q = e^{\ln 2} = 2$。
