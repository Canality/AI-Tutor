# 【A_033】递推数列求项（成都一诊）

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P33 变式2.3.1
&gt; 
&gt; **来源**：2019年成都一诊理15

**题目：**

记 $S_n$ 为数列 $\{a_n\}$ 的前 $n$ 项和，且 $a_1 = 4$，$a_{n+1} = S_n$，则 $a_5 = $ ______。

---

## 二、答案与解析

### 答案

**32**

### 解析

**方法一：逐项计算**

由$a_{n+1} = S_n$可得：
- $a_2 = S_1 = a_1 = 4$
- $a_3 = S_2 = a_1 + a_2 = 4 + 4 = 8$
- $a_4 = S_3 = a_1 + a_2 + a_3 = 4 + 4 + 8 = 16$
- $a_5 = S_4 = a_1 + a_2 + a_3 + a_4 = 4 + 4 + 8 + 16 = 32$

**方法二：构造递推关系**

由$a_{n+1} = S_n$（$n \geq 1$），得当$n \geq 2$时，$a_n = S_{n-1}$。

两式相减：
$$a_{n+1} - a_n = S_n - S_{n-1} = a_n$$

即：
$$a_{n+1} = 2a_n \quad (n \geq 2)$$

这说明从第2项起，$\{a_n\}$是等比数列，公比为$2$。

已知$a_2 = S_1 = 4$，所以：
- $a_3 = 2a_2 = 8$
- $a_4 = 2a_3 = 16$
- $a_5 = 2a_4 = 32$

故填$32$。

## 三、分步教学引导（用于AI Agent）

**步骤1：理解递推关系**
- **引导语**："$a_{n+1} = S_n$表示什么含义？$a_2$等于多少？"
- **关键提示**：$a_2 = S_1 = a_1$
- **期望学生回答**：$a_2 = 4$

**步骤2：计算后续项（方法1）**
- **引导语**："接着计算$a_3$，注意$a_3 = S_2 = a_1 + a_2$。"
- **关键提示**：$S_2$是前两项和
- **期望学生回答**：$a_3 = 4 + 4 = 8$

**步骤3：发现规律（方法2）**
- **引导语**："观察$a_{n+1} = S_n$和$a_n = S_{n-1}$（$n \geq 2$），两式相减得到什么？"
- **关键提示**：$S_n - S_{n-1} = a_n$
- **期望学生回答**：$a_{n+1} - a_n = a_n$，即$a_{n+1} = 2a_n$

**步骤4：确定数列性质**
- **引导语**："从第几项开始，这是什么数列？首项和公比是多少？"
- **关键提示**：$n \geq 2$时，$a_{n+1} = 2a_n$，即$a_3 = 2a_2$，$a_4 = 2a_3$...
- **期望学生回答**：从第2项起是等比数列，首项$a_2 = 4$，公比$q = 2$

**步骤5：计算$a_5$**
- **引导语**："利用等比数列性质或逐项计算求$a_5$。"
- **关键提示**：$a_5 = a_2 \cdot q^3 = 4 \times 8 = 32$或逐项累加
- **期望学生回答**：$32$

## 四、标签信息（用于分类和检索）

### 解题方法标签
- 递推法（逐项计算）
- 构造等比数列（发现$a_{n+1} = 2a_n$）
- $a_n$与$S_n$关系

### 易错点标签
- 错误认为从第1项起就是等比数列（实际上$a_2 = 4$，$a_1 = 4$，但$a_2/a_1 = 1 \neq 2$）
- 计算$S_n$时漏项（如$S_4$漏掉某一项）
- 混淆$a_{n+1} = S_n$中的下标（误认为$a_n = S_{n+1}$）

### 题目特征标签
- 诊断题（成都一诊）
- 递推数列
- 分段等比（从第2项起）
- 基础计算题
