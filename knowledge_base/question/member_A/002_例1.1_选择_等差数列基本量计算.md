# 【A_002】等差数列基本量计算（2019全国卷）

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P2 例1.1
&gt; 
&gt; **来源**：2019年全国I卷理科第9题

**题目：**

记 $S_n$ 为等差数列 $\{a_n\}$ 的前 $n$ 项和。已知 $S_4 = 0$，$a_5 = 5$，则（ ）。

A. $a_n = 2n - 5$

B. $a_n = 3n - 10$

C. $S_n = 2n^2 - 8n$

D. $S_n = \frac{1}{2}n^2 - 2n$

---

## 二、答案与解析

### 答案

**A**

### 解析

**步骤1：设定基本量**

设等差数列 $\{a_n\}$ 的首项为 $a_1$，公差为 $d$。

根据等差数列通项公式和前 $n$ 项和公式：
$$a_n = a_1 + (n-1)d$$
$$S_n = na_1 + \frac{n(n-1)}{2}d$$

**步骤2：建立方程组**

由已知条件 $S_4 = 0$ 和 $a_5 = 5$，代入公式得：

$$
\begin{cases}
a_1 + 4d = 5 & \text{(由 } a_5 = 5\text{)} \\
4a_1 + 6d = 0 & \text{(由 } S_4 = 0\text{，即 } 4a_1 + \frac{4 \times 3}{2}d = 0\text{)}
\end{cases}
$$

**步骤3：解方程组**

由第二个方程：$4a_1 + 6d = 0$，化简得 $2a_1 + 3d = 0$，即 $a_1 = -\frac{3}{2}d$。

代入第一个方程：
$$-\frac{3}{2}d + 4d = 5$$
$$\frac{5}{2}d = 5$$
$$d = 2$$

回代得：$a_1 = -\frac{3}{2} \times 2 = -3$。

所以 $\begin{cases} a_1 = -3 \\ d = 2 \end{cases}$。

**步骤4：求通项公式与前n项和**

通项公式：
$$a_n = a_1 + (n-1)d = -3 + (n-1) \times 2 = 2n - 5$$

前 $n$ 项和：
$$S_n = na_1 + \frac{n(n-1)}{2}d = -3n + \frac{n(n-1)}{2} \times 2 = -3n + n(n-1) = n^2 - 4n$$

**步骤5：验证选项**

- A. $a_n = 2n - 5$ ✓
- B. $a_n = 3n - 10$ ✗
- C. $S_n = 2n^2 - 8n$ ✗（应为 $n^2 - 4n$）
- D. $S_n = \frac{1}{2}n^2 - 2n$ ✗

故选 **A**。

## 三、分步教学引导（用于AI Agent）

### 整体思路引导

**步骤1：识别已知条件**
- **引导语**："题目给了我们哪两个关于等差数列的条件？请用数学符号表示出来。"
- **关键提示**：一个是第5项的值，一个是前4项的和
- **期望学生回答**：$a_5 = 5$ 和 $S_4 = 0$

**步骤2：设定基本量**
- **引导语**："等差数列由哪两个基本量确定？我们通常用什么字母表示？"
- **关键提示**：首项和公差
- **期望学生回答**：首项 $a_1$ 和公差 $d$

**步骤3：列方程**
- **引导语**："请写出 $a_5$ 关于 $a_1$ 和 $d$ 的表达式，以及 $S_4$ 关于 $a_1$ 和 $d$ 的表达式。"
- **关键提示**：$a_5 = a_1 + 4d$，$S_4 = 4a_1 + 6d$
- **期望学生回答**：得到方程组 $\begin{cases} a_1 + 4d = 5 \\ 4a_1 + 6d = 0 \end{cases}$

**步骤4：解方程组**
- **引导语**："解这个二元一次方程组，求出 $a_1$ 和 $d$ 的值。"
- **关键提示**：可以用代入法或消元法，比如从第二个方程解出 $a_1 = -\frac{3}{2}d$
- **期望学生回答**：$a_1 = -3$，$d = 2$

**步骤5：验证选项**
- **引导语**："现在我们知道了 $a_1 = -3$ 和 $d = 2$，请计算通项公式 $a_n$ 和前 $n$ 项和 $S_n$，看看对应哪个选项。"
- **关键提示**：$a_n = -3 + 2(n-1) = 2n - 5$
- **期望学生回答**：选 A

## 五、关联拓展

### 相关公式总结

**等差数列基本公式：**
1. 通项公式：$a_n = a_1 + (n-1)d = a_m + (n-m)d$
2. 前 $n$ 项和：$S_n = \frac{n(a_1 + a_n)}{2} = na_1 + \frac{n(n-1)}{2}d$
3. 下标性质：若 $m + n = p + q$，则 $a_m + a_n = a_p + a_q$
