# 【A_013_3】等比数列基本量计算

## 一、题目内容

**题目：**

已知等比数列 $\{a_n\}$ 中各项均为正数，$S_n$ 是其前 $n$ 项和，且满足 $2S_3 = 8a_1 + 3a_2$，$a_4 = 16$，则 $S_4 =$ ______。

## 二、答案与解析

### 答案

$30$

### 解析

**步骤1：建立方程**

设首项为 $a_1$，公比为 $q$（$q &gt; 0$）。

$$S_3 = a_1 + a_2 + a_3 = a_1(1 + q + q^2)$$

由 $2S_3 = 8a_1 + 3a_2$：
$$2a_1(1 + q + q^2) = 8a_1 + 3a_1q$$

除以 $a_1$（$a_1 &gt; 0$）：
$$2(1 + q + q^2) = 8 + 3q$$
$$2 + 2q + 2q^2 = 8 + 3q$$
$$2q^2 - q - 6 = 0$$

**步骤2：求解公比**

$$(2q + 3)(q - 2) = 0$$

解得 $q = 2$ 或 $q = -\frac{3}{2}$（舍去，因为各项为正）。

**步骤3：求首项**

由 $a_4 = a_1 q^3 = 16$：
$$a_1 \cdot 8 = 16$$
$$a_1 = 2$$

**步骤4：计算 $S_4$**

$$S_4 = \frac{a_1(1 - q^4)}{1 - q} = \frac{2(1 - 16)}{1 - 2} = \frac{2(-15)}{-1} = 30$$

## 三、分步教学引导

**步骤1：表示 $S_3$**
- **引导语**："用 $a_1$ 和 $q$ 表示 $S_3$ 和 $a_2$。"
- **期望学生回答**：$S_3 = a_1(1+q+q^2)$，$a_2 = a_1q$

**步骤2：建立方程**
- **引导语**："代入条件，化简得到关于 $q$ 的二次方程。"
- **期望学生回答**：$2q^2 - q - 6 = 0$

**步骤3：求解并验证**
- **引导语**："解方程并舍去不符合条件的根。"
- **期望学生回答**：$q = 2$

**步骤4：计算结果**
- **引导语**："求 $a_1$ 并计算 $S_4$。"
- **期望学生回答**：$a_1 = 2$，$S_4 = 30$
