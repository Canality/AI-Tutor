# 【A_014】等比数列基本量计算与分类讨论

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P14 例1.24
&gt; 
&gt; **来源**：2018年新课标Ⅲ卷文理第17题

**题目：**

已知等比数列 $\{a_n\}$ 中，$a_1 = 1$，$a_5 = 4a_3$。

(I) 求 $\{a_n\}$ 的通项公式；

(II) 记 $S_n$ 为 $\{a_n\}$ 的前 $n$ 项和，若 $S_m = 63$，求 $m$。

---

## 二、答案与解析

### 答案

(I) $a_n = 2^{n-1}$ 或 $a_n = (-2)^{n-1}$；

(II) $m = 6$。

### 解析

**步骤1：分析第(I)问，建立方程求公比**

设等比数列 $\{a_n\}$ 的公比为 $q$。

根据等比数列通项公式 $a_n = a_1 q^{n-1}$，由 $a_1 = 1$ 得：
- $a_3 = q^2$
- $a_5 = q^4$

由条件 $a_5 = 4a_3$，代入得：
$$q^4 = 4q^2$$

整理得：
$$q^4 - 4q^2 = 0 \Rightarrow q^2(q^2 - 4) = 0$$

解得 $q^2 = 0$（舍去，等比数列公比不能为0）或 $q^2 = 4$。

所以 $q = 2$ 或 $q = -2$。

**步骤2：写出通项公式**

- 当 $q = 2$ 时，$a_n = 1 \cdot 2^{n-1} = 2^{n-1}$；
- 当 $q = -2$ 时，$a_n = 1 \cdot (-2)^{n-1} = (-2)^{n-1}$。

**步骤3：分析第(II)问，分类讨论求 $m$**

由第(I)问知公比 $q$ 有两种情况，需分别讨论：

**情况1：当 $q = 2$ 时**

前 $n$ 项和公式：
$$S_n = \frac{a_1(1-q^n)}{1-q} = \frac{1-2^n}{1-2} = 2^n - 1$$

由 $S_m = 63$ 得：
$$2^m - 1 = 63 \Rightarrow 2^m = 64 \Rightarrow m = 6$$

**情况2：当 $q = -2$ 时**

前 $n$ 项和公式：
$$S_n = \frac{1-(-2)^n}{1-(-2)} = \frac{1-(-2)^n}{3}$$

由 $S_m = 63$ 得：
$$\frac{1-(-2)^m}{3} = 63 \Rightarrow 1-(-2)^m = 189 \Rightarrow (-2)^m = -188$$

由于 $(-2)^m$ 当 $m$ 为正整数时，若 $m$ 为偶数则值为正，若 $m$ 为奇数则值为负，但 $|-188| = 188$ 不是2的整数次幂，故此方程无正整数解。

**步骤4：综合结论**

综上所述，$m = 6$。

## 三、分步教学引导（用于AI Agent）

### 第(I)问引导

**步骤1：识别已知条件**
- **引导语**："题目告诉我们这是一个什么数列？已知哪些具体条件？"
- **关键提示**：等比数列，$a_1=1$，以及 $a_5$ 与 $a_3$ 的关系
- **期望学生回答**：等比数列，首项为1，$a_5 = 4a_3$

**步骤2：设基本量列方程**
- **引导语**："等比数列由首项和公比确定，设公比为 $q$，请用 $q$ 表示 $a_3$ 和 $a_5$。"
- **关键提示**：$a_n = a_1 q^{n-1} = q^{n-1}$
- **期望学生回答**：$a_3 = q^2$，$a_5 = q^4$

**步骤3：求解公比**
- **引导语**："将表达式代入 $a_5 = 4a_3$，得到什么方程？解出 $q$ 的所有可能值。"
- **关键提示**：$q^4 = 4q^2$，注意 $q \neq 0$
- **期望学生回答**：$q^2 = 4$，所以 $q = 2$ 或 $q = -2$

**步骤4：写出通项**
- **引导语**："根据 $q$ 的两个取值，分别写出通项公式。"
- **关键提示**：分情况写出 $a_n = 2^{n-1}$ 和 $a_n = (-2)^{n-1}$

### 第(II)问引导

**步骤5：选择和公式**
- **引导语**："要求 $S_m = 63$，我们需要用到等比数列的哪个公式？"
- **关键提示**：$S_n = \frac{a_1(1-q^n)}{1-q}$（$q \neq 1$）
- **期望学生回答**：前 $n$ 项和公式

**步骤6：分类计算**
- **引导语**："因为第(I)问有两个可能的 $q$ 值，我们需要分别验证。先算 $q=2$ 的情况。"
- **关键提示**：$S_m = 2^m - 1 = 63$
- **期望学生回答**：$2^m = 64$，$m=6$

**步骤7：验证另一种情况**
- **引导语**："现在验证 $q=-2$ 的情况，$S_m = 63$ 是否成立？"
- **关键提示**：$\frac{1-(-2)^m}{3} = 63 \Rightarrow (-2)^m = -188$
- **期望学生回答**：无正整数解，舍去

**步骤8：得出最终结论**
- **引导语**："综合两种情况，$m$ 的值是多少？"
- **期望学生回答**：$m = 6$

### 解题方法标签

- 未验证 $q=-2$ 时 $m$ 是否存在（直接认为无解而忽略检验）

## 五、关联拓展

### 相关公式总结

**等比数列核心公式：**
1. 通项公式：$a_n = a_1 q^{n-1} = a_m q^{n-m}$
2. 前 $n$ 项和：$S_n = \begin{cases} na_1, & q=1 \\ \frac{a_1(1-q^n)}{1-q} = \frac{a_1(q^n-1)}{q-1}, & q \neq 1 \end{cases}$

**分类讨论触发点：**
- 公比 $q$ 的符号（正/负）
- 公比 $q$ 是否等于1
- 等比中项的正负（$G^2 = ab$，$G = \pm\sqrt{ab}$）
