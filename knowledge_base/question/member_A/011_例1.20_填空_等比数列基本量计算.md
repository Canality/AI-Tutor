# 【A_011】等比数列基本量计算（2018东北四市高考）

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P11 例1.20
&gt; 
&gt; **来源**：2018年东北四市高考第15题

**题目：**

已知等比数列 $\{a_n\}$ 中各项均为正数，$S_n$ 是其前 $n$ 项和，且满足 $2S_3 = 8a_1 + 3a_2$，$a_4 = 16$，则 $S_4 =$ ______。

---

## 二、答案与解析

### 答案

**30**

### 解析

**步骤1：设定基本量**

设等比数列 $\{a_n\}$ 的首项为 $a_1$，公比为 $q$（$q&gt;0$，因为各项均为正数）。

**步骤2：建立方程**

由 $2S_3 = 8a_1 + 3a_2$，且 $S_3 = a_1 + a_2 + a_3 = a_1(1+q+q^2)$，$a_2 = a_1q$。

代入得：
$$2a_1(1+q+q^2) = 8a_1 + 3a_1q$$

因为 $a_1 \neq 0$，两边除以 $a_1$：
$$2(1+q+q^2) = 8 + 3q$$
$$2 + 2q + 2q^2 = 8 + 3q$$
$$2q^2 - q - 6 = 0$$

**步骤3：求解公比**

解方程 $2q^2 - q - 6 = 0$：
$$(2q+3)(q-2) = 0$$

得 $q = 2$ 或 $q = -\frac{3}{2}$。

因为各项均为正数，所以 $q = 2$。

**步骤4：求解首项**

由 $a_4 = 16$，即 $a_1q^3 = 16$，代入 $q=2$：
$$a_1 \cdot 8 = 16$$
$$a_1 = 2$$

**步骤5：计算 $S_4$**

$$S_4 = \frac{a_1(1-q^4)}{1-q} = \frac{2(1-16)}{1-2} = \frac{2(-15)}{-1} = 30$$

故填 **30**。

## 三、分步教学引导（用于AI Agent）

### 整体思路引导

**步骤1：识别已知条件**
- **引导语**："题目给出了关于前3项和的等式，以及第4项的值。等比数列由哪两个基本量确定？"
- **关键提示**：首项和公比
- **期望学生回答**：首项 $a_1$ 和公比 $q$

**步骤2：列出 $S_3$ 的表达式**
- **引导语**："用 $a_1$ 和 $q$ 表示 $S_3$ 和 $a_2$，代入第一个等式 $2S_3 = 8a_1 + 3a_2$。"
- **关键提示**：$S_3 = a_1(1+q+q^2)$，$a_2 = a_1q$
- **期望学生回答**：得到关于 $q$ 的方程 $2q^2 - q - 6 = 0$

**步骤3：解方程并取舍**
- **引导语**："解这个二次方程，得到两个解。根据题目条件'各项均为正数'，应该取哪个值？"
- **关键提示**：公比必须为正数
- **期望学生回答**：$q = 2$（舍去 $q = -\frac{3}{2}$）

**步骤4：求首项并计算 $S_4$**
- **引导语**："利用 $a_4 = 16$ 求出 $a_1$，然后计算前4项和。"
- **关键提示**：$a_4 = a_1q^3$
- **期望学生回答**：$a_1 = 2$，$S_4 = 30$

## 四、标签信息（用于分类和检索）

### 解题方法标签
- 基本量法（设 $a_1, q$）
- 方程思想
- 公式法

### 易错点标签
- 忘记舍去负公比（$q = -\frac{3}{2}$ 未舍去导致后续计算错误）
- $S_3$ 公式记错（误记为 $3a_1$ 或忘记包含 $a_3$）
- 计算 $S_4$ 时符号错误（$1-q = -1$ 处理不当）

## 五、关联拓展

### 等比数列基本公式

1. **通项公式**：$a_n = a_1q^{n-1}$
2. **前 $n$ 项和**：
   - $q=1$ 时，$S_n = na_1$
   - $q \neq 1$ 时，$S_n = \frac{a_1(1-q^n)}{1-q} = \frac{a_1-a_nq}{1-q}$
