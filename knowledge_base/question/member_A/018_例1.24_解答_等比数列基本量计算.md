# 【A_018】等比数列基本量计算（新课标III卷）

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P18 例1.24
&gt; 
&gt; **来源**：2018年新课标III卷文理第17题

**题目：**

等比数列 $\{a_n\}$ 中，$a_1 = 1$，$a_5 = 4a_3$。

(I) 求 $\{a_n\}$ 的通项公式；

(II) 记 $S_n$ 为 $\{a_n\}$ 的前 $n$ 项和，若 $S_m = 63$，求 $m$。

---

## 二、答案与解析

### 答案

(I) $a_n = 2^{n-1}$ 或 $a_n = (-2)^{n-1}$；

(II) $m = 6$。

### 解析

**(I) 求通项公式**

设等比数列 $\{a_n\}$ 的公比为 $q$。

由 $a_5 = 4a_3$，得 $a_1q^4 = 4a_1q^2$。

因为 $a_1 = 1 \neq 0$，所以 $q^4 = 4q^2$。

即 $q^2(q^2 - 4) = 0$，解得 $q^2 = 0$（舍去）或 $q^2 = 4$。

所以 $q = 2$ 或 $q = -2$。

当 $q = 2$ 时，$a_n = 2^{n-1}$；

当 $q = -2$ 时，$a_n = (-2)^{n-1}$。

**(II) 求 $m$ 的值**

**情况1：** 若 $a_n = (-2)^{n-1}$，则
$$S_n = \frac{1 \cdot [1 - (-2)^n]}{1 - (-2)} = \frac{1 - (-2)^n}{3}$$

令 $S_m = 63$，得 $\frac{1 - (-2)^m}{3} = 63$。

即 $1 - (-2)^m = 189$，所以 $(-2)^m = -188$。

此方程无正整数解，舍去。

**情况2：** 若 $a_n = 2^{n-1}$，则
$$S_n = \frac{1 \cdot (1 - 2^n)}{1 - 2} = 2^n - 1$$

令 $S_m = 63$，得 $2^m - 1 = 63$。

即 $2^m = 64 = 2^6$，所以 $m = 6$。

综上，$m = 6$。

## 三、分步教学引导（用于AI Agent）

### 第(I)问引导

**步骤1：建立方程**
- **引导语**："已知 $a_1=1$，$a_5=4a_3$，如何用公比 $q$ 表示这些项？"
- **关键提示**：$a_5 = q^4$，$a_3 = q^2$
- **期望学生回答**：$q^4 = 4q^2$，解得 $q = \pm 2$

**步骤2：写出通项**
- **引导语**："分两种情况写出通项公式。"
- **期望学生回答**：$a_n = 2^{n-1}$ 或 $a_n = (-2)^{n-1}$

### 第(II)问引导

**步骤3：分类讨论求和**
- **引导语**："等比数列求和公式是什么？需要注意什么？"
- **关键提示**：$S_n = \frac{a_1(1-q^n)}{1-q}$，需分 $q=2$ 和 $q=-2$ 讨论
- **期望学生回答**：分别代入计算

**步骤4：验证解的合理性**
- **引导语**："当 $q=-2$ 时，方程 $(-2)^m = -188$ 有解吗？"
- **关键提示**：$|-2|^m = 2^m$ 是2的幂，188不是2的幂
- **期望学生回答**：无解，舍去

**步骤5：确定答案**
- **引导语**："当 $q=2$ 时，解方程 $2^m = 64$。"
- **期望学生回答**：$m = 6$

### 相关公式总结

**等比数列求和：**
