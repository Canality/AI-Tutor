# 【A_016_3】等比数列对数求和与指数综合

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P16 变式1.23.2
&gt; 
&gt; **来源**：2014年广东卷理科第13题

**题目：**

若等比数列 $\{a_n\}$ 的各项均为正数，且 $a_{10} a_{11} + a_9 a_{12} = 2e^5$，则 $\ln a_1 + \ln a_2 + \cdots + \ln a_{20} = $ ______。

---

## 二、答案与解析

### 答案

$50$

### 解析

**步骤1：利用等比数列性质化简已知条件**

由于 $10 + 11 = 9 + 12 = 21$，所以 $a_{10} a_{11} = a_9 a_{12}$。

设 $a_{10} a_{11} = a_9 a_{12} = M$，则：
$$M + M = 2e^5$$
$$2M = 2e^5$$
$$M = e^5$$

即 $a_{10} a_{11} = e^5$。

**步骤2：化简对数求和**

$$\ln a_1 + \ln a_2 + \cdots + \ln a_{20} = \ln(a_1 a_2 \cdots a_{20})$$

**步骤3：利用性质计算乘积**

在等比数列中，$a_1 a_{20} = a_2 a_{19} = \cdots = a_{10} a_{11} = e^5$。

共有 $10$ 对，所以：
$$a_1 a_2 \cdots a_{20} = (e^5)^{10} = e^{50}$$

**步骤4：得出结果**

$$\ln(e^{50}) = 50$$

## 三、分步教学引导（用于AI Agent）

**步骤1：分析已知条件**
- **引导语**："观察$a_{10}a_{11}+a_9a_{12}$，下标有何特点？"
- **关键提示**：$10+11=9+12=21$
- **期望学生回答**：$a_{10}a_{11}=a_9a_{12}$，设其值为$M$，则$2M=2e^5$，$M=e^5$

**步骤2：转化对数求和**
- **引导语**："对数之和等于什么？"
- **关键提示**：$\ln a_1 + \cdots + \ln a_{20} = \ln(a_1 a_2 \cdots a_{20})$
- **期望学生回答**：真数的乘积的对数

**步骤3：计算乘积**
- **引导语**："20项乘积，利用等比性质两两配对，可以配成多少对？每对的值是多少？"
- **关键提示**：$a_1a_{20}=a_2a_{19}=\cdots=a_{10}a_{11}=e^5$，共10对
- **期望学生回答**：乘积为$(e^5)^{10}=e^{50}$

**步骤4：最终计算**
- **引导语**："计算$\ln(e^{50})$。"
- **期望学生回答**：$50$

## 五、关联拓展

**等比数列前$n$项积公式：**
若$\{a_n\}$为等比数列，则$P_n = a_1 a_2 \cdots a_n = (a_1 a_n)^{n/2} = a_{\frac{n+1}{2}}^n$（$n$为奇数时）。
