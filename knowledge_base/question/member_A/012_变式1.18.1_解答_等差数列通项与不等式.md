# 【A_012_3】等差数列通项与不等式

## 一、题目内容

&gt; **来源**：2019年全国I卷文科第18题

**题目：**

记 $S_n$ 为等差数列 $\{a_n\}$ 的前 $n$ 项和，已知 $S_9 = -a_5$。

(I) 若 $a_3 = 4$，求 $\{a_n\}$ 的通项公式；

(II) 若 $a_1 &gt; 0$，求使得 $S_n \geq a_n$ 的 $n$ 的取值范围。

## 二、答案与解析

### 答案

(I) $a_n = -2n + 10$

(II) $1 \leq n \leq 10$，$n \in \mathbb{N}^*$

### 解析

**步骤1：利用 $S_9 = -a_5$ 建立关系**

$$S_9 = \frac{9(a_1 + a_9)}{2} = 9a_5 = -a_5$$

因此 $10a_5 = 0$，即 $a_5 = 0$。

**步骤2：第(I)问求解**

由 $a_3 = 4$ 和 $a_5 = 0$：
$$a_5 - a_3 = 2d = -4$$
$$d = -2$$

$$a_1 = a_3 - 2d = 4 - 2(-2) = 8$$

通项公式：
$$a_n = 8 + (n-1)(-2) = 10 - 2n$$

**步骤3：第(II)问求解**

由 $a_5 = 0$ 得 $a_1 + 4d = 0$，即 $a_1 = -4d$。

由于 $a_1 &gt; 0$，所以 $d &lt; 0$。

不等式 $S_n \geq a_n$：
$$na_1 + \frac{n(n-1)}{2}d \geq a_1 + (n-1)d$$

代入 $a_1 = -4d$（注意 $d &lt; 0$，除以 $d$ 时不等号变向）：
$$-4nd + \frac{n(n-1)}{2}d \geq -4d + (n-1)d$$

除以 $d$（负数）：
$$-4n + \frac{n(n-1)}{2} \leq -4 + (n-1)$$

化简：
$$-8n + n^2 - n \leq -8 + 2n - 2$$
$$n^2 - 9n + 10 \leq 0$$

解得：
$$\frac{9 - \sqrt{41}}{2} \leq n \leq \frac{9 + \sqrt{41}}{2}$$

近似为 $1.3 \leq n \leq 7.7$？让我重新计算。

重新计算：
$$S_n = na_1 + \frac{n(n-1)}{2}d = n(-4d) + \frac{n(n-1)}{2}d = d\left(-4n + \frac{n^2-n}{2}\right) = \frac{d}{2}(n^2 - 9n)$$

$$a_n = -4d + (n-1)d = d(n-5)$$

不等式：
$$\frac{d}{2}(n^2 - 9n) \geq d(n-5)$$

由于 $d &lt; 0$，除以 $d$ 变号：
$$\frac{1}{2}(n^2 - 9n) \leq n - 5$$
$$n^2 - 9n \leq 2n - 10$$
$$n^2 - 11n + 10 \leq 0$$
$$(n-1)(n-10) \leq 0$$

所以 $1 \leq n \leq 10$。

## 三、分步教学引导

**步骤1：简化 $S_9 = -a_5$**
- **引导语**："$S_9$ 可以用中间项 $a_5$ 表示为什么？"
- **关键提示**：$S_9 = 9a_5$
- **期望学生回答**：$9a_5 = -a_5$，所以 $a_5 = 0$

**步骤2：第(I)问求公差**
- **引导语**："已知 $a_3 = 4$ 和 $a_5 = 0$，求 $d$ 和 $a_1$。"
- **关键提示**：$d = \frac{a_5 - a_3}{2}$
- **期望学生回答**：$d = -2$，$a_1 = 8$，$a_n = 10 - 2n$

**步骤3：第(II)问建立不等式**
- **引导语**："用 $a_1 = -4d$ 表示 $S_n$ 和 $a_n$，注意 $d &lt; 0$。"
- **关键提示**：不等式两边除以负数要变号
- **期望学生回答**：$1 \leq n \leq 10$
