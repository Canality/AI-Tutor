# 【A_015_4】等比数列与对数求和

## 一、题目内容

**题目：**

若等比数列 $\{a_n\}$ 的各项均为正数，且 $a_{10} a_{11} + a_9 a_{12} = 2e^5$，则 $\ln a_1 + \ln a_2 + \cdots + \ln a_{20} =$ ______。

## 二、答案与解析

### 答案

$50$

### 解析

**步骤1：化简条件**

由等比数列性质：
$$a_{10} a_{11} = a_9 a_{12} = a_1 a_{20}$$

所以：
$$2a_{10} a_{11} = 2e^5$$
$$a_{10} a_{11} = e^5$$

**步骤2：计算对数和**

$$\ln a_1 + \ln a_2 + \cdots + \ln a_{20} = \ln(a_1 a_2 \cdots a_{20})$$

**步骤3：配对求积**

$$a_1 a_{20} = a_2 a_{19} = \cdots = a_{10} a_{11} = e^5$$

共 $10$ 对，所以：
$$a_1 a_2 \cdots a_{20} = (e^5)^{10} = e^{50}$$

**步骤4：求对数**

$$\ln(e^{50}) = 50$$

## 三、分步教学引导

**步骤1：化简条件**
- **引导语**："$a_{10}a_{11}$ 与 $a_9a_{12}$ 有什么关系？"
- **期望学生回答**：相等，都等于 $e^5$

**步骤2：配对**
- **引导语**："20项可以配成多少对？每对的乘积是多少？"
- **期望学生回答**：10对，每对 $e^5$

**步骤3：计算总积和对数**
- **期望学生回答**：$50$
