# 【A_015_2】等比数列与对数综合

## 一、题目内容

**题目：**

等比数列 $\{a_n\}$ 满足 $a_n &gt; 0$，$n \in \mathbb{N}^*$，且 $a_3 a_{2n-3} = 2^{2n}$（$n \geq 2$），则当 $n \geq 1$ 时，$\log_2 a_1 + \log_2 a_2 + \cdots + \log_2 a_{2n-1} =$（ ）。

A. $n(2n-1)$  
B. $(n+1)^2$  
C. $n^2$  
D. $(n-1)^2$

## 二、答案与解析

### 答案

**A**（$n(2n-1)$）

### 解析

**步骤1：利用等比数列性质**

由 $a_3 a_{2n-3} = a_n^2 = 2^{2n}$（因为 $3 + (2n-3) = 2n = n + n$）。

所以 $a_n = 2^n$（因为 $a_n &gt; 0$）。

**步骤2：化简求和式**

设 $S = \log_2 a_1 + \log_2 a_2 + \cdots + \log_2 a_{2n-1}$。

则 $S = \log_2 (a_1 a_2 \cdots a_{2n-1})$。

**步骤3：计算乘积**

$$a_1 a_2 \cdots a_{2n-1} = 2^1 \cdot 2^2 \cdots 2^{2n-1} = 2^{1+2+\cdots+(2n-1)} = 2^{\frac{(2n-1)(2n)}{2}} = 2^{n(2n-1)}$$

**步骤4：求对数**

$$S = \log_2 2^{n(2n-1)} = n(2n-1)$$

## 三、分步教学引导

**步骤1：确定通项**
- **引导语**："由 $a_3 a_{2n-3} = a_n^2 = 2^{2n}$，得 $a_n = ?$"
- **期望学生回答**：$a_n = 2^n$

**步骤2：对数性质**
- **引导语**："对数和可以转化为乘积的对数。"
- **期望学生回答**：$\log_2 (a_1 a_2 \cdots a_{2n-1})$

**步骤3：指数求和**
- **引导语**："计算 $1+2+\cdots+(2n-1)$。"
- **期望学生回答**：$n(2n-1)$
