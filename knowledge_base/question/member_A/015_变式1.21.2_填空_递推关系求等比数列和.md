# 【A_015_2】递推关系判定等比数列

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P15 变式1.21.2
&gt; 
&gt; **来源**：2018年合肥质检

**题目：**

已知数列 $\{a_n\}$ 中，$a_1 = 2$，且 $\frac{a_{n+1}^2}{a_n} = 4(a_{n+1} - a_n)$（$n \in \mathbb{N}^*$），则其前 $9$ 项和 $S_9 = $ ______。

---

## 二、答案与解析

### 答案

$1022$

### 解析

**步骤1：化简递推关系**

由 $\frac{a_{n+1}^2}{a_n} = 4(a_{n+1} - a_n)$，两边同乘 $a_n$（显然 $a_n &gt; 0$，否则递推无意义）：

$$a_{n+1}^2 = 4a_n a_{n+1} - 4a_n^2$$

移项整理：
$$a_{n+1}^2 - 4a_n a_{n+1} + 4a_n^2 = 0$$

**步骤2：因式分解判定数列类型**

左边是完全平方：
$$(a_{n+1} - 2a_n)^2 = 0$$

所以：
$$a_{n+1} = 2a_n$$

因此 $\{a_n\}$ 是等比数列，公比 $q = 2$。

**步骤3：求前9项和**

已知 $a_1 = 2$，$q = 2$，项数 $n = 9$。

$$S_9 = \frac{a_1(1 - q^9)}{1 - q} = \frac{2(1 - 2^9)}{1 - 2} = \frac{2(1 - 512)}{-1} = 2 \times 511 = 1022$$

---

## 三、分步教学引导（用于AI Agent）

**步骤1：观察结构**
- **引导语**："递推式左边是平方除以$a_n$，右边是4倍的差，尝试去分母化简。"
- **关键提示**：两边乘$a_n$消去分母

**步骤2：识别完全平方**
- **引导语**："整理后得到$a_{n+1}^2 - 4a_n a_{n+1} + 4a_n^2 = 0$，这可以因式分解吗？"
- **关键提示**：$(x-2y)^2 = x^2-4xy+4y^2$
- **期望学生回答**：$(a_{n+1}-2a_n)^2=0$

**步骤3：确定数列性质**
- **引导语**："由$a_{n+1}=2a_n$，这是什么数列？"
- **关键提示**：等比数列，公比为2

**步骤4：计算求和**
- **引导语**："代入等比数列求和公式，计算$S_9$。"
- **关键提示**：$S_9 = 2(2^9-1)/(2-1) = 2\times 511 = 1022$

## 五、关联拓展

**常见递推转等比类型：**
- $a_{n+1}^2 = a_n a_{n+2}$（等比中项形式）
- $a_{n+1} = pa_n$（直接等比）
- $a_{n+1}^2 - pa_n a_{n+1} + qa_n^2 = 0$（可因式分解为等比）
