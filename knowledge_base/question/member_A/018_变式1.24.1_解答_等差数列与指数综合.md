# 【A_018_2】等差数列与指数综合（北京卷）

## 一、题目内容

&gt; **原题位置**：P18 变式1.24.1
&gt; 
&gt; **来源**：2018年北京文第15题

**题目：**

设 $\{a_n\}$ 是等差数列，且 $a_1 = \ln 2$，$a_2 + a_3 = 5\ln 2$。

(I) 求 $\{a_n\}$ 的通项公式；

(II) 求 $e^{a_1} + e^{a_2} + \cdots + e^{a_n}$。

---

## 二、答案与解析

### 答案

(I) $a_n = n\ln 2$；

(II) $2^{n+1} - 2$。

### 解析

**(I) 求通项公式**

设等差数列 $\{a_n\}$ 的公差为 $d$。

由 $a_2 + a_3 = 5\ln 2$，得 $(a_1 + d) + (a_1 + 2d) = 5\ln 2$。

即 $2a_1 + 3d = 5\ln 2$。

代入 $a_1 = \ln 2$：
$$2\ln 2 + 3d = 5\ln 2$$
$$3d = 3\ln 2$$
$$d = \ln 2$$

所以 $a_n = a_1 + (n-1)d = \ln 2 + (n-1)\ln 2 = n\ln 2$。

**(II) 求指数和**

首先 $e^{a_n} = e^{n\ln 2} = (e^{\ln 2})^n = 2^n$。

所以 $\{e^{a_n}\}$ 是首项为 $e^{a_1} = 2$，公比为 $2$ 的等比数列。

求和：
$$e^{a_1} + e^{a_2} + \cdots + e^{a_n} = 2 + 2^2 + \cdots + 2^n = \frac{2(1 - 2^n)}{1 - 2} = 2(2^n - 1) = 2^{n+1} - 2$$

## 三、分步教学引导（用于AI Agent）

### 第(I)问引导

**步骤1：建立方程**
- **引导语**："设公差为 $d$，用 $a_1$ 和 $d$ 表示 $a_2$ 和 $a_3$，代入已知条件。"
- **关键提示**：$a_2 = a_1 + d$，$a_3 = a_1 + 2d$
- **期望学生回答**：$2a_1 + 3d = 5\ln 2$，解得 $d = \ln 2$

**步骤2：写出通项**
- **引导语**："代入通项公式 $a_n = a_1 + (n-1)d$。"
- **期望学生回答**：$a_n = n\ln 2$

### 第(II)问引导

**步骤3：化简 $e^{a_n}$**
- **引导语**："$e^{n\ln 2}$ 等于什么？利用指数对数互化。"
- **关键提示**：$e^{\ln x} = x$，$(e^{\ln 2})^n = 2^n$
- **期望学生回答**：$e^{a_n} = 2^n$

**步骤4：识别数列类型**
- **引导语**："数列 $\{e^{a_n}\}$ 是什么数列？首项和公比分别是多少？"
- **期望学生回答**：等比数列，首项2，公比2

**步骤5：求和**
- **引导语**："用等比数列求和公式计算。"
- **期望学生回答**：$S_n = 2^{n+1} - 2$

## 五、关联拓展

### 重要结论
**等差数列的指数化：**
若 $\{a_n\}$ 是等差数列，则 $\{e^{a_n}\}$ 是等比数列。

**证明：** $\frac{e^{a_{n+1}}}{e^{a_n}} = e^{a_{n+1}-a_n} = e^d$（常数）。
