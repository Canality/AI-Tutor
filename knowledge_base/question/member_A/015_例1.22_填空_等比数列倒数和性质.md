# 【A_015_3】等比数列倒数和与性质应用

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P15 例1.22
&gt; 
&gt; **来源**：2018年石家庄模拟

**题目：**

在等比数列 $\{a_n\}$ 中，若 $a_7 + a_8 + a_9 + a_{10} = \frac{15}{8}$，$a_8 a_9 = -\frac{9}{8}$，则 $\frac{1}{a_7} + \frac{1}{a_8} + \frac{1}{a_9} + \frac{1}{a_{10}} = $ ______。

---

## 二、答案与解析

### 答案

$-\frac{5}{3}$

### 解析

**步骤1：分析倒数和的结构**

$$\frac{1}{a_7} + \frac{1}{a_{10}} = \frac{a_7 + a_{10}}{a_7 a_{10}}$$
$$\frac{1}{a_8} + \frac{1}{a_9} = \frac{a_8 + a_9}{a_8 a_9}$$

**步骤2：利用等比数列性质**

在等比数列中，若 $m + n = p + q$，则 $a_m a_n = a_p a_q$。

由于 $7 + 10 = 8 + 9 = 17$，所以：
$$a_7 a_{10} = a_8 a_9 = -\frac{9}{8}$$

**步骤3：合并计算**

原式 $= \frac{a_7 + a_{10}}{a_7 a_{10}} + \frac{a_8 + a_9}{a_8 a_9}$

由于 $a_7 a_{10} = a_8 a_9$，提取公因式：

$$= \frac{(a_7 + a_{10}) + (a_8 + a_9)}{a_8 a_9} = \frac{a_7 + a_8 + a_9 + a_{10}}{a_8 a_9}$$

**步骤4：代入数值**

$$= \frac{\frac{15}{8}}{-\frac{9}{8}} = \frac{15}{8} \times \left(-\frac{8}{9}\right) = -\frac{15}{9} = -\frac{5}{3}$$

## 三、分步教学引导（用于AI Agent）

**步骤1：观察目标式**
- **引导语**："要求四个倒数的和，先两两组合通分，看看有什么规律？"
- **关键提示**：$\frac{1}{a_7}+\frac{1}{a_{10}}=\frac{a_7+a_{10}}{a_7a_{10}}$

**步骤2：联系已知条件**
- **引导语**："分子变成了$a_7+a_8+a_9+a_{10}$，这正是已知条件。分母呢？"
- **关键提示**：$a_7a_{10}$与$a_8a_9$的关系

**步骤3：应用等比性质**
- **引导语**："观察下标：7+10=？，8+9=？，这说明什么？"
- **关键提示**：$7+10=8+9=17$，所以$a_7a_{10}=a_8a_9$
- **期望学生回答**：分母相同，都是$-\frac{9}{8}$

**步骤4：整体代换**
- **引导语**："现在可以合并分子，直接代入已知值计算。"
- **关键提示**：原式$=\frac{15/8}{-9/8}=-\frac{5}{3}$

## 五、关联拓展

**等比数列重要性质：**
若 $\{a_n\}$ 为等比数列，则 $\{\frac{1}{a_n}\}$ 也是等比数列，公比为 $\frac{1}{q}$。

**倒数和公式：**
若 $a_k \cdot a_{n+1-k} = C$（常数），则倒序相加可得 $\sum \frac{1}{a_k} = \frac{\sum a_k}{C}$。
