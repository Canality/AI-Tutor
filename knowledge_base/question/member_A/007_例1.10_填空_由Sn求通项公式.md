# 【A_007】由Sn求通项公式（2014广东改编）

## 一、题目内容

&gt; **原题位置**：P7 例1.10
&gt; 
&gt; **来源**：2014年广东文19改编

**题目：**

设各项均为正数的数列 $\{a_n\}$ 的前 $n$ 项和为 $S_n$，且满足 $S_n^2 - (n^2 + n - 3)S_n - 3(n^2 + n) = 0$，$n \in \mathbb{N}^*$，则 $\{a_n\}$ 的通项公式为 ______。

---

## 二、答案与解析

### 答案

$$a_n = 2n$$

### 解析

**步骤1：因式分解求 $S_n$**

观察方程 $S_n^2 - (n^2 + n - 3)S_n - 3(n^2 + n) = 0$，将其视为关于 $S_n$ 的二次方程。

因式分解得：
$$(S_n + 3)[S_n - (n^2 + n)] = 0$$

**步骤2：确定 $S_n$ 的表达式**

由于数列各项均为正数，故 $S_n &gt; 0$，因此 $S_n + 3 \neq 0$。

所以 $S_n = n^2 + n$。

**步骤3：利用 $a_n$ 与 $S_n$ 的关系求通项**

当 $n = 1$ 时：
$$a_1 = S_1 = 1^2 + 1 = 2$$

当 $n \geq 2$ 时：
$$a_n = S_n - S_{n-1} = (n^2 + n) - [(n-1)^2 + (n-1)]$$
$$= n^2 + n - (n^2 - 2n + 1 + n - 1)$$
$$= n^2 + n - n^2 + n$$
$$= 2n$$

**步骤4：验证 $n=1$ 的情况**

当 $n=1$ 时，$a_1 = 2 \times 1 = 2$，与步骤3结果一致。

故 $\{a_n\}$ 的通项公式为 $a_n = 2n$，$n \in \mathbb{N}^*$。

## 三、分步教学引导（用于AI Agent）

### 整体思路引导

**步骤1：分析方程结构**
- **引导语**："题目给出了一个关于 $S_n$ 的二次方程，我们首先要做什么？"
- **关键提示**：把 $S_n$ 看作未知数，尝试因式分解
- **期望学生回答**：对方程进行因式分解，得到 $(S_n + 3)[S_n - (n^2 + n)] = 0$

**步骤2：确定 $S_n$ 的值**
- **引导语**："因式分解后得到两个可能的解，如何判断哪个是正确的？"
- **关键提示**：题目告诉我们数列各项均为正数，这意味着 $S_n$ 必须大于0
- **期望学生回答**：因为 $a_n &gt; 0$，所以 $S_n &gt; 0$，故 $S_n = n^2 + n$（舍去 $S_n = -3$）

**步骤3：求 $a_n$ 的通项**
- **引导语**："现在我们知道了 $S_n = n^2 + n$，如何求 $a_n$？"
- **关键提示**：利用 $a_n$ 与 $S_n$ 的基本关系，注意要分 $n=1$ 和 $n \geq 2$ 讨论
- **期望学生回答**：$a_n = S_n - S_{n-1}$（$n \geq 2$），$a_1 = S_1$

**步骤4：计算并验证**
- **引导语**："计算 $S_n - S_{n-1}$，并验证 $n=1$ 时是否满足。"
- **关键提示**：展开 $(n-1)^2$ 时注意符号
- **期望学生回答**：$a_n = 2n$（$n \geq 2$），且 $n=1$ 时也满足，故 $a_n = 2n$

## 四、标签信息（用于分类和检索）

### 解题方法标签
- 类讨论型（由 $S_n$ 求 $a_n$）
- 因式分解法（解关于 $S_n$ 的二次方程）
- 公式法（$a_n = S_n - S_{n-1}$）
- 验证法（验证 $n=1$ 的连续性）

### 易错点标签
- 忘记舍去 $S_n = -3$（忽略"各项为正"的条件）
- 因式分解错误（十字相乘法不熟练）
- 未验证 $n=1$ 时的一致性（直接认为 $a_n = 2n$ 对所有 $n$ 成立）
- 计算 $S_{n-1}$ 时展开错误（$(n-1)^2$ 展开为 $n^2 - 2n + 1$）

### 题目特征标签
- 高考改编题（广东卷）
- 中等难度（构造思想）
- 由和求通项
- 二次方程与数列结合

## 五、关联拓展

### 相关方法总结

**由 $S_n$ 求 $a_n$ 的标准步骤：**
1. 求出 $S_n$ 的表达式（或利用已知）
2. 当 $n = 1$ 时，$a_1 = S_1$
3. 当 $n \geq 2$ 时，$a_n = S_n - S_{n-1}$
4. 验证 $n=1$ 是否满足第3步的表达式，若满足则合并，否则写成分段形式

**常见 $S_n$ 形式：**
- $S_n = An^2 + Bn$（对应等差数列）
- $S_n = Aq^n - A$（对应等比数列）
