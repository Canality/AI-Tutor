# 【A_032_2】根式构造求通项（构造法进阶）

## 一、题目内容

&gt; **原题位置**：P32 例2.2
&gt; 
&gt; **来源**：教材例题

**题目：**

已知数列 $\{a_n\}$ 各项均为正数，满足 $a_n + \frac{1}{a_n} = 2S_n$，求 $\{a_n\}$ 的通项公式。

---

## 二、答案与解析

### 答案

$$a_n = \sqrt{n} - \sqrt{n-1} \quad (n \in \mathbb{N}^*)$$

### 解析

**步骤1：求首项 $a_1$**

当 $n = 1$ 时，$a_1 = S_1$，代入已知条件：
$$a_1 + \frac{1}{a_1} = 2a_1$$
两边同乘$a_1$（$a_1 &gt; 0$）：
$$a_1^2 + 1 = 2a_1^2$$
解得$a_1^2 = 1$，因为$a_1 &gt; 0$，所以$a_1 = 1$。

**步骤2：构造关于$S_n$的递推关系**

当 $n \geq 2$ 时，$a_n = S_n - S_{n-1}$，代入已知条件：
$$S_n - S_{n-1} + \frac{1}{S_n - S_{n-1}} = 2S_n$$

移项整理：
$$\frac{1}{S_n - S_{n-1}} = 2S_n - (S_n - S_{n-1}) = S_n + S_{n-1}$$

两边同乘$(S_n - S_{n-1})$：
$$1 = (S_n + S_{n-1})(S_n - S_{n-1}) = S_n^2 - S_{n-1}^2$$

即：
$$S_n^2 - S_{n-1}^2 = 1 \quad (n \geq 2)$$

**步骤3：求$S_n$的表达式**

由$S_n^2 - S_{n-1}^2 = 1$可知，数列$\{S_n^2\}$是等差数列，公差为$1$。

首项$S_1^2 = a_1^2 = 1$。

因此：
$$S_n^2 = S_1^2 + (n-1) \times 1 = 1 + n - 1 = n$$

因为数列各项均为正数，所以$S_n &gt; 0$，故：
$$S_n = \sqrt{n}$$

**步骤4：求$a_n$的通项公式**

当 $n \geq 2$ 时：
$$a_n = S_n - S_{n-1} = \sqrt{n} - \sqrt{n-1}$$

验证$n = 1$时：
$$\sqrt{1} - \sqrt{0} = 1 = a_1$$

因此，通项公式为：
$$a_n = \sqrt{n} - \sqrt{n-1} \quad (n \in \mathbb{N}^*)$$

## 三、分步教学引导（用于AI Agent）

**步骤1：分析已知条件**
- **引导语**："观察等式$a_n + \frac{1}{a_n} = 2S_n$，左边是$a_n$及其倒数，右边是$2S_n$。我们已知什么关系可以连接$a_n$和$S_n$？"
- **关键提示**：$a_n = S_n - S_{n-1}$（$n \geq 2$），$a_1 = S_1$
- **期望学生回答**：利用$a_n$与$S_n$的关系消去$a_n$

**步骤2：处理首项**
- **引导语**："当$n=1$时，$a_1 = S_1$，代入等式求解$a_1$。"
- **关键提示**：得到方程$a_1 + \frac{1}{a_1} = 2a_1$
- **期望学生回答**：$a_1 = 1$

**步骤3：消元构造**
- **引导语**："当$n \geq 2$时，将$a_n = S_n - S_{n-1}$代入原式，然后整理，尝试将分式移到一边。"
- **关键提示**：$\frac{1}{S_n - S_{n-1}} = S_n + S_{n-1}$
- **期望学生回答**：得到$(S_n - S_{n-1})(S_n + S_{n-1}) = 1$

**步骤4：识别平方差结构**
- **引导语**："左边$(S_n - S_{n-1})(S_n + S_{n-1})$可以用什么公式化简？"
- **关键提示**：平方差公式
- **期望学生回答**：$S_n^2 - S_{n-1}^2 = 1$

**步骤5：判断新数列类型**
- **引导语**："$S_n^2 - S_{n-1}^2 = 1$说明数列$\{S_n^2\}$是什么数列？首项是多少？"
- **关键提示**：相邻两项差为常数1
- **期望学生回答**：等差数列，首项$S_1^2 = 1$，公差$d = 1$

**步骤6：求$S_n$**
- **引导语**："根据等差数列通项公式求$S_n^2$，然后开方。注意什么条件？"
- **关键提示**：各项均为正数，所以$S_n &gt; 0$
- **期望学生回答**：$S_n = \sqrt{n}$

**步骤7：回代求$a_n$**
- **引导语**："利用$a_n = S_n - S_{n-1}$求通项，并验证$n=1$的情况。"
- **关键提示**：$S_{n-1} = \sqrt{n-1}$
- **期望学生回答**：$a_n = \sqrt{n} - \sqrt{n-1}$

## 四、标签信息（用于分类和检索）

### 解题方法标签
- 构造法（构造$\{S_n^2\}$为等差数列）
- 消元法（消去$a_n$保留$S_n$）
- 平方差公式
- 类讨论型（分段求通项）

### 易错点标签
- 忘记验证各项为正数的条件（开方时取正负问题）
- 代入$a_n = S_n - S_{n-1}$后无法正确整理代数式
- 未验证$n=1$时最终通项公式是否成立
- 混淆$S_n^2$与$(S_n)^2$的书写

### 题目特征标签
- 构造法进阶
- 根式型通项
- 平方差结构
- 正项数列约束

## 五、关联拓展

### 相关方法总结

**构造等差数列的常见情形：**
1. $S_n^2 - S_{n-1}^2 = c$（常数）→ 构造$\{S_n^2\}$
2. $\frac{1}{S_n} - \frac{1}{S_{n-1}} = c$ → 构造$\{\frac{1}{S_n}\}$

**平方差公式在数列中的应用：**
$$(S_n - S_{n-1})(S_n + S_{n-1}) = S_n^2 - S_{n-1}^2 = a_n(2S_n - a_n)$$

## 六、备注

- **本题地位**：构造法求通项的经典例题，体现"反其道而行之"（消去$a_n$求$S_n$）的数学思想。
- **教学价值**：训练学生观察代数结构的能力，掌握平方差公式的灵活运用。
- **关键技巧**：当直接求$a_n$困难时，可先求$S_n$再反推$a_n$。
- **学生常见困难**：
  1. 不敢将$a_n = S_n - S_{n-1}$代入（觉得太复杂）
  2. 代数变形能力不足，无法从$\frac{1}{S_n - S_{n-1}} = S_n + S_{n-1}$看出平方差结构
