# 【A_014_2】等差数列与指数求和综合

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P14 变式1.24.1
&gt; 
&gt; **来源**：2018年北京卷文科第15题

**题目：**

设 $\{a_n\}$ 是等差数列，且 $a_1 = \ln 2$，$a_2 + a_3 = 5\ln 2$。

(I) 求 $\{a_n\}$ 的通项公式；

(II) 求 $e^{a_1} + e^{a_2} + \cdots + e^{a_n}$。

---

## 二、答案与解析

### 答案

(I) $a_n = n\ln 2$（或 $a_n = \ln(2^n)$）；

(II) $2^{n+1} - 2$。

### 解析

**步骤1：求等差数列公差**

设等差数列 $\{a_n\}$ 的公差为 $d$。

已知 $a_1 = \ln 2$，则：
- $a_2 = a_1 + d = \ln 2 + d$
- $a_3 = a_1 + 2d = \ln 2 + 2d$

由条件 $a_2 + a_3 = 5\ln 2$，代入得：
$$(\ln 2 + d) + (\ln 2 + 2d) = 5\ln 2$$

$$2\ln 2 + 3d = 5\ln 2$$

$$3d = 3\ln 2$$

$$d = \ln 2$$

**步骤2：求通项公式**

根据等差数列通项公式：
$$a_n = a_1 + (n-1)d = \ln 2 + (n-1)\ln 2 = n\ln 2$$

利用对数性质，也可写成：
$$a_n = \ln(2^n)$$

**步骤3：分析第(II)问，化简 $e^{a_n}$**

由第(I)问知 $a_n = n\ln 2 = \ln(2^n)$，所以：
$$e^{a_n} = e^{\ln(2^n)} = 2^n$$

**步骤4：求和（等比数列求和）**

原式 $e^{a_1} + e^{a_2} + \cdots + e^{a_n}$ 即：
$$2^1 + 2^2 + \cdots + 2^n$$

这是首项为 $2$，公比为 $2$，项数为 $n$ 的等比数列求和。

根据等比数列求和公式：
$$S_n = \frac{2(1-2^n)}{1-2} = \frac{2(2^n-1)}{2-1} = 2(2^n-1) = 2^{n+1} - 2$$

## 三、分步教学引导（用于AI Agent）

### 第(I)问引导

**步骤1：识别数列类型与已知条件**
- **引导语**："题目明确告诉我们这是什么数列？已知哪些条件？"
- **关键提示**：等差数列，$a_1 = \ln 2$，$a_2 + a_3 = 5\ln 2$
- **期望学生回答**：等差数列，首项为 $\ln 2$，且第二项加第三项等于 $5\ln 2$

**步骤2：设公差并表示项**
- **引导语**："设公差为 $d$，用 $a_1$ 和 $d$ 表示 $a_2$ 和 $a_3$。"
- **关键提示**：$a_2 = \ln 2 + d$，$a_3 = \ln 2 + 2d$
- **期望学生回答**：正确写出两项表达式

**步骤3：建立方程求解**
- **引导语**："将 $a_2$ 和 $a_3$ 的表达式代入 $a_2 + a_3 = 5\ln 2$，解出 $d$。"
- **关键提示**：合并同类项，$2\ln 2 + 3d = 5\ln 2$
- **期望学生回答**：$3d = 3\ln 2$，所以 $d = \ln 2$

**步骤4：写出通项**
- **引导语**："现在用 $a_1$ 和 $d$ 写出通项公式，并化简。"
- **关键提示**：$a_n = \ln 2 + (n-1)\ln 2 = n\ln 2$
- **期望学生回答**：$a_n = n\ln 2$ 或 $\ln(2^n)$

### 第(II)问引导

**步骤5：理解指数运算**
- **引导语**："观察要求的式子 $e^{a_1} + e^{a_2} + \cdots + e^{a_n}$，先计算一般项 $e^{a_k}$ 等于什么？"
- **关键提示**：利用对数恒等式 $e^{\ln x} = x$
- **期望学生回答**：$e^{a_k} = e^{k\ln 2} = (e^{\ln 2})^k = 2^k$

**步骤6：识别求和类型**
- **引导语**："那么原式变成什么数列的求和？首项、公比、项数分别是多少？"
- **关键提示**：$2^1 + 2^2 + \cdots + 2^n$
- **期望学生回答**：等比数列求和，首项2，公比2，项数n

**步骤7：应用求和公式**
- **引导语**："代入等比数列求和公式计算结果。"
- **关键提示**：$S_n = \frac{a_1(q^n-1)}{q-1} = \frac{2(2^n-1)}{1} = 2^{n+1}-2$
- **期望学生回答**：$2^{n+1} - 2$

## 五、关联拓展

### 核心知识链接

**对数与指数的互化：**
- $e^{\ln x} = x$（$x &gt; 0$）
- $e^{k\ln a} = (e^{\ln a})^k = a^k$
- $\ln(a^n) = n\ln a$

**等差数列与等比数列的复合：**
若 $\{a_n\}$ 是等差数列，则 $\{e^{a_n}\}$ 是等比数列（当公差为常数时，公比为 $e^d$）。
反之，若 $\{a_n\}$ 是等比数列且各项为正，则 $\{\ln a_n\}$ 是等差数列。
