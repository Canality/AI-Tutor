# 【A_009_2】两个等差数列和的比值应用

## 一、题目内容

&gt; **原题位置**：P9 例1.13

**题目：**

已知两个等差数列 $\{a_n\}$ 和 $\{b_n\}$ 的前 $n$ 项和分别为 $A_n$ 和 $B_n$，且 $\frac{A_n}{B_n} = \frac{7n + 45}{n + 3}$，则使得 $\frac{a_n}{b_n}$ 为整数的正整数 $n$ 的个数是（ ）。

A. 2  
B. 3  
C. 4  
D. 5

---

## 二、答案与解析

### 答案

**D**

### 解析

**步骤1：利用等差数列重要性质**

对于等差数列，有结论：
$$\frac{a_n}{b_n} = \frac{A_{2n-1}}{B_{2n-1}}$$

**步骤2：代入已知条件**

$$\frac{a_n}{b_n} = \frac{A_{2n-1}}{B_{2n-1}} = \frac{7(2n-1) + 45}{(2n-1) + 3} = \frac{14n - 7 + 45}{2n + 2} = \frac{14n + 38}{2n + 2}$$

**步骤3：化简分式**

$$\frac{14n + 38}{2n + 2} = \frac{7n + 19}{n + 1} = \frac{7(n + 1) + 12}{n + 1} = 7 + \frac{12}{n + 1}$$

**步骤4：确定整数条件**

要使 $\frac{a_n}{b_n}$ 为整数，需要 $\frac{12}{n+1}$ 为整数，即 $(n+1)$ 是 $12$ 的正约数。

$12$ 的正约数有：$1, 2, 3, 4, 6, 12$。

对应 $n+1 \in \{1, 2, 3, 4, 6, 12\}$，即 $n \in \{0, 1, 2, 3, 5, 11\}$。

由于 $n$ 为正整数，$n \geq 1$，所以 $n \in \{1, 2, 3, 5, 11\}$。

共 **5** 个。

故选 **D**。

## 三、分步教学引导（用于AI Agent）

### 整体思路引导

**步骤1：回忆重要结论**
- **引导语**："题目给出了两个等差数列前 $n$ 项和的比值，要求通项的比值，应该用什么结论？"
- **关键提示**：$\frac{a_n}{b_n} = \frac{A_{2n-1}}{B_{2n-1}}$（第 $n$ 项比等于前 $2n-1$ 项和的比）
- **期望学生回答**：利用性质 $\frac{a_n}{b_n} = \frac{A_{2n-1}}{B_{2n-1}}$

**步骤2：代入计算**
- **引导语**："将 $2n-1$ 代入 $\frac{A_n}{B_n} = \frac{7n + 45}{n + 3}$ 中的 $n$。"
- **关键提示**：注意替换所有 $n$ 为 $2n-1$
- **期望学生回答**：$\frac{7(2n-1) + 45}{(2n-1) + 3} = \frac{14n + 38}{2n + 2}$

**步骤3：分式化简**
- **引导语**："将 $\frac{14n + 38}{2n + 2}$ 化简，使其出现常数加真分式的形式。"
- **关键提示**：分子凑出分母的倍数，$14n + 38 = 7(2n + 2) + 24$？不对，重新算：$7(2n+2) = 14n+14$，$38-14=24$。或者先约分：$\frac{7n+19}{n+1} = 7 + \frac{12}{n+1}$
- **期望学生回答**：化简为 $7 + \frac{12}{n+1}$

**步骤4：确定整数解**
- **引导语**："要使结果为整数，$\frac{12}{n+1}$ 必须是整数，这意味着什么？"
- **关键提示**：$n+1$ 必须是 $12$ 的约数，且 $n \geq 1$
- **期望学生回答**：$n+1 \in \{2, 3, 4, 6, 12\}$（注意 $n+1=1$ 时 $n=0$ 不是正整数），所以 $n \in \{1, 2, 3, 5, 11\}$，共5个

## 五、关联拓展

### 相关结论总结

**重要定理（结论总结1.1）：**
若 $\{a_n\}$ 与 $\{b_n\}$ 为等差数列，且前 $n$ 项和分别为 $S_n$ 和 $T_n$，则：
$$\frac{a_m}{b_m} = \frac{S_{2m-1}}{T_{2m-1}}$$

**证明：**
$$S_{2m-1} = \frac{(2m-1)(a_1 + a_{2m-1})}{2} = \frac{(2m-1) \cdot 2a_m}{2} = (2m-1)a_m$$
同理 $T_{2m-1} = (2m-1)b_m$，所以 $\frac{a_m}{b_m} = \frac{S_{2m-1}}{T_{2m-1}}$。

## 六、备注

- **本题地位**：等差数列中最经典的结论应用题，高考常考，必须掌握 $\frac{a_n}{b_n} = \frac{A_{2n-1}}{B_{2n-1}}$ 这一性质。
- **教学价值**：强化核心结论记忆，训练分式化简和整除分析能力。
- **常见错误**：$n+1=1$ 对应 $n=0$ 不是正整数，必须排除；$12$ 的约数容易漏掉 $1$（虽然本题要排除）或 $12$。
