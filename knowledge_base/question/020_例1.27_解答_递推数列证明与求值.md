# 【A_020_2】递推数列证明与求值（2015广东卷）

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P20 例1.27
&gt; 
&gt; **来源**：2015年广东卷文科第19题

**题目：**

设数列 $\{a_n\}$ 的前 $n$ 项和为 $S_n$，$n \in \mathbb{N}^*$。已知 $a_1=1$，$a_2=\frac{3}{2}$，$a_3=\frac{5}{4}$，且当 $n \geq 2$ 时，
$$4S_{n+2} + 5S_n = 8S_{n+1} + S_{n-1}$$

(I) 求 $a_4$ 的值；

(II) 证明 $\left\{a_{n+1} - \frac{1}{2}a_n\right\}$ 为等比数列。

---

## 二、答案与解析

### 答案

**(I)** $a_4 = \frac{7}{8}$；**(II)** 证明见解析

### 解析

**步骤1：求 $a_4$ 的值**

当 $n=2$ 时，代入递推式：
$$4S_4 + 5S_2 = 8S_3 + S_1$$

展开：
$$4(a_1+a_2+a_3+a_4) + 5(a_1+a_2) = 8(a_1+a_2+a_3) + a_1$$

代入已知值 $a_1=1, a_2=\frac{3}{2}, a_3=\frac{5}{4}$：
$$4\left(1+\frac{3}{2}+\frac{5}{4}+a_4\right) + 5\left(1+\frac{3}{2}\right) = 8\left(1+\frac{3}{2}+\frac{5}{4}\right) + 1$$

计算：
$$4\left(\frac{15}{4}+a_4\right) + \frac{25}{2} = 8 \times \frac{15}{4} + 1$$
$$15 + 4a_4 + 12.5 = 30 + 1$$
$$4a_4 = \frac{7}{2}$$

解得：
$$a_4 = \frac{7}{8}$$

**步骤2：递推关系变形**

原式 $4S_{n+2} + 5S_n = 8S_{n+1} + S_{n-1}$（$n \geq 2$）

移项：
$$4(S_{n+2}-S_{n+1}) + (S_n-S_{n-1}) = 4(S_{n+1}-S_n)$$

即：
$$4a_{n+2} + a_n = 4a_{n+1}$$

整理得：
$$a_{n+2} - a_{n+1} = -\frac{1}{4}a_n$$

**步骤3：构造等比数列**

要证 $\left\{a_{n+1} - \frac{1}{2}a_n\right\}$ 为等比数列，考察：

$$a_{n+2} - \frac{1}{2}a_{n+1} = \left(a_{n+1} - \frac{1}{4}a_n\right) - \frac{1}{2}a_{n+1} = \frac{1}{2}a_{n+1} - \frac{1}{4}a_n = \frac{1}{2}\left(a_{n+1} - \frac{1}{2}a_n\right)$$

令 $b_n = a_{n+1} - \frac{1}{2}a_n$，则 $b_{n+1} = \frac{1}{2}b_n$。

首项 $b_1 = a_2 - \frac{1}{2}a_1 = \frac{3}{2} - \frac{1}{2} = 1$。

故 $\left\{a_{n+1} - \frac{1}{2}a_n\right\}$ 是**首项为1，公比为 $\frac{1}{2}$ 的等比数列**。

## 三、分步教学引导（用于AI Agent）

**步骤1：理解递推式结构**
- **引导语**："这个递推式涉及四个不同的和式 $S_{n+2}, S_{n+1}, S_n, S_{n-1}$，如何利用 $a_n = S_n - S_{n-1}$ 化简？"
- **关键提示**：将和式差转化为通项
- **期望学生回答**：$S_{n+2}-S_{n+1}=a_{n+2}$，$S_{n+1}-S_n=a_{n+1}$，$S_n-S_{n-1}=a_n$

**步骤2：建立通项递推**
- **引导语**："代入后得到关于 $a_n$ 的三项递推关系是什么？"
- **关键提示**：$4a_{n+2} + a_n = 4a_{n+1}$
- **期望学生回答**：$4a_{n+2} - 4a_{n+1} = -a_n$

**步骤3：构造目标数列**
- **引导语**："要证明 $a_{n+1} - \frac{1}{2}a_n$ 是等比数列，需要验证 $a_{n+2} - \frac{1}{2}a_{n+1}$ 与它的关系。利用步骤2的结论进行代换。"
- **关键提示**：将 $a_{n+2} = a_{n+1} - \frac{1}{4}a_n$ 代入
- **期望学生回答**：$a_{n+2} - \frac{1}{2}a_{n+1} = \frac{1}{2}(a_{n+1} - \frac{1}{2}a_n)$，公比为 $\frac{1}{2}$

**步骤4：计算首项**
- **引导语**："这个等比数列的首项是什么？用题目给出的 $a_1, a_2$ 计算。"
- **关键提示**：$n=1$ 时，$a_2 - \frac{1}{2}a_1$
- **期望学生回答**：$1$

## 五、关联拓展

### 相关公式总结

**和式递推转通项技巧：**
- $S_{n+k} - S_{n+k-1} = a_{n+k}$
- 当出现 $S_{n+2}, S_{n+1}, S_n, S_{n-1}$ 等多阶和式时，通常作差转化为通项的三项递推

**构造等比数列常用形式：**
- 若 $a_{n+2} = pa_{n+1} + qa_n$，可构造 $a_{n+2} - \alpha a_{n+1} = \beta(a_{n+1} - \alpha a_n)$
- 本题 $\alpha = \frac{1}{2}$，$\beta = \frac{1}{2}$
