# 【A_017】等比数列对数求和（山西四校联考）

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P17 例1.23
&gt; 
&gt; **来源**：2015年山西省四校联考

**题目：**

等比数列 $\{a_n\}$ 满足 $a_n &gt; 0, n \in \mathbb{N}^*$，且 $a_3 a_{2n-3} = 2^{2n} (n \geq 2)$，则当 $n \geq 1$ 时，$\log_2 a_1 + \log_2 a_2 + \cdots + \log_2 a_{2n-1} = $（ ）。

A. $n(2n-1)$

B. $(n+1)^2$

C. $n^2$

D. $(n-1)^2$

---

## 二、答案与解析

### 答案

**A**

### 解析

**步骤1：利用等比数列性质求通项**

由 $a_3 \cdot a_{2n-3} = 2^{2n}$，根据等比数列性质：若 $m+n=p+q$，则 $a_m \cdot a_n = a_p \cdot a_q$。

注意到 $3 + (2n-3) = 2n$，所以 $a_3 \cdot a_{2n-3} = a_n \cdot a_n = a_n^2$。

因此 $a_n^2 = 2^{2n} = (2^n)^2$。

又因为 $a_n &gt; 0$，所以 $a_n = 2^n$。

**步骤2：化简对数求和式**

令 $S = \log_2 a_1 + \log_2 a_2 + \cdots + \log_2 a_{2n-1}$。

由 $a_n = 2^n$，得 $\log_2 a_k = \log_2 2^k = k$。

所以 $S = 1 + 2 + 3 + \cdots + (2n-1)$。

**步骤3：利用倒序相加或等差数列求和**

这是首项为1，末项为 $2n-1$，项数为 $2n-1$ 的等差数列。

$$S = \frac{(2n-1)(1 + 2n-1)}{2} = \frac{(2n-1) \cdot 2n}{2} = n(2n-1)$$

故选 **A**。

## 三、分步教学引导（用于AI Agent）

### 整体思路引导

**步骤1：分析已知条件**
- **引导语**："题目给出了等比数列的什么条件？$a_3 a_{2n-3} = 2^{2n}$ 这个等式中，两个下标有什么关系？"
- **关键提示**：$3 + (2n-3) = 2n$，联想到等比数列性质 $a_m \cdot a_n = a_p \cdot a_q$（当 $m+n=p+q$ 时）
- **期望学生回答**：$a_3 \cdot a_{2n-3} = a_n^2 = 2^{2n}$，所以 $a_n = 2^n$

**步骤2：转化对数求和**
- **引导语**："现在我们知道了 $a_n = 2^n$，那么 $\log_2 a_k$ 等于什么？"
- **关键提示**：$\log_2 2^k = k$
- **期望学生回答**：$\log_2 a_k = k$，所以原式等于 $1 + 2 + \cdots + (2n-1)$

**步骤3：计算等差数列和**
- **引导语**："这是一个什么数列的求和？有多少项？"
- **关键提示**：首项1，末项 $2n-1$，共 $2n-1$ 项的等差数列
- **期望学生回答**：$S = \frac{(2n-1)(1 + 2n-1)}{2} = n(2n-1)$
