# 【A_002】等差数列性质应用（2019江苏卷）

## 一、题目内容

&gt; **原题位置**：《新高掌3数列与不等式》P2 变式1.2.1
&gt; 
&gt; **来源**：2019年江苏卷第8题

**题目：**

已知数列 $\{a_n\}$ 是等差数列，$S_n$ 是其前 $n$ 项和，若 $a_2a_5 + a_8 = 0$，$S_9 = 27$，则 $S_8$ 的值是 ______。

---

## 二、答案与解析

### 答案

**16**

### 解析

**步骤1：设定基本量**

设等差数列 $\{a_n\}$ 的首项为 $a_1$，公差为 $d$。

通项公式：$a_n = a_1 + (n-1)d$
前 $n$ 项和：$S_n = na_1 + \frac{n(n-1)}{2}d$

**步骤2：翻译已知条件**

条件1：$a_2a_5 + a_8 = 0$
- $a_2 = a_1 + d$
- $a_5 = a_1 + 4d$  
- $a_8 = a_1 + 7d$

代入得：$(a_1 + d)(a_1 + 4d) + (a_1 + 7d) = 0$

条件2：$S_9 = 27$
$$S_9 = 9a_1 + \frac{9 \times 8}{2}d = 9a_1 + 36d = 27$$

化简得：$a_1 + 4d = 3$

**步骤3：利用等差中项性质简化**

注意到 $a_1 + 4d = a_5$，所以 $a_5 = 3$。

又 $S_9 = \frac{9(a_1 + a_9)}{2} = 9a_5 = 27$，验证一致。
