# 【C_072】裂项相消法经典题（2011全国卷）

## 一、题目内容

> **原题位置**：《新高考你真的掌握了吗？》P72 例3.9
> 
> **来源**：2011年全国卷理科第20题（没标注就填未知） 

**题目：**

设数列 $\{a_n\}$ 满足 $a_1 = 0$ 且 $\frac{1}{1-a_{n+1}} - \frac{1}{1-a_n} = 1$。

**(I)** 求 $\{a_n\}$ 的通项公式；

**(II)** 设 $b_n = \frac{1-\sqrt{a_{n+1}}}{\sqrt{n}}$，$S_n$ 为 $\{b_n\}$ 的前 $n$ 项和，证明：$S_n < 1$。

---

## 二、答案与解析

### (I) 求通项公式

**答案：** $a_n = 1 - \frac{1}{n}$，$n \in \mathbb{N}^*$

**解析：**

**关键观察**：递推式 $\frac{1}{1-a_{n+1}} - \frac{1}{1-a_n} = 1$ 的结构特点

**步骤1：构造新数列**

令 $c_n = \frac{1}{1-a_n}$，则递推式变为：
$$c_{n+1} - c_n = 1$$

**步骤2：识别等差数列**

$\{c_n\}$ 是以 **1为公差** 的等差数列。

**步骤3：求首项**

$$c_1 = \frac{1}{1-a_1} = \frac{1}{1-0} = 1$$

**步骤4：求 $c_n$ 通项**

$$c_n = c_1 + (n-1) \cdot 1 = 1 + (n-1) = n$$

**步骤5：回代求 $a_n$**

由 $c_n = \frac{1}{1-a_n} = n$，得：
$$1 - a_n = \frac{1}{n}$$
$$a_n = 1 - \frac{1}{n}$$

---

### (II) 证明 $S_n < 1$

**答案：** 证明见解析

**解析：**

**步骤1：求 $a_{n+1}$**

由 (I) 知 $a_{n+1} = 1 - \frac{1}{n+1} = \frac{n}{n+1}$

**步骤2：化简 $b_n$**

$$b_n = \frac{1-\sqrt{a_{n+1}}}{\sqrt{n}} = \frac{1-\sqrt{\frac{n}{n+1}}}{\sqrt{n}}$$

分子分母同乘 $\sqrt{n+1}$：
$$= \frac{\sqrt{n+1} - \sqrt{n}}{\sqrt{n} \cdot \sqrt{n+1}}$$

**步骤3：裂项变形**

$$b_n = \frac{\sqrt{n+1} - \sqrt{n}}{\sqrt{n(n+1)}} = \frac{1}{\sqrt{n}} - \frac{1}{\sqrt{n+1}}$$

> **技巧说明**：这是根式型的裂项，利用了 $\frac{\sqrt{n+1}-\sqrt{n}}{\sqrt{n}\cdot\sqrt{n+1}} = \frac{\sqrt{n+1}}{\sqrt{n}\cdot\sqrt{n+1}} - \frac{\sqrt{n}}{\sqrt{n}\cdot\sqrt{n+1}} = \frac{1}{\sqrt{n}} - \frac{1}{\sqrt{n+1}}$

**步骤4：求和（裂项相消）**

$$S_n = \sum_{k=1}^{n} b_k = \sum_{k=1}^{n} \left(\frac{1}{\sqrt{k}} - \frac{1}{\sqrt{k+1}}\right)$$

展开写出各项：
$$= \left(1 - \frac{1}{\sqrt{2}}\right) + \left(\frac{1}{\sqrt{2}} - \frac{1}{\sqrt{3}}\right) + \left(\frac{1}{\sqrt{3}} - \frac{1}{\sqrt{4}}\right) + \cdots + \left(\frac{1}{\sqrt{n}} - \frac{1}{\sqrt{n+1}}\right)$$

**步骤5：观察抵消规律**

中间项全部抵消，只剩 **首项** 和 **末项**：
$$S_n = 1 - \frac{1}{\sqrt{n+1}}$$

**步骤6：证明不等式**

$$\because n \geq 1, \therefore \sqrt{n+1} > 1, \therefore \frac{1}{\sqrt{n+1}} > 0$$

$$\therefore S_n = 1 - \frac{1}{\sqrt{n+1}} < 1$$

**证毕。**

## 三、分步教学引导（用于AI Agent）

### 第(I)问引导

**步骤1：观察递推式结构**
- **引导语**："观察递推式 $\frac{1}{1-a_{n+1}} - \frac{1}{1-a_n} = 1$，你觉得哪个部分可以看作一个整体？"
- **关键提示**：把 $\frac{1}{1-a_n}$ 看作新数列的第 $n$ 项
- **期望学生回答**：令 $c_n = \frac{1}{1-a_n}$

**步骤2：变形识别数列类型**
- **引导语**："如果令 $c_n = \frac{1}{1-a_n}$，那么递推式变成了什么形式？这是什么数列？"
- **关键提示**：变成 $c_{n+1} - c_n = 1$，是等差数列
- **期望学生回答**：$c_{n+1} - c_n = 1$，等差数列

**步骤3：确定等差数列的首项**
- **引导语**："等差数列需要知道首项，$c_1$ 等于多少？"
- **关键提示**：$c_1 = \frac{1}{1-a_1} = \frac{1}{1-0} = 1$
- **期望学生回答**：$c_1 = 1$

**步骤4：写出 $c_n$ 的通项公式**
- **引导语**："现在可以写出 $c_n$ 的通项了"
- **关键提示**：$c_n = c_1 + (n-1)d = 1 + (n-1) \cdot 1 = n$
- **期望学生回答**：$c_n = n$

**步骤5：回代求 $a_n$**
- **引导语**："知道了 $c_n = n$，而 $c_n = \frac{1}{1-a_n}$，怎么求 $a_n$？"
- **关键提示**：解方程 $\frac{1}{1-a_n} = n$
- **期望学生回答**：$a_n = 1 - \frac{1}{n}$

---

### 第(II)问引导

**步骤1：代入化简 $b_n$**
- **引导语**："先把 $a_{n+1}$ 代入 $b_n$ 的表达式。由 (I) 知 $a_{n+1}$ 等于什么？"
- **关键提示**：$a_{n+1} = 1 - \frac{1}{n+1} = \frac{n}{n+1}$
- **期望学生回答**：$b_n = \frac{1-\sqrt{\frac{n}{n+1}}}{\sqrt{n}}$

**步骤2：分子有理化**
- **引导语**："分子 $1 - \sqrt{\frac{n}{n+1}}$ 可以怎么变形？试试通分。"
- **关键提示**：$= \frac{\sqrt{n+1}-\sqrt{n}}{\sqrt{n+1}}$
- **期望学生回答**：分子分母同乘 $\sqrt{n+1}$

**步骤3：识别裂项形式**
- **引导语**："现在 $b_n = \frac{\sqrt{n+1}-\sqrt{n}}{\sqrt{n}\cdot\sqrt{n+1}}$，这可以拆成什么形式？"
- **关键提示**：拆成两个分式之差
- **期望学生回答**：$b_n = \frac{1}{\sqrt{n}} - \frac{1}{\sqrt{n+1}}$

**步骤4：写出求和展开式**
- **引导语**："写出 $S_n$ 的前几项和后几项，看看哪些项会抵消？"
- **关键提示**：写出 $k=1,2,3$ 和 $k=n-2,n-1,n$ 的项
- **期望学生回答**：中间项全部抵消

**步骤5：确定剩余项**
- **引导语**："抵消后，$S_n$ 等于什么？"
- **关键提示**：只剩首项 1 和末项 $-\frac{1}{\sqrt{n+1}}$
- **期望学生回答**：$S_n = 1 - \frac{1}{\sqrt{n+1}}$

**步骤6：完成证明**
- **引导语**："现在证明 $S_n < 1$，你观察到了什么？"
- **关键提示**：$\frac{1}{\sqrt{n+1}} > 0$，所以 $S_n = 1 - \text{正数} < 1$
- **期望学生回答**：因为 $\frac{1}{\sqrt{n+1}} > 0$，所以 $S_n < 1$

## 五、关联拓展（可选，比较特殊）

### 相关公式（本书P72下方）

**裂项相消常用公式：**

1. **分式型**：
   $$\frac{1}{n(n+k)} = \frac{1}{k}\left(\frac{1}{n} - \frac{1}{n+k}\right)$$

2. **连乘型**：
   $$\frac{1}{n(n+1)(n+2)} = \frac{1}{2}\left[\frac{1}{n(n+1)} - \frac{1}{(n+1)(n+2)}\right]$$

3. **奇数型**：
   $$\frac{1}{(2n-1)(2n+1)} = \frac{1}{2}\left(\frac{1}{2n-1} - \frac{1}{2n+1}\right)$$

4. **根式型（本题所用）**：
   $$\frac{1}{\sqrt{n+k}+\sqrt{n}} = \frac{1}{k}(\sqrt{n+k}-\sqrt{n})$$
