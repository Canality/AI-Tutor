## 一、产品定位与演进路线
### 1.1 产品愿景
打造一款面向高中数学数列学习的AI智能辅导系统，通过苏格拉底式教学法和个性化推荐，帮助学生真正理解数学思维，而非死记硬背。

### 1.2 版本演进
| 版本 | 阶段 | 核心能力 | 状态 |
| --- | --- | --- | --- |
| V1.0 | MVP | 基础问答 + 题目管理 | ✅ 已完成 |
| V2.0 | rag增强 | RAG + 向量数据库 | ✅ 已完成 |
| V3.0 | 智能个性化 | 认知诊断 + 个性化路径 | 🚧 规划中 |

### 1.3 V3.0 核心目标
- 精准诊断: 通过BKT+IRT算法，精准定位学生能力水平
- 千人千面: 基于认知状态，动态调整教学策略
- 智能推荐: Advisor Agent主动规划学习路径
- 数据闭环: 完整的学习数据追踪与反馈

## 四、功能优先级矩阵
### 4.1 V3.0 开发优先级
```
高价值 ↑

P0：BKT/IRT诊断、Advisor推荐、Redis缓存
P1：掌握度看板、错题本、复习提醒
P2：学习报告、知识树、激励机制
P3：社交功能、多语言、移动端适配

低难度 →
```
